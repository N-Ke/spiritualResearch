# フェイスレス動画自動生成システム 初心者マニュアル

## はじめに

このマニュアルは、**フェイスレス動画自動生成システム**を初めて使う方のためのガイドです。

「フェイスレス動画って何？」「何から始めればいいの？」という疑問に、すべて答えます。

---

## フェイスレス動画とは？

**顔出しなし**で作る動画のこと。以下の要素で構成されます：

| 要素 | 説明 |
|------|------|
| AI音声ナレーション | テキストをAIが読み上げる（ElevenLabs等） |
| テキストオーバーレイ | 画面に表示するテロップ・キーワード |
| B-roll映像素材 | ストック映像やグラフィック素材 |
| BGM・効果音 | 場面に合った音楽と効果音 |

**メリット：**
- 顔出し不要 → 身バレリスクなし
- 量産可能 → AI音声＋素材の組み合わせで効率的
- 属人性が低い → チームでの運用も容易

---

## このシステムでできること

```
ジャンルを決める
    ↓
ターゲットの悩みを自動リサーチ（100〜150個）
    ↓
コンセプト（どんな発信者になるか）を設計
    ↓
キャラクター（声の個性）を自動生成
    ↓
ネタ出し（何の動画を作るか）を自動計画
    ↓
動画台本を5媒体向けに一括生成
    ↓
→ あとはAI音声ツールで音声を作り、映像を編集して投稿するだけ
```

## 対応する5つの媒体

| 媒体 | 形式 | 特徴 |
|------|------|------|
| **TikTok** | ショート動画（15〜180秒） | バイラル力が最強。認知拡大に最適 |
| **Instagram リール** | ショート動画（15〜90秒） | 保存されやすい。ライフスタイル系に強い |
| **Facebook リール** | ショート動画（15〜90秒） | 30代以上へのリーチに強い |
| **YouTube ショート** | ショート動画（〜60秒） | チャンネル登録への導線に最適 |
| **YouTube ロング動画** | 通常動画（8〜30分） | 信頼構築・収益化の本命 |

**おすすめの組み合わせ：**
- **認知拡大** → TikTok + IG Reels + FB Reels
- **ファン化** → YouTube ショート → YouTube ロング動画
- **全部やる** → 5媒体すべて（このシステムなら一括生成可能）

## 最初の一歩：3ステップで始める

### ステップ1：初回セットアップ（30〜60分、最初の1回だけ）

AIに **「初回セットアップを開始して」** と言うだけ。

以下の3つが順番に自動で進みます：

1. **ターゲットリサーチ** — あなたのジャンルで、誰がどんな悩みを持っているかを100〜150個棚卸し
2. **コンセプト設計** — どの悩みを、どう解決する人になるかを決定
3. **キャラクター設計** — AIが心理学フレームワークを使って、あなた専用の「声のキャラクター」を自動生成

**やることは「OK」を出すだけ。** 各ステップの確認ポイントで内容を見て、OKなら次に進みます。

### ステップ2：ネタ出し（毎日5分）

AIに **「今日のネタを出して」** と言う。

悩みデータベース × キャラクターの視点から、動画テーマを3つ提案してくれます。
好きなものを選ぶだけ。

### ステップ3：台本生成（毎日10分）

AIに **「この情報で動画台本を作って」** と言い、テーマを伝える。

5媒体分の台本（ナレーション・テロップ・映像指示・BGM指示）が一括生成されます。

## 生成される台本の中身

各台本には以下が含まれます：

```
━━━━━━━━━━━━━━━━━━━━
台本の構成要素
━━━━━━━━━━━━━━━━━━━━

1. ナレーション原稿
   → AI音声で読み上げるテキスト
   → キャラクターの「声」が反映される

2. テキストオーバーレイ指示
   → 画面に表示するテロップの内容とタイミング
   → 音なしでも伝わる設計

3. 映像素材（B-roll）指示
   → 各シーンで使う映像のイメージ説明
   → ストック素材のキーワード候補

4. BGM・効果音指示
   → 場面に合ったBGMのムード
   → 効果音のタイミングと種類

5. CTA（行動喚起）
   → 媒体ごとの最適なCTA
   → フォロー/保存/チャンネル登録 等
```

## 台本から動画を作るまでの流れ

```
[このシステムが生成]
台本（ナレーション + テロップ + 映像指示 + BGM指示）
    ↓
[あなたがやること]
1. AI音声ツールでナレーション音声を生成
   例：ElevenLabs、VOICEVOX、CoeFont 等

2. 映像素材を調達
   例：Pexels、Pixabay、Canva 等（無料素材多数）

3. 動画編集ツールで組み合わせ
   例：CapCut、Canva動画、DaVinci Resolve 等

4. 各媒体に投稿
```

**所要時間の目安：** 
- ショート動画：1本あたり15〜30分
- YouTube ロング動画：1本あたり1〜3時間

## キャラクターについて

### キャラクターとは？

このシステムで言う「キャラクター」は、あなたの動画の **「声の個性」** です。

心理学フレームワーク（MBTI・ストレングスファインダー・エニアグラム）を組み合わせて、以下を自動設計します：

- **どんな切り口で語るか**（判断構造）
- **どんな言い方をするか**（表現ルール）
- **何を大事にするか**（価値観）
- **何を言わないか**（禁止事項）

### なぜキャラクターが必要？

- **一貫性** → 毎回同じ「声」で語ることで、視聴者が安心してフォローできる
- **差別化** → 同じテーマでも、キャラクターの切り口で他の発信者と差がつく
- **効率** → AIがキャラクターの「声」を再現するので、毎回トーンを考える必要がない

### キャラクターの成長

キャラクターは固定ではありません。動画の反応データやトレンドを取り込んで、人間のように成長します。

- **変わらないもの** → 根本的な判断構造、価値観
- **成長するもの** → 知識、表現力、視聴者との関係性

月1回程度、AIに「反応が良かった動画を分析して」と言えば、キャラクターが成長します。

## FAQ

**Q: 心理学の知識がないけど使える？**
A: はい。AIがすべて自動で選定・組み合わせます。質問に答えるだけでOKです。

**Q: 1回のセットアップでずっと使える？**
A: はい。セットアップは最初の1回だけ。あとは毎日「ネタ出し→台本生成」の繰り返しです。

**Q: 複数のジャンルで使える？**
A: はい。ジャンルごとにセットアップを実行すれば、複数のキャラクターを運用できます。

**Q: 顔出し動画にも使える？**
A: はい。台本の構造は顔出し動画にもそのまま使えます。映像素材指示を無視すればOKです。

**Q: 1日何本くらい作れる？**
A: 台本生成だけなら、1回の指示で5媒体分を一括生成できます。動画制作（音声＋編集）の時間が別途必要です。

**Q: どの媒体から始めるべき？**
A: まずは **TikTok + YouTube ショート** の2つがおすすめ。認知拡大＋チャンネル登録の両方を狙えます。慣れてきたらIG Reels、FB Reels、YouTube ロングを追加していきましょう。

**Q: 生成された台本はそのまま使っていい？**
A: 基本的にはそのまま使えるクオリティですが、確認ステップが必ず挟まるので、気になる箇所は修正できます。
