# 動画媒体別出力フォーマット

このファイルは、フェイスレス動画自動生成システムのPhase 3で参照される。
各動画媒体の特性と、キャラクターパラメータに基づく表現ルールを定義する。

---

## 1. 媒体別基本特性

### 1-1. TikTok

| 項目 | 仕様 |
|------|------|
| 動画形式 | 縦型（9:16）、15〜180秒 |
| 最適な尺 | 30〜60秒（完視聴率が最も高い） |
| エンゲージメント特性 | いいね、コメント、シェア、保存、デュエット |
| コンテンツ寿命 | 中程度（FYPでバイラル可能、数日〜数週間） |
| アルゴリズム傾向 | 完視聴率最重視、初動エンゲージメント、リピート再生 |
| ユーザー層 | 10〜30代中心、エンタメ＋学び系が強い |

**TikTokで効果的なパターン：**
- 最初の1秒で引きつけるフック（テキスト＋音声同時）
- 情報密度を高く、テンポよく展開
- ループ構造（最後→最初がつながる）
- トレンド音源の活用
- テキストオーバーレイを多用（音なしでも伝わるように）
- 「Part 1」等のシリーズ化で続きを見せる

**TikTokで避けるべきこと：**
- 冒頭3秒の無駄（ロゴ、挨拶、前置き）
- 単調な展開（視聴離脱の原因）
- 横型映像の流用（縦型ネイティブが必須）
- 他媒体のウォーターマークが入った映像

### 1-2. Instagram リール

| 項目 | 仕様 |
|------|------|
| 動画形式 | 縦型（9:16）、15〜90秒 |
| 最適な尺 | 15〜30秒（Explore表示に有利） |
| エンゲージメント特性 | いいね、保存、シェア、コメント |
| コンテンツ寿命 | 中〜長（Explore経由で長期流入） |
| アルゴリズム傾向 | 保存率・シェア率重視、Explore表示、リーチ |
| ユーザー層 | 20〜40代、ライフスタイル＋学び系 |

**IG リールで効果的なパターン：**
- 美しいビジュアル＋情報のバランス
- 「保存したくなる」実用的な情報
- カルーセル風の知識系コンテンツ
- ハッシュタグ戦略（関連3〜5個 + ニッチ5〜10個）
- キャプションに詳細情報を追加（SEO効果）
- フィードのビジュアル統一感を意識

**IG リールで避けるべきこと：**
- TikTokロゴ付きの動画転載
- 統一感のないコンテンツ
- 過度な宣伝（フォロワー離脱の原因）
- エンタメに寄りすぎ（IGは「価値ある情報」が求められる）

### 1-3. Facebook リール

| 項目 | 仕様 |
|------|------|
| 動画形式 | 縦型（9:16）、15〜90秒 |
| 最適な尺 | 30〜60秒 |
| エンゲージメント特性 | いいね、コメント、シェア |
| コンテンツ寿命 | 短〜中程度 |
| アルゴリズム傾向 | シェア率重視、コメント数、リーチ |
| ユーザー層 | 30〜50代中心、実用情報＋共感系 |

**FB リールで効果的なパターン：**
- 「共感→シェア」の流れを意識（30代以上が共感するテーマ）
- 実用的なライフハック・仕事術
- 感情を動かすストーリーテリング
- テキストオーバーレイを大きく見やすく
- 問いかけ型でコメントを誘発

**FB リールで避けるべきこと：**
- 若者向けすぎるトレンド・ノリ
- テキストが小さい（FB層は大きめテキスト必須）
- コンテンツの質が低い転載
- 過激なフック（FB層には不快に映る）

### 1-4. YouTube ショート

| 項目 | 仕様 |
|------|------|
| 動画形式 | 縦型（9:16）、最大60秒 |
| 最適な尺 | 30〜50秒 |
| エンゲージメント特性 | いいね、コメント、チャンネル登録 |
| コンテンツ寿命 | 中〜長（検索経由の流入もあり） |
| アルゴリズム傾向 | 視聴維持率、チャンネル登録転換率 |
| ユーザー層 | 幅広い（YouTubeユーザー全体にリーチ） |

**YT ショートで効果的なパターン：**
- ロング動画への導線として活用
- 「続きはメイン動画で」のフック
- 1つのトピックに絞ったミニ解説
- チャンネル登録を自然に促すCTA
- ロング動画の最も面白い部分を切り出し

**YT ショートで避けるべきこと：**
- サムネイルに頼れない（自動生成される）
- 説明欄が限定的（情報はすべて動画内に）
- 低品質な映像（YouTube視聴者の目が肥えている）

### 1-5. YouTube ロング動画

| 項目 | 仕様 |
|------|------|
| 動画形式 | 横型（16:9）、8〜30分 |
| 最適な尺 | 10〜15分（広告収益最適化領域） |
| エンゲージメント特性 | 視聴維持率、コメント、チャンネル登録、高評価 |
| コンテンツ寿命 | 非常に長い（検索経由で年単位の流入） |
| アルゴリズム傾向 | 視聴維持率、CTR（サムネ＋タイトル）、セッション時間 |
| ユーザー層 | 目的を持って視聴する層（学習・解決意欲が高い） |

**YouTube ロング動画で効果的なパターン：**
- SEOを意識したタイトル＋サムネイル設計
- フック（最初の15秒で離脱を防ぐ）
- チャプター構成で視聴しやすく
- 導入→展開→結論の3部構成
- ストーリーテリング＋データ＋具体例
- エンドスクリーン＋カードでの関連動画誘導
- コメント欄での質問誘導

**YouTube ロング動画で避けるべきこと：**
- テキストの棒読み感（AI音声でも抑揚を意識）
- 起伏のない単調な展開
- 結論を最後まで引っ張りすぎ（20%離脱ポイントで価値を示す）
- サムネイルとタイトルのミスマッチ

## 2. フェイスレス動画の共通構成要素

すべての動画台本に含める6要素：

### 2-1. ナレーション原稿

```
- AI音声で読み上げるテキスト
- キャラクターの「声」（トーン・判断構造・口癖）を反映
- 話し言葉として自然に聞こえるよう構成
- 句読点・間・強調を指示含む
```

### 2-2. テキストオーバーレイ指示

```
- 画面上に表示するテロップの内容とタイミング
- キーワード・数字の強調表示
- フック用の大きな文字指示
- 音なしでも内容が伝わる設計
```

### 2-3. 映像素材（B-roll）指示

```
- 各セクションで使用する映像のイメージ説明
- ストック素材のキーワード候補
- アニメーション・グラフィックの指示
- 画面切り替えのタイミング
```

### 2-4. BGM・効果音指示

```
- 場面に合ったBGMのムード指示（例：「前向きで軽快なBGM」）
- 転換点での効果音指示（例：「ティン！という気づき効果音」）
- BGM切り替えのタイミング
```

### 2-5. カット指示

```
- シーンの切り替えタイミング（秒単位）
- トランジションの種類（カット/フェード/スライド）
- テンポの指示（速い/普通/ゆっくり）
```

### 2-6. CTA（行動喚起）

```
- 媒体ごとの最適なCTA
- TikTok: フォロー + コメント誘導 + Part 2予告
- IG Reels: 保存 + フォロー + コメント
- FB Reels: シェア + コメント
- YT Shorts: チャンネル登録 + ベルマーク + ロング動画誘導
- YT ロング: チャンネル登録 + コメント + 関連動画
```
