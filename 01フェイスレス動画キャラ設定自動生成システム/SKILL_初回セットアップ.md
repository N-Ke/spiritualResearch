# 初回セットアップ自動シーケンス

## このスキルの目的

**「初回セットアップを開始して」の一言で、ターゲットリサーチ → コンセプト設計 → キャラクター設計まで一気通貫で実行する。**

ユーザーは各Phaseの確認ポイントで「OK」を出すだけ。
個別スキルを1つずつ呼び出す必要はない。

---

## 起動条件

ユーザーが以下のいずれかの意図を示したとき：
- 「初回セットアップを開始して」
- 「最初から全部やって」
- 「ゼロから始めたい」
- 「ジャンルを決めたので設定して」
- 初めてこのシステムを使う意図が読み取れた場合

---

## Phase 2: コンセプト設計

### 実行内容

`SKILL_コンセプト設計.md` の全Stepを実行する。

### 手順

1. Phase 1の結果を自動で読み込む（ユーザーに再入力を求めない）

```

```

2. 悩みクラスターの選定（Step 2）

```

選び方のコツ：
- 1〜3個に絞る
- 自分の経験・スキルと一致するもの
- 迷ったら「お任せ」で、AIが最適な組み合わせを提案します

3. ポジショニングの選定（Step 3）

```

2. 専門家型（知識で語る）
3. 伴走型（一緒に考える）
4. キュレーター型（情報を整理する）
5. 挑戦者型（リアルタイムで共有する）

4. コンセプト文の生成（Step 4）+ 差別化軸の明確化（Step 5）

## 中断・再開の対応

### 途中で中断した場合

```
セットアップを途中で中断しても問題ありません。
再開するときは：

- Phase 1まで完了している → 「コンセプト設計から再開して」
- Phase 2まで完了している → 「キャラクター設計から再開して」
- 最初からやり直したい → 「初回セットアップを最初からやり直して」
```

### やり直したい場合

```
特定のPhaseだけやり直すこともできます：
- 「ターゲットリサーチをやり直して」→ Phase 1に戻る
- 「コンセプトを変えたい」→ Phase 2に戻る
- 「キャラクターを作り直して」→ Phase 3に戻る

※ 前のPhaseに戻ると、それ以降のPhaseも再実行が必要です
```

## ショートカットコマンド

| コマンド | 動作 |
|---------|------|
| 「初回セットアップを開始して」 | Phase 1から全自動開始 |
| 「最初から全部やって」 | 同上 |
| 「コンセプト設計から再開して」 | Phase 2から再開 |
| 「キャラクター設計から再開して」 | Phase 3から再開 |
| 「やり直し」 | Phase 1に戻って最初から |

## エラーハンドリング

### 既にセットアップ済みの場合

```
既に以下が作成済みのようです：
- ターゲットリサーチ結果: [あり/なし]
- コンセプトシート: [あり/なし]
- キャラクター設計書: [あり/なし]

上書きしますか？
1.「上書き」→ 最初からやり直し
2.「足りないところだけ」→ 未完了のPhaseから開始
3.「キャンセル」→ セットアップを中止
```

### 入力が不十分な場合

Phase Aの入力が不十分でも止めない。
`SKILL_ターゲットリサーチ.md` の補助フロー（A-3, A-4）に従って、AIが補助する。
それでも不十分なら「不明」で仮置きし、Phase D以降の出力を見て修正する（10分ルール）。
