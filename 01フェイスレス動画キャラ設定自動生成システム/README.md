# フェイスレス動画自動生成システム

## これは何か

性格心理学フレームワーク（MBTI・ストレングスファインダー・エニアグラム）を組み合わせて **動画発信用のキャラクター（ペルソナ）を自動設計** し、そのキャラクターの「声」で **フェイスレス（顔出しなし）動画の台本を一括生成** するシステムです。

ジャンルを決めるだけで、ターゲットリサーチ → コンセプト設計 → キャラクター設計 → ネタ出し → 台本生成 → 成長まで、一気通貫で運用できます。

**フェイスレス動画とは：** 顔出しなしで AI 音声ナレーション＋テキストオーバーレイ＋B-roll 映像素材で構成される動画です。

---

## 対応媒体

| 媒体 | 形式 | 推奨尺 |
|------|------|--------|
| **TikTok** | ショート動画（縦型） | 15〜180秒 |
| **Instagram リール** | ショート動画（縦型） | 15〜90秒 |
| **Facebook リール** | ショート動画（縦型） | 15〜90秒 |
| **YouTube ショート** | ショート動画（縦型） | 15〜60秒 |
| **YouTube ロング動画** | 通常動画（横型） | 8〜30分 |

---

## 何ができるか

1. **ターゲットの悩みリサーチ** - 汎用プロンプトで悩みを100〜150個棚卸しし、12クラスターに整理
2. **コンセプト設計** - 悩みクラスターを元に「誰のどんな悩みを、どう解決する人か」を定義
3. **キャラクターの自動設計** - 心理学パラメータから「判断構造」を持つ一貫したペルソナを生成
4. **コンテンツのネタ出し** - 悩み × キャラクターの視点から、日/週/月単位で動画テーマを計画
5. **動画台本の一括生成** - 5媒体向けの台本（ナレーション・テロップ・映像指示込み）を同時に生成
6. **キャラクターの成長** - 動画の反応やトレンドを取り込み、人間のように成長する

## すぐに始める方法

### 一番カンタンな始め方

AIに **「初回セットアップを開始して」** と指示するだけ。

ターゲットリサーチ → コンセプト設計 → キャラクター設計が自動で順番に進みます。
各ステップの確認ポイントで「OK」を出すだけで、個別にスキルを呼び出す必要はありません。

以下は各ステップの詳細説明です。

### Step 1: ターゲットの悩みを調べる

AIに **「ターゲットをリサーチして」** と指示。

ジャンル・ゴール・制約を入力するだけで、100〜150個の悩みを一気に棚卸し。
12個のクラスターに整理され、そのまま動画ネタ・商品設計に使えます。

詳しい使い方は `ターゲットリサーチ_初心者マニュアル.md` を参照。

### Step 2: コンセプトを設計する

AIに **「コンセプトを設計して」** と指示。

悩みクラスターを元に「自分はどの悩みを、どう解決する人か」を一文で定義します。

### Step 3: キャラクターを作る

AIに **「キャラクターを作って」** と指示。

コンセプトに最適なキャラクターをAIが自動設計します。
MBTI・ストレングスファインダー・エニアグラムを組み合わせた一貫した判断構造を持つペルソナが完成します。

### Step 4: ネタを出す

AIに **「今週のネタを出して」** と指示。

悩みデータベース × キャラクターの視点から、動画テーマを自動生成します。
1日分・1週間分・1ヶ月分から選べます。

### Step 5: 動画台本を生成する

AIに **「この情報で動画台本を作って」** と指示し、テーマや素材を渡す。

キャラクターの「声」で、5媒体向けの台本を一括生成します。

### Step 6: 動画を制作・投稿する

生成された台本をもとに、AI音声ツール＋映像素材で動画を制作し投稿します。

### Step 7: キャラクターを育てる

AIに **「反応が良かった動画を分析して」** と指示。

動画の視聴データやトレンドを元に、キャラクターの知識と判断力が成長します。

## 実行の順序

```
初回セットアップ（1回やれば完了）
  1. SKILL_ターゲットリサーチ.md → 悩みを100〜150個棚卸し + クラスター化
  2. SKILL_コンセプト設計.md     → ポジション・差別化・コンセプト文を決定
  3. SKILL.md                   → キャラクター設計

日常運用（毎日 or 毎週）
  4. SKILL_ネタ出し.md               → テーマ決定
  5. SKILL_コンテンツ生成.md          → 動画台本一括生成
     SKILL_ショート動画台本生成.md    → TikTok/IG Reels/FB Reels/YT Shorts台本
     SKILL_YouTube長尺台本生成.md     → YouTube ロング動画台本
     → AI音声生成 → 映像制作 → 投稿

定期メンテナンス（月1回程度）
  6. 成長・進化システム → キャラクターの成長
  7. ターゲットリサーチ → 悩みデータベースの更新
```

## フェイスレス動画の構成要素

各台本には以下の要素が含まれます：

| 要素 | 説明 |
|------|------|
| **ナレーション原稿** | AI音声で読み上げるテキスト（キャラクターの声） |
| **テキストオーバーレイ** | 画面上に表示するテロップ・キーワード |
| **映像素材指示** | B-roll映像・ストック素材の内容指示 |
| **BGM/効果音** | 場面に合った音楽・効果音の指示 |
| **カット指示** | シーンの切り替えタイミングと演出 |
| **CTA** | 視聴者へのアクション誘導 |

## 動作環境

- AIアシスタント（Claude、ChatGPT、Cursor等）が必要
- ファイルはすべてMarkdown形式（.md）
- 特別なソフトウェアのインストールは不要
- Obsidianで管理すると便利（必須ではない）
- 動画制作には別途AI音声ツール（ElevenLabs等）と映像編集ツールが必要

## FAQ

**Q: キャラクター設計にどのくらい時間がかかりますか？**
A: AIとの対話で10~20分程度です。3つの質問に答えるだけで自動生成されます。

**Q: 1つのジャンルで複数のキャラクターを作れますか？**
A: はい。同じジャンルでもパラメータを変えることで、全く異なるキャラクターを複数作成できます。

**Q: フェイスレスだけでなく顔出し動画にも使えますか？**
A: はい。台本の構造は顔出し動画にも使えます。フェイスレス特有の映像素材指示を無視すれば、そのまま顔出し動画の台本として活用できます。

**Q: 心理学の知識がなくても使えますか？**
A: はい。MBTI等の知識はナレッジベースに内蔵されているので、AIが自動で最適な組み合わせを選定します。

**Q: 生成された台本をそのまま使っていいですか？**
A: 基本的にはそのまま使えるクオリティで生成されます。ただし、最終確認のステップを必ず挟む設計になっているので、必要に応じて微調整してください。

**Q: ショート動画とロング動画の使い分けは？**
A: ショート動画（TikTok/Reels/Shorts）は認知拡大・新規獲得に、YouTube ロング動画は信頼構築・ファン化・収益化に最適です。両方を組み合わせた運用を推奨します。
