# 動画台本テンプレート

各動画媒体向けの台本を、キャラクターの判断構造に基づいて生成するためのテンプレート集。

---

## 1. 共通：動画の骨格設計

### 1-1. コンテンツの5類型

すべての動画は、以下の5類型のいずれかに分類される。
キャラクターのパラメータとテーマに基づいて、最適な類型を選択する。

#### 類型A：問題提起型（常識を疑う）

```
構造：常識の提示 → 否定 → 再定義 → 新しい視点 → 行動提案
推奨尺：ショート30〜60秒 / ロング10〜15分

最適なキャラクター：
- MBTI: Te/Ti主導、Ni補助
- エニアグラム: タイプ1, 5, 8
- SF: 戦略性、分析思考、指令性

効果：権威性の構築、差別化、シェアされやすい
動画特性：フックが強烈、コメント欄が活性化しやすい
```

#### 類型B：ハウツー型（解決策を提示する）

```
構造：悩みの共感 → 原因の特定 → 解決策（ステップ形式） → 実践の後押し
推奨尺：ショート30〜60秒 / ロング10〜20分

最適なキャラクター：
- MBTI: Si/Te主導
- エニアグラム: タイプ1, 3, 6
- SF: 達成欲、責任感、規律性

効果：実用性、保存されやすい、検索に強い
動画特性：SEOに強い、YouTube ロングで最も効果的
```

#### 類型C：ストーリーテリング型（体験で語る）

```
構造：きっかけ → 苦労・失敗 → 転機 → 学び → 視聴者への示唆
推奨尺：ショート30〜60秒 / ロング8〜15分

最適なキャラクター：
- MBTI: Fi/Fe主導、Se補助
- エニアグラム: タイプ2, 4, 7
- SF: コミュニケーション、共感性、親密性

効果：共感力、信頼構築、ファン化
動画特性：ナレーションの感情表現が重要、BGM演出が効果的
```

#### 類型D：Before-After型（変化を見せる）

```
構造：Before状態の提示 → 転換点 → After状態 → 方法の解説 → 再現可能性の提示
推奨尺：ショート15〜45秒 / ロング10〜15分

最適なキャラクター：
- MBTI: Se/Te主導
- エニアグラム: タイプ3, 7, 8
- SF: 達成欲、最上志向、活発性

効果：ビジュアル的にインパクト大、TikTokでバイラルしやすい
動画特性：映像素材の質が重要、数字での対比が効果的
```

#### 類型E：リスト・ランキング型（情報を整理する）

```
構造：テーマ導入 → アイテム1 → アイテム2 → ... → まとめ → 最重要アイテムの再強調
推奨尺：ショート30〜60秒 / ロング8〜20分

最適なキャラクター：
- MBTI: Te/Si主導、Ne補助
- エニアグラム: タイプ1, 5, 6
- SF: 分析思考、収集心、学習欲

効果：保存されやすい、シリーズ化しやすい
動画特性：テロップと構造化が重要、チャプター機能と相性良い
```

### 1-2. 類型の自動選定ロジック

```
IF テーマが業界の常識に関連 AND キャラクターのエニアグラムがタイプ8
    → 類型A（問題提起型）

ELSE IF ユーザーが「ハウツー」「やり方」を求めている
    → 類型B（ハウツー型）

ELSE IF テーマが個人的な経験に関連
    → 類型C（ストーリーテリング型）

ELSE IF テーマに数値的な変化がある
    → 類型D（Before-After型）

ELSE IF テーマが「○選」「ランキング」に適している
    → 類型E（リスト・ランキング型）

ELSE
    → キャラクターの MBTI主機能 × エニアグラムから最適な類型を選定
```

## 4. フェイスレス動画の演出テンプレート

### 4-1. 映像素材のカテゴリ

```
【使用する映像素材の種類】

1. B-roll映像（ストック素材）
   - テーマに関連する風景・人物・作業シーン
   - 感情を表現する映像（喜び、困惑、達成感等）

2. テキストアニメーション
   - キーワードが動的に表示される
   - ステップ番号が順番に出現する

3. グラフ・図解
   - データの視覚化
   - フロー図、比較表
   - Before-After対比

4. アイコン・イラスト
   - 概念を簡潔に表現
   - ポイントマーカー

5. 画面録画（必要な場合）
   - ツールの使い方デモ
   - ウェブサイトの紹介
```

### 4-2. NAR音声（ナレーション）品質基準

```
【AI音声ナレーション品質基準】
- 一文は20〜30文字以内
-「。」で0.5秒の間、「、」で0.3秒の間
- 体言止めを効果的に使用
- 専門用語は直後に説明を追加
- 強調ワードは【】で指示
- 語尾パターンの統一（キャラクター一貫性）
```

## 5. 品質チェックリスト

### 全媒体共通

- [ ] 核心メッセージが明確で、ブレていないか
- [ ] キャラクターの「声」でナレーションされているか（AI感がないか）
- [ ] ターゲットの悩みに対する解決策が具体的か
- [ ] CTA（行動喚起）が含まれているか
- [ ] 音なしでも内容が伝わるか（テロップ設計）

### ショート動画用チェック

- [ ] 最初の1〜3秒で視聴者を引きつけるフックがあるか
- [ ] 推奨尺に収まっているか
- [ ] テンポが良く、無駄な間がないか
- [ ] 映像の切り替えが適切か（3〜5秒に1カット）
- [ ] 媒体別のCTAが適切か

### YouTube ロング動画用チェック

- [ ] タイトルにSEOキーワードが含まれているか
- [ ] 最初の15秒にフックがあるか
- [ ] チャプター構成が論理的か
- [ ] 視聴維持率を意識した構成か（中間フックあり）
- [ ] エンドスクリーンでのCTAがあるか
- [ ] ナレーションが口語体で自然に聞こえるか
- [ ] 映像素材が10秒以上同じものが続いていないか
