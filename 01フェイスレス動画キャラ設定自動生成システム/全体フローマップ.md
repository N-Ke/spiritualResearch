# 全体フローマップ

## システム全体像

```
+-----------------------------------------------------------------------+
|                                                                       |
|   フェイスレス動画自動生成システム - 全体フロー                           |
|                                                                       |
|   ユーザーがやること: [ジャンルを決める] → [ネタを選ぶ] → [OKを出す]      |
|   AIがやること:       台本生成までのすべて                                |
|   ユーザーがやること: AI音声生成 → 映像編集 → 投稿                       |
|                                                                       |
+-----------------------------------------------------------------------+
```

## 運用モデル：3つのパターン

### パターンA: フル自動（推奨）

全スキルを連携させる最も効率的なパターン。

```
初回:  ターゲットリサーチ → コンセプト設計 → キャラクター設計
  |
月初:  ターゲットリサーチ更新 + 月間ネタ計画
  |
毎週:  週間ネタ計画の確認・調整
  |
毎日:  ネタ選択 → 台本生成 → AI音声生成 → 映像編集 → 投稿
  |
月末:  反応分析 → キャラクター成長 → 悩みDB更新
```

ユーザーの所要時間: 初回セットアップ30~60分、台本生成は1日10~15分（確認とOK）
動画制作: AI音声生成＋映像編集に別途30~60分/本

### パターンB: テーマ直接入力

ネタ出しスキルを使わず、ユーザーが直接テーマを指定するパターン。

```
ユーザー: 「[テーマ]で動画台本を作って」
  |
  → SKILL_コンテンツ生成.md が直接実行
  |
  → 5媒体の台本を生成
```

### パターンC: 単一媒体集中

特定の媒体だけに集中するパターン。

```
ユーザー: 「TikTok台本を作って」
  |
  → SKILL_ショート動画台本生成.md を直接起動
  |
  → TikTok向け台本を生成

ユーザー: 「YouTube動画の台本を作って」
  |
  → SKILL_YouTube長尺台本生成.md を直接起動
  |
  → YouTube ロング動画台本を生成
```

## クイックリファレンス: 困ったときの対応表

| 困りごと | 使うスキル | 指示の例 |
|---------|-----------|---------|
| 初めて使う・全部やりたい | SKILL_初回セットアップ | 「初回セットアップを開始して」 |
| 誰に向けて動画を作ればいいか分からない | SKILL_ターゲットリサーチ | 「ターゲットをリサーチして」 |
| どんな発信者になるか決めたい | SKILL_コンセプト設計 | 「コンセプトを設計して」 |
| キャラクターがまだない | SKILL.md | 「キャラクターを作って」 |
| 何の動画を作ればいいか分からない | SKILL_ネタ出し | 「今日のネタを出して」 |
| テーマは決まったが台本が書けない | SKILL_コンテンツ生成 | 「この情報で動画台本を作って」 |
| TikTok台本だけ欲しい | SKILL_ショート動画台本生成 | 「TikTok台本を作って」 |
| リール台本だけ欲しい | SKILL_ショート動画台本生成 | 「リール台本を作って」 |
| YouTubeショート台本だけ欲しい | SKILL_ショート動画台本生成 | 「YouTubeショート台本を作って」 |
| YouTubeロング動画の台本が欲しい | SKILL_YouTube長尺台本生成 | 「YouTube動画の台本を作って」 |
| キャラクターがマンネリ化した | 成長・進化システム | 「キャラクターを成長させたい」 |
| 同じネタの繰り返しになった | SKILL_ネタ出し | 「切り口を変えて」 |
| フックが弱い | SKILL_ショート/YouTube長尺 | 「フックを変えて」 |
| トーンが合わない | SKILL_コンテンツ生成 | 「もっと優しく」「もっと攻撃的に」 |
