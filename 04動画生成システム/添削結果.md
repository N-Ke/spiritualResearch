## 第1回添削：テスト台本 1回目 → 2回目（融合版）

**作成日:** 2026年3月18日
**対象台本:** 2026年の年金新ルール、知らなかった夫と調べていた夫

---

### A. フィードバック（荒木の所感）

#### 良かった点

| カテゴリ | 内容 |
|---------|------|
| ナレーションによる差別化 | ナレーション主体の構成により、よくある「主人公が物語を進めていく漫画」とは異なるポジション取りができている |
| 要点のまとまり | 解説寄りのナレーションが入っていることで、要点が整理されている。ストーリーを求めて集まる視聴者に分かりやすく届く |
| 行動に移せる情報 | 「明日から市役所に行こう」と思える具体的な対処法がある。問い合わせ先の電話番号も載っており、**本当にその人の人生の手助けをしている動画** |
| 分かりやすさ | 年金に詳しくない人でも理解しやすい。具体的な計算式や「500万円の差」というインパクトの強い数字が効果的 |
| 制作しやすさ | セリフが少ない分、感情曲線に過度にこだわらなくて済む。情報提供に近い内容で制作難易度も取り組みやすい |

#### 気になった点

| カテゴリ | 内容 |
|---------|------|
| 典子のキャラクター | 「だから言ったのに」という態度が気になる。ただし「私もしっかりしなきゃ」と響く視聴者もいるはず。コメント欄での議論にもつながる可能性あり |
| 競合との比較 | 競合はストーリー重視の朗読劇スタイル。今回の台本は解説寄りの構成で、ストーリー性がやや控えめ。ただし「とっつきやすく分かりやすい内容」「視聴者のアクションを促す」という独自の強みがある |

---

### B. 荒木の自己分析（1回目→2回目で何を変えたか）

#### 発見した根本課題

> **1回目の感情曲線：グラフは忙しいのに、視聴者の感情が揺れない**

**原因：** 内容がパターン化していた

| パターン | 1回目の構造 | 問題点 |
|---------|-----------|--------|
| 失敗側→成功側 | 原田が失敗 → 長谷川が正解を教える | 3回とも同じ構造の繰り返し |
| 成功側の役割 | 「答えはこうだよ」と解説する | 解説係になっており、人間味がない |
| 正志のリアクション | 毎回「驚く」だけ | 3回とも同じ反応＝成長が見えない |

#### 実施した改善方針

**成功側もただ淡々と答えを知っているわけではなく、「自分たちも一歩間違えば失敗側だった」という描写を入れる。**

### C. シーン別の具体的修正内容

#### シーン5（保険・長谷川）── 最も大きな変更

**❌ 1回目（解説型）**
```
美代子「正志さん、1年目は任意継続よ。保険料の上限があるから、あなたの場合はこっちが安い。
2年目から国保に切り替えるわ」
正志「…お前がいなかったら、俺も何も知らないまま国保に入ってたな」
```

**✅ 2回目（葛藤＋体験型）**
```
美代子「正志さん、猶予なんてないの。任意継続は20日よ。
20日を1日でも過ぎたら、もう選べないの」（声が少し震えながら）
正志「…20日？ そんな短いのか。いや、でもまだ17日あるだろ？」（慌ててカレンダーを見る）
美代子「だから今日やるの。私が市役所に電話して国保の保険料を先に聞くから、
あなたは会社に任意継続の保険料を確認して。どっちが安いか比べましょう」
```

| 要素 | 1回目 | 2回目 |
|------|-------|-------|
| スコア | +2（学び） | +1（共感） |
| 美代子の立場 | 最初から準備済み | 市役所時代の記憶で焦っている |
| 正志の行動 | 受け身で感心 | 慌てつつも自分も動く |
| タイミング | 退職前に余裕を持って準備 | 期限3日前にギリギリ動いた |
| 視聴者の感覚 | 「正解を教えてもらった」 | 「この人も危なかったんだ…自分みたいだ」 |

#### シーン7（年金・長谷川）── 不安と確認プロセスの追加

**❌ 1回目：** 美代子がスマホで即答。不安や迷いが一切ない
**✅ 2回目：** 美代子も2時間調べて混乱。翌日年金事務所に電話して確認してようやく確信

| 要素 | 1回目 | 2回目 |
|------|-------|-------|
| スコア | +2（学び） | +1（学び） |
| 視聴者の感覚 | 「専門家の妻がいれば安心」 | 「プロでも迷うのか…でも調べれば分かるんだ」 |

#### シーン9（退職・長谷川）── 正志の成長表現

**❌ 1回目：** 正志「えっ、そんなルールがあるのか？」（3回とも同じ「驚き」リアクション）
**✅ 2回目：** 正志「分かった。今回は迷わないよ。保険の時に学んだからな。期限は待ってくれない」

| 回 | 1回目の台本 | 2回目の台本 |
|----|-----------|-----------|
| 保険（シーン5） | 感心して受け身 | 慌てるが自分も動く |
| 年金（シーン7） | 驚いて受け身 | 半信半疑だが調べることに合意 |
| 退職（シーン9） | 驚いて受け身 | 自ら「迷わない」と宣言 |

#### シーン11（教訓）── メッセージの変化

**✅ 2回目で追加された段落：**
```
じゃあ長谷川さんが特別だったかというと、そうでもないんです。
保険の手続きも、ギリギリまで動かなかった。
年金のルール変更も、美代子さんですら最初は混乱した。
年金事務所に電話して、ようやく確信が持てた。
違いは1つだけ。「分からなかったけど、調べた」。それだけなんです。
```

**メッセージの変化：**
- 1回目：「知るのに遅すぎることはない」
- 2回目：「完璧じゃなくてもいい。調べただけで違う」（行動のハードルが下がった）

### F. 次回（3回目）への修正指示

#### 修正1：ナレーション比率の削減

> **現在のナレーション比率を1とした場合、それを0.5に下げる。**
> **代わりに主人公たちのやり取りやセリフ回しを増やす。**

**現状の問題：** ほぼナレーター（杉山誠）が話しており、登場人物のセリフが少ない。

**具体的な方針：**
- ナレーションで説明していた制度の仕組みや数字を、登場人物の会話の中で自然に伝える
- 夫婦間のやり取り、感情のぶつかり合い、葛藤の表現をセリフで描く
- ナレーターは「場面転換」「視聴者への問いかけ」「感情的な橋渡し」に役割を絞る
- キャラクターの個性やリアクションがセリフを通じて伝わるようにする

**変換イメージ：**

❌ ナレーション主体（現状）:
```
【ナレーション】
原田さんは退職した翌日から「しばらくゆっくりしよう」とソファから動きませんでした。
届いた書類は机の上に積み上がっていく。典子さんが「これ、期限があるんじゃないの？」
と聞いても、「後で見るよ」の一点張り。
そして気づいた時には、任意継続の申請期限である20日間が過ぎていました。

【セリフ】（2行だけ）
典子「もう届いて3週間経ってるわよ？」
和夫「…え？ 20日？」
```

✅ セリフ主体（目指す形）:
```
【ナレーション】（短く場面をセット）
最初の分岐点は、退職直後にやってきました。

【セリフ】（やり取りで展開）
和夫「しばらくゆっくりしようかな。37年働いたんだ、1ヶ月くらい休んだっていいだろう」
典子「…あなた、この封筒の山、見てる？ もう5通くらい届いてるわよ」
和夫「ああ、後で見るよ。どうせ挨拶状か何かだろう」
典子「でもこれ、『20日以内に届け出』って赤字で書いてあるの。もう届いて3週間…」
和夫「…え？ 20日？ そんな短いのか？」
典子「あなた…もう、過ぎてるわよ」

【ナレーション】（結果を簡潔に補足）
たった20日間の怠慢で、年間13万円の差が生まれました。
```

#### 修正2：失敗側（原田・典子）のリアクションのパターン化解消

和夫の「驚き」「無知」の表現が3回とも似ている可能性があるため、シーンごとにリアクションを変化させる。

#### 修正3：典子の能動性

フィードバックで「典子も封筒を開ければよかったのでは？」という指摘あり。
典子が完全に受け身ではなく、「言おうとしたが言えなかった」葛藤を描くことで、キャラクターの能動性とリアリティを両立させる。

## 第2回添削：テスト台本 2回目 → 3回目（セリフ強化版）

**作成日:** 2026年3月18日
**対象台本:** 2026年の年金新ルール（3回目・セリフ強化版）

---

### A. 良かった点（荒木の所感）

| カテゴリ | 内容 |
|---------|------|
| セリフの増やし方 | 主人公たちのやり取りやセリフ回しの増やし方がとても良い。ナレーションで語っていた情報をセリフに変換する手法が自然 |
| ナレーション補助の構成 | 主人公だけで解説的な会話を進めると冗長になりがちだが、途中でナレーターが補助役として入ってくる構成が良い。セリフとナレーションの役割分担がうまくできている |
| 典子の能動性 | 失敗側の奥さんが無頓着ではなく、しっかり考えようとしている姿が表に出ている。「なぜそんなに無頓着なんだ」というコメントで荒れそうなポイントを防げている |

---

### B. 修正点

#### 修正1：シーン5のタイムライン矛盾（修正済み）

**問題：** 「まだ17日ある」と言っているのに「あと3日だった」と矛盾していた

| 項目 | 修正前 | 修正後 |
|------|--------|--------|
| 退職からの日数 | 退職3日目 | 退職17日目 |
| 正志のセリフ | 「まだ17日あるだろ？」 | 「もう17日経ってるじゃないか。あと3日しかないのか！？」 |
| 辻褄 | 17日残っているのに「3日前」は矛盾 | 17日経過→残り3日→ギリギリの緊迫感が成立 |

#### 修正2：成功夫婦のセリフが「友人を見下す」印象になる問題（修正済み）

**問題：** 長谷川夫婦が「原田と同じにならなくてよかった」という言い回しをしており、友人に対して「教えてやれよ」と感じる / ずる賢さ・性格の悪さが見えてしまう

**修正箇所：**

| シーン | 修正前 | 修正後 |
|--------|--------|--------|
| シーン7 | 「美代子がいなかったら、俺も**原田と同じように**断っていた。それが怖い」 | 「美代子が調べてくれなかったら、俺も**古い情報のまま**フルタイムを断っていた。**危なかった**」 |
| シーン11 | 「美代子がいなかったら、俺も**原田と同じだった**」 | 「俺たちだって、正直**ギリギリだった**よな。保険は期限の3日前だったし」 |

**変更の意図：** 原田の名前を出さず、自分たちの危うさにフォーカスすることで、「人を見下す嫌な奴」ではなく「自分たちも同じ立場だったと自覚している謙虚な夫婦」に見える

---

### C. 次回（4回目）への申し送り

- セリフとナレーションの役割分担の構成は良いバランス → 次回もこの比率を維持してよい
- 典子の能動性の描写方法 → 次回も「行動しようとするが阻まれる」パターンを活かしてよい
- 成功側キャラクターが他の登場人物を名指しで比較するセリフは避ける → 自分たちの体験にフォーカスする
- タイムライン（日数・期限）は必ず辻褄を確認すること
