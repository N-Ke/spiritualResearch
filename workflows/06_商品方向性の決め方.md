## このレポートで分かること

キャラが決まったら、**「何を売るか」の大枠**を Phase 1 で決める。建築に例えると、土地・設計思想・施工者の人柄のあとに **「どんな建物か」のビジョン** を描く段階。

読み終えると **Phase 2 で何を作るか** が見え、商品制作に着手しやすくなる（詳細設計は Phase 2）。

## なぜ Phase 1 で商品の方向性を決めるのか

**方向性なしで Phase 2 に入ると起きやすいこと**

- 「何を作るか」で迷い、停滞しやすい  
- 動画テーマと商品が噛み合わない  
- 完成後に「売れないかも」と不安だけが残る  

**方向性あり**

- 「このクラスターで Lite を作る」と着手しやすい  
- 動画 → 商品の導線を設計しやすい  
- 根拠を持って制作に集中しやすい  

### Phase 1 で決めること（ざっくり）

- どの**悩みクラスター**を商品にするか  
- **グレード**（Lite / Standard / Premium）と**価格帯の目安**  
- **動画との連動**イメージ  

### Phase 2 で決めること（詳細）

- 具体的な商品テーマ・切り口  
- 章立て・構成・本文・付属物（教材では Report 08 以降）  

## 商品テーマの選び方

Report 03 の**悩みクラスター**は、すでに「お金を払ってでも解決したい悩み」の束に近い。**ゼロからテーマを捻り出す必要はない**（教材の整理）。

### 全体の流れ

1. **Step 1** — 悩みクラスターを評価する  
2. **Step 2** — 動画 → 商品の導線を考える  
3. **Step 3** — グレードと価格帯を仮決定する  
4. **Step 4** — 商品方向性シートに書き出す  

**所要時間の目安**: 約30分 → 完了後、Phase 2 で詳細企画（教材 Report 08）

### Step 1：悩みクラスターを評価する

12 クラスターを、次の観点で見る（教材）。

|  | キャラが解決しにくい | キャラが解決しやすい |
|--|----------------------|----------------------|
| **悩みが深い** | △ 学習が要る | ★ **最初の商品の最有力** |
| **悩みが浅い** | ✕ 不向き | ○ **無料記事・短尺向き** |

**スコアリング（各5点満点・合計20点）**

1. **悩みの深さ** — 1:興味 / 3:かなり困っている / 5:切実  
2. **解決可能性** — 手順・やり方で解けるか（1〜5）  
3. **キャラとの適合** — 知識・経験（1〜5）  
4. **動画との連動** — 動画で興味を引けるか（1〜5）  

**合計の上位 2〜3** が商品候補。**最初の1商品は最高スコアのクラスターから**が無難（教材）。

### Step 2：「動画 → 商品」の導線を考える

商品は **動画を見た人** に売る。テーマがずれると興味が続きにくい。

| 層 | 役割（教材の整理） |
|----|----------------------|
| **動画（無料）** | **What** — 何をすべきか・概要 |
| **LINE（無料）** | **Why** — なぜ必要かを深める・信頼 |
| **商品（有料）** | **How** — 具体手順・テンプレ・実例 |

**良い連動の例**

- 動画：「AI 台本生成で時短する方法」（概要）  
- 商品：「AI 台本生成の完全手順」（How）  
→ 興味の延長線上  

**悪い連動の例**

- 動画：「副業の種類紹介」  
- 商品：「投資の始め方」  
→ 興味の方向がずれる  

### Step 3：グレードと価格帯を仮決定する

| グレード | 文字数目安 | 価格帯目安 | 所要目安 | おすすめ度（教材） |
|----------|------------|------------|----------|---------------------|
| **Lite** | 5,000〜10,000字 | 500〜3,000円 | 1〜2週間 | ★★★ |
| **Standard** | 15,000〜30,000字 | 3,000〜10,000円 | 2〜4週間 | ★★ |
| **Premium** | 50,000字〜 | 10,000〜50,000円 | 1〜2ヶ月 | ★ |

**まず Lite で1本作り切る**（教材の推奨）

1. 「作って売る」体験が最短で得られる  
2. 完成までが短い  
3. 価格が低く購入ハードルが下がりやすい  
4. 売れたあとに拡張できる  

**展開のイメージ（教材）**

Lite を1つ売る → 反応を見て Standard → ラインナップが育ったら Premium を検討。

### Step 4：商品方向性シートに書き出す

| 項目 | 例 |
|------|-----|
| 商品テーマ（仮） | 「副業 YouTube AI 台本生成 スタートガイド」 |
| 対象クラスター | クラスター3「台本・制作」＋クラスター6「AI 活用」 等 |
| 解決する悩み（上位3） | 1. 具体的な作り方が分からない 2. 指示の出し方が分からない 3. 品質が低い |
| グレード | Lite |
| 価格帯（仮） | 1,980円（例・税込かどうかは販売先に合わせる） |
| 動画との連動 | 動画「AI 台本の NG 行動3選」→ 商品「高品質にする手順」 |
| 一言 | 「〇〇で悩む人向け、△△を使った□□」 |

## ジャンル別の具体例（教材・価格は目安）

**副業 × YouTube**  
Lite / Standard / Premium の三段でロードマップ・マスター系（価格は例）。

**年金 × 老後資金**  
シミュレーター活用法 → 設計ガイド → ライフプラン（**金融広告・助言に該当しうる表現に注意**）。

**ダイエット × 習慣**  
短時間習慣 → 習慣設計 → 長期プログラム（**医療効果の断定は避ける**）。

**子育て × 教育費**  
シミュレーション入門 → 年齢別ガイド → トータルプラン。

**共通**: **Lite から** → 反応を見て Standard / Premium。

## 商品と「無料記事」の使い分け

12 クラスターのうち**商品にするのは 2〜3 個**が目安。残りは無料コンテンツの素材。

|  | 悩みが浅い | 悩みが深い |
|--|------------|------------|
| **解決が複雑** | 動画ネタ（短尺） | **商品化**（有料で How） |
| **解決がシンプル** | SNS Tips | **無料記事**（LINE 特典向き） |

無料で価値 → 信頼 → 商品、の流れ（教材）。

## 初心者が陥りやすい失敗と回避

### 1. 最初から高単価だけを目指す

Premium は完成が遅く、挫折・自信低下につながりやすい。**Lite で1件売れた体験**が先（教材）。

### 2. テーマが決まらない

- スコアリング（5分）  
- 上位2つに絞る  
- **より詳しい方**を選ぶ  
- それでも迷うなら**先に出た方**  
「完璧な選択」より**始める**（教材）。

### 3. 動画連動を考えない

**導線チェック（3問）**

1. このテーマで動画を**10本以上**イメージできるか？  
2. 動画後に「この商品が欲しい」と**自然**に思えるか？  
3. 動画が What・商品が How に**役割分担**できているか？  

### 4. 方向性を完璧にしようとする

- 60% → 約30分で決められる（**Phase 1 はここで十分**という整理）  
- 80% → Phase 2 の企画で  
- 95% → 市場・読者反応で  
- 100% → 決まらない  

残りは**出してから改善**（教材）。

## AI への指示方法

### 1. 初回セットアップ

「**初回セットアップを開始して**」— キャラ後に商品方向の提案フェーズへ（本ボルトの `01フェイスレス…`）。

### 2. 商品方向だけ個別実行

悩みクラスター一覧＋コンセプト＋キャラ概要を貼り、次を出力させる（教材）。

1. 各クラスターのスコア（4基準×5点）  
2. 最初の商品テーマ案3パターン  
3. 想定グレード・価格帯  
4. 動画連動イメージ（各3本）  
5. 推奨と理由  

### 3. ラインナップ全体

Lite 候補3、Standard 2、Premium 1、リリース順、アップセル・クロスセル案（教材）。

## FAQ

| Q | A（要約） |
|---|-----------|
| 最初から高単価？ | **非推奨**が多い。Lite で体験してから。 |
| 複数同時制作？ | **1つずつ**。中途半端になりやすい。 |
| 無料記事と商品？ | 無料＝What/Why の入口、有料＝How の詳細（教材）。 |

## まとめ

- Phase 1 は **方向性だけでよい**（詳細は Phase 2）  
- **クラスター評価 → 導線 → グレード → シート**  
- **Lite から**、「作って売る」体験を先に  
- **動画と商品は連動**（What → How）  
- **60% の決定で進む**
