## このレポートで分かること

**13 種類のテンプレ（型）**から商品の**骨格（目次・構成）**を組む考え方。白紙で唸る・AI に丸投げ・見切り発車・他人目次のコピーは、**型がない**と起きやすい（教材）。構成がブレたまま本文に入ると挫折しやすい、という整理。

## なぜテンプレートを使うのか

### 1. 辞書型を避ける

**辞書型（失敗しやすい例）**  
YouTube とは／AI の種類／チャンネル作成… → **百科の羅列**に近く、有料の理由が弱くなりやすい。

**ロードマップ型（ナビに近い例）**  
なぜ伸びないか → ジャンル選び → AI 台本の手順 → 公開後の1アクション → 30日計画 等、**A→B の道筋**が見える。

読者は「概要の辞書」より **どう歩けばいいかのナビ** を求めている、という主張（教材）。

### 2. 迷いを減らす

型があると「次に何を書くか」がテンプレ上で決まり、**速度と完走率**に効く（教材）。

### 3. 期待値の流れ

共感 → 期待 → 納得 → 実践 → 信頼、など**心理の流れ**をテンプレに組み込みやすい（教材）。

## テンプレ選定の3基準

### 1. ターゲットの「段階」

| 段階 | 向きやすい型（例） |
|------|---------------------|
| 問題にまだ気づいていない | 啓蒙・Why 系 |
| 問題は分かるが解決法が分からない | ロードマップ、NG 回避 |
| 解決法は知るが具体が分からない | チートシート、ケーススタディ |
| 実践しているが結果が出ない | 診断・処方、アップグレード |

### 2. 価格帯（目安・教材）

| 価格帯 | 向きやすいテンプレ | 読者の期待イメージ |
|--------|---------------------|---------------------|
| 低（〜3,000円目安） | チートシート、チェックリスト | サクッと使う |
| 中（3,000〜10,000円） | ロードマップ、NG 回避 | 体系的に |
| 高（10,000円〜） | ケーススタディ、フレームワーク | 深さ・実績 |

### 3. 自分の強み

| 強み | 向きやすいテンプレ |
|------|---------------------|
| 経験・実績 | ケーススタディ、ビフォーアフター |
| 理論・知識 | フレームワーク、比較分析 |
| ツール | チートシート、テンプレ集 |
| 失敗経験 | NG 回避、復活ストーリー |

## 主要テンプレの詳細（要約）

### ❶ ロードマップ型

「何から始めればいいか」向け。**全体像**で安心感を出す。

**構成例**  
序章（なぜ失敗してきたか）→ Step1 準備 → Step2 基礎 → Step3 実践 → Step4 改善 → 終章（30日後のあなたへ）

**注意**: Step は **7 未満**目安で圧迫を減らす。各 Step に「なぜ」も入れる（教材）。

### ❷ NG 回避型

**損失回避**（プロスペクト理論）を活かす。得より損を避けたい心理に寄せる（教材）。

**構成例**  
序章 → NG①〜③ → NG④〜⑦ → 根本原因と正しい考え方 → 今日からの正解行動 → 終章

### ❹ ケーススタディ型

**具体事例**の説得力。タイトル例に教材の「再生600回で2万円」等があるが、**再現を約束するものではない**。

**注意**: 事例 **2〜3 以上**、失敗・苦労、**根拠となる数字**（虚偽は NG）（教材）。

### ⑩ チートシート型

**コピペで使える**即効性。低〜中価格帯に相性が良いことが多い（教材）。

### ❼ フレームワーク型

**独自の枠組み**で差別化。中級以上に響きやすい（教材）。

## テンプレ選択フローチャート（教材）

```
ターゲットは初心者？
  YES → 全体像が要る？ → ❶ ロードマップ
      → 失敗が怖い？   → ❷ NG 回避
      → それ以外       → ❸ チェックリスト
  NO  → 実績・証拠が要る？
        → 複数事例 → ❹ ケーススタディ
        → 1 事例   → ❺ ビフォーアフター
      → 効率がテーマ？ → ⑩ チートシート
      → それ以外   → ❼ フレームワーク
```

## 序章で勝負の8割（教材の主張）

本編が良くても、序章で離脱されやすい。**序章の4要素**:

1. **共感** — 同じ悩みを抱えていたこと  
2. **権威** — 方法・実績（**誇大・虚偽は NG**）  
3. **約束** — 読了後の変化（**個人差・努力条件**を添える）  
4. **注意** — 魔法ではない、本気の人だけ、等 → **ふるい**と信頼の両立（教材）

## テンプレ選びの失敗例（教材）

1. **流行りだけ**で選ぶ → ターゲットの欲求とズレる  
2. **自分の得意だけ**で選ぶ → 初心者には具体や物語が要ることも  
3. **1 型に固執** → 商品ごとに型を変える（入門はロードマップ、中級は事例 等）

## 迷ったとき（教材）

| Q | 目安 |
|---|------|
| ロードマップ vs NG 回避 | スタート地点にいる → ロードマップ／途中で躓き → NG 回避 |
| ケーススタディ vs ビフォーアフター | 証拠重視 → 前者／ストーリー重視 → 後者 |
| チートシート vs テンプレ集 | 厳選が売り → 前者／網羅が売り → 後者 |

**序章**  
- [ ] 共感・権威・約束・注意（**約束は法令上安全な表現に**）  

**各章**  
- [ ] その章が**必要な理由**  
- [ ] 章末に**次にやること**  
- [ ] 具体例・ケース

| Q | A |
|---|-----|
| テンプレの組み合わせ？ | **メイン1つ＋サブ**が扱いやすい（例：ロードマップ＋各 Step の NG）（教材） |

## まとめ

1. **白紙で目次を考えすぎない** — **型に当てはめる**  
2. **辞書ではなくロードマップ（ナビ）**  
3. **序章**に共感・権威・約束・注意（表現は適法に）  
4. **ターゲット段階**から逆算して型を選ぶ  
5. **チェックリスト**で押してから執筆（所要 **30〜60 分**目安、教材）
