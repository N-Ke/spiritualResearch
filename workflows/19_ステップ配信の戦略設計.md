## 戦略設計とは（教材）

**D-BE_UNICK** の8要素を、**配信カレンダー上の「配列」**として決める工程。本文を書く**前**に、**物語の骨**を固定するイメージ。

**設計を飛ばすと起きやすいこと（教材）**

| 末路 | 内容 |
|------|------|
| つぎはぎ | 通同士の因果が弱い |
| セールス感露骨 | 信頼前にオファー、または最後の唐突オファー |
| 修正地獄 | 書き終えてから構成変更 |

**先に設計するメリット（教材）**

1. **一貫性** — 1本の道筋  
2. **各通の役割が明確** — 執筆・AI 指示が具体化  
3. **AI 指示の精度** — 「5通目は I+E 中心、約2500字」等  

## なぜ「順番」が文字数より先か（教材）

感情の**波**は**要素の順序**で作りやすい。文字数は二次。

| 悪い例 | 読者反応 |
|--------|----------|
| 1通目から E+U（実績と今すぐ） | 「誰？売り込み？」 |

| 良い方向性（一例） | 役割 |
|--------------------|------|
| 1通目 D+K（宣言＋小さな一歩） | 差別化＋初動 |
| 以降 B・E で信頼、**十分なあと** N | 反発を減らす |
| I で解決の型、C で感情、K を段階的に強く | `17`・`18` と整合 |

※以下の「売れる順番」は**タイプ・商品で調整**するテンプレ。

### 配列の一例（教材・要カスタム）

1通目 **D+K** → 2通目 **B**（＋C の伏線可）→ 3通目 **E** → 4通目 **N+U** → 5通目 **I** → 6通目 **C+予告** → 7〜8 **K**（オファー開始）→ 9〜10 **U+フォロー**（**虚偽の締切は禁止**）

## シーケンスタイプ比較（教材）

| 項目 | タイプ A：分岐後セールス型 | タイプ B：高額・長尺型 |
|------|---------------------------|-------------------------|
| 読者 | 関心が既に高い想定 | まだ浅い想定 |
| 期間の例 | 10日 | 10日 |
| 通数の例 | 10〜13 | 20〜23 |
| 価格の目安（教材） | 〜数万円帯に寄せやすい | 高額・検討が長い商品 |
| 特徴 | 短期集中 | 価値提供と教育を厚く |
| 動画 | 1〜3本連動の例 | 4本＋ウェビナー等の例 |

### タイプ A — フェーズ（教材）

1. **Phase 1（Day 1–2 / 序盤通）** — 着火：D → B → E  
2. **Phase 2（Day 3–5）** — 信頼：E・C・N・I  
3. **Phase 3（Day 5–7）** — 決断支援：U・I・K  
4. **Phase 4（Day 8–10）※任意** — 再招待：C・B・K  

### タイプ B — ざっくり3フェーズ（教材）

- **Phase 1**（序盤）：D・E・B、特典・動画・導線  
- **Phase 2**（中盤）：N・I・E・C、動画＋メール＋ワーク等  
- **Phase 3**（後半）：U・K・B・E、FAQ・声・最終 CTA（**表現は規約・景表法**）  

### どちらを選ぶか（教材）

**A** — 分岐リスト、中価格帯、説明が比較的シンプル、早期決断が自然  
**B** — 接触が浅い、高額、競合差分の説明が要る、個別・ウェビナー導線

## つまずき（教材）

1. **8要素を8通に機械分割** → **N・I は厚く**、**D・K は繰り返し可**  
2. **1通目に全部** → **1通目は D+小さな K** に集中する例  
3. **N が早すぎ** → **D→B→E** のあと  
4. **U＝煽り** → **気づき・構造**（`17`）  
5. **配列なしで AI 全文** → **人間が配列**、AI は通ごと  

---

## 迷ったとき（教材）

**配信頻度**  
- **毎日** — 熱は維持しやすい、しつこさのリスク  
- **間隔** — 消化時間は取れる、冷めるリスク  

**セールス期間**  
- **短め** — 教育が十分なとき  
- **長め** — 高額・慎重層  

**動画本数**  
- **1本** — シンプル商品・既に教育済み  
- **3本** — 中価格帯の王道イメージ  
- **4本＋ウェビナー** — 高額・個別

## まとめ（極意・教材）

1. **順番（配列）**が感情の波の核。バラ撒かない。
