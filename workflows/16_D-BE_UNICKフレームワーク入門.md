## このレポートで分かること

**D-BE_UNICK**（8要素）の全体像、**ステップ配信**の役割、**共鳴系**と煽り系の対比、タイプ A/B の骨子、設計用プロンプト例（教材）。

## はじめに

商品があっても、**信頼のない場所でいきなり URL** だけ出すと買われにくい（教材の整理）。

| アマチュアに見えやすい例 | プロの導線（教材の型） |
|--------------------------|-------------------------|
| 動画概要に直リンク | まず **LINE** 等へ |
| 初対面で売り込み | **ステップ**で心理ブロックを外してからオファー |

**ステップ配信**＝順番に送り、感情・納得のプロセスを作る仕組み。  
その中で「買うまでの心理の順番」を整理したフレームの一つが **D-BE_UNICK**（教材上の呼称）。

## なぜステップ配信か（教材）

1. **初対面では財布が開きにくい** — 未知の個人への即オファーは警戒されやすい  
2. **情報だけでは買わない** — 「分かった気」で終わり、**共鳴・オファーまでの設計**が別途必要、という主張  
3. **心理ブロックが複数ある** — 信頼・効果・今じゃなくていい・失敗恐怖 等を、**通を分けて**外していく  

## なぜ D-BE_UNICK か（教材）

| 煽り寄りのイメージ | D-BE_UNICK の整理（教材） |
|--------------------|----------------------------|
| 不安の増幅 → 今すぐ | **違いの宣言 → 共鳴 → 仲間への招待** |
| 恐怖で動かす | **共鳴・自己決定を尊重した導き** |

※ PASONA / AIDMA 等との**厳密な学術比較**は本稿の範囲外。

### 倫理原則（教材）

1. 読者を**責めない**・**恐怖だけ**で動かさない  
2. **嘘・過度な煽り**を排除  
3. **仲間・長期関係**を意識  
4. **自己決定**を尊重しつつ、選択肢を提示  

### 「セールスは悪ではない」について（教材）

「読者の課題に合う商品を、自信を持って勧める」という**姿勢の話**。**特効薬・医者**は比喩。**医療効果の暗示は別法規**に注意。

|----|------|----------------|--------------|
| **D** | Declaration | 宣言 | 「この人は違う」 |
| **B** | Benefit | 恩恵 | 得られる変化を鮮明に |
| **E** | Evidence | 証拠 | 主張の裏づけ |
| **U** | Urgency | 必然性 | 今動く理由（煽りではなく気づき寄り） |
| **N** | Nameless Enemy | 名もなき敵 | 妨げの**外部要因**を名指し |
| **I** | Innovation | パラダイムシフト | 新しい勝ち方 |
| **C** | Case Study | 体感事例 | 追体験レベルの描写 |
| **K** | Kickstart | 最初の一歩 | 小さく踏み出せる CTA |

### Phase の整理（教材上の二層）

- **Phase 1（コンセプト側）**: **D → B → E → U**（着火〜必然性）  
- **Phase 2（日々の文脈）**: **N → I → C → K**（敵の特定〜一歩）

※配信設計では**通ごとに複数要素を混ぜる**のが基本（後述）。

### 8要素と心理ブロック（教材）

| 要素 | 外しやすいブロック（例） | 読者の心の声（例） |
|------|---------------------------|-------------------|
| D | 「また同じ発信者？」 | 「この人は違うかも」 |
| B | 「で、私に何が？」 | 「これなら苦しみが終わるイメージが持てる」 |
| E | 「本当に結果出る？」 | 「理屈に合う・自分にもできそう」 |
| U | 「明日でいい」 | 「今動かないとまずいかも」 |
| N | 「自分がダメなせい？」 | 「構造や古い常識のせいだったのか」 |
| I | 「また根性論？」 | 「考え方が変わった」 |
| C | 「あなただから」 | 「同じ立場からここまでいけるのか」 |
| K | 「何から？」 | 「まずこれだけでいいのか」 |

## 各要素の要点（教材）

### D — Declaration

- **1通目に必ず**（原則）  
- スタンス・信念で**差を見せる**  
- 例の方向性: 「楽して稼ぐも苦行も否定する」「常識を疑う」「仲間として扱う」等  

### B — Benefit

- **機能的** → **体験的** → **変容的**の3層イメージ  
- 数値例（月30万等）は**ジャンル・実態に合わせ、虚偽禁止**  

### E — Evidence

カテゴリ例（教材）: データ・事例・第三者の声・権威（**出典必須**）・自己体験  

**重要**: 「327名中87%」等は**教材ダミー**。**掲載する数字はすべて検証可能に**。  

### U — Urgency（必然性）

| 避けたい例 | D-BE_UNICK 流（教材） |
|------------|-------------------------|
| 嘘の残り枠 | **内的・状況的・構造的**な「今」の理由 |
| 脅しの締切 | 気づき・文脈に基づく必然性 |

### N — Nameless Enemy

- **読者本人を敵にしない**  
- 古い常識・構造・誤った教え等を**共に倒す相手**に  

### I — Innovation

- N のあとに**解決の型**を出す（**FAQ 原則**）  
- 手順は**3ステップ以内**に圧縮しやすい、という指針（教材）  

### C — Case Study

- 時・場所・数値・会話・身体感 等で**臨場感**  
- **完璧すぎ**は逆効果 — 失敗・揺れも適度に  

### K — Kickstart

| 避けたい例 | 望ましい方向性（教材） |
|------------|-------------------------|
| 命令の購入一択 | **招待・仲間・まず一歩** |

## ステップ配信の基本構造（教材）

### タイプ A：分岐後セールス型（例：10日間）

**Phase 1（Day 1–2 目安）**  
1通目: **D + K** ／ 2通目: **D + B** ／ 3通目: **D + E** ／ 4通目: **C + B + N**  
→ 「違う・味方」の下地  

**Phase 2（Day 3–5）**  
5通目: **I + E** ／ 6通目: **U + K** ／ 7通目: **C + E**  
→ 信頼の補強  

**Phase 3（Day 5–7）**  
8通目: **I + E + B** ／ 9通目: **U + K + B** ／ 10通目: **全要素の凝縮**  
→ 決断支援  

### タイプ B：高額向けの長尺例（教材：10日・23通イメージ）

- **Phase 1**（Day 1–3 / 6通）: 価値・信頼・導線  
- **Phase 2**（Day 4–7 / 8通）: 教育・**N** の浸透 等  
- **Phase 3**（Day 8–10 / 9通）: セールス・不安解消・**構造的理由ある限定**  

### 迷ったとき（教材）

| タイプ A | タイプ B |
|----------|----------|
| 既に関心が高いリスト | 新規育成が主 |
| 低〜中価格帯のイメージ | 高価格帯のイメージ |
| 短期で決断促進 | じっくり信頼 |

価格帯は**目安**— 実商品と相談。

### 煽り系 vs 共鳴系（対比・教材）

| 煽り系に見えやすい | 共鳴系（整理） |
|--------------------|----------------|
| 恐怖一辺倒 | 共感・仲間 |
| 嘘の限定 | **構造上の**限定のみ |
| 読者否定 | **外部の敵**を指定し解放 |
| 上から教える | **一緒に歩く** |

## つまずき（教材）

1. **有益情報の送りすぎ** → How は商品内。**Why / What** をステップで  
2. **初日から売り** → 前半は **D・B** 中心、オファーは信頼後  
3. **読者責め** → 敵は**外**  
4. **嘘の緊急** → 構造的理由のある限定のみ  
5. **1通1要素だけ** → **2〜3要素を自然に混ぜる**

## FAQ（要約）

| Q | A（教材） |
|---|-----------|
| 何通？ | 低〜3万円目安で5〜7通、中帯で10〜13、高額で15〜23 等（個別調整） |
| 8要素は順番厳守？ | **厳密順は不要**。**D は1通目**、**N のあと I**、**K は段階的に強度** 等の原則 |
| LINE 以外？ | Lステップ・メルマガ・DM 等でも**考え方は流用可**（ツール仕様は別） |
| 文字数？ | 800〜2,500字目安（通の役割で変動） |
| 必須？ | 有料をリスト経由で売るなら**ステップ型が効率的**、という主張 |
