## このレポートで分かること

キャラクター設計は、フェイスレスで**他チャンネルと差がつきやすい要因**の一つ。心理学系フレームで **「判断構造」** を決め、AI に **3つの質問**に答える流れで、一貫した**独自キャラ**を設計する考え方が身につく（本ボルトでは `01フェイスレス…` の初回セットアップで実行）。

## なぜキャラクターが必要か

顔出し配信は「顔」と「声」で記憶に残りやすい。フェイスレスはその分、**別の軸で独自性**が要る。

| 課題 | 内容 |
|------|------|
| ① | 顔が見えず「誰の動画か」区別しにくい |
| ② | 同じ AI・同型テンプレだと**他と似る** |
| ③ | テンプレ量産は **Inauthentic（信頼性）** として問題視されうる |

**整理（教材）**: フレームに基づくキャラ設計 → **一貫した判断・語り口** → どの動画でも「このチャンネルだ」と分かりやすくなる、という狙い。

### コンテンツの差（例）

**キャラなし（誰が書いても同型になりやすい）**

「年金の種類は3つです」「iDeCo の始め方を解説します」

**キャラあり（視点が染み込む例）**

「12年やってきて痛感したのは、落とし穴が制度より**思い込み側**にあることです」  
「iDeCo の前に、**自分のリスク耐性**を知らないと後悔しやすい理由があります」

## キャラクター設計の仕組み（3レイヤー）

「適当なキャラ名」だけでなく、**判断の癖**まで決めると一貫しやすい、という整理。

```
レイヤー1：思考パターン（MBTI）
  「どう考え、どう判断するか」→ 16タイプから1つ（AIがコンセプトに合わせて提案することも）
    ↓
レイヤー2：強み（ストレングスファインダー）
  「何が得意で、何に価値を置くか」→ 34資質から Top5
    ↓
レイヤー3：根底の動機（エニアグラム）
  「なぜそう動くか、何を恐れるか」→ 9タイプから1つ
```

**狙い（教材）**

- 一貫した**判断構造**  
- テーマが変わっても**その人らしい**言い方  
- 台本生成時に **同じ「声」** を再現しやすい  

### 1. MBTI（16タイプ）— 思考の傾向

| 軸 | タイプA | タイプB |
|----|---------|---------|
| エネルギー | E（外向） | I（内向） |
| 情報 | S（感覚・事実） | N（直感・可能性） |
| 判断 | T（論理） | F（感情・価値） |
| 外界との向き | J（計画） | P（柔軟） |

**ジャンル別の相性例（教材・あくまで例）**

- 副業×効率化 → INTJ 的な「戦略・効率」  
- 健康×習慣 → ISFJ 的な「丁寧・守る」  
- 投資×データ → ISTJ 的な「正確・信頼」  
- 人間関係×共感 → ENFJ 的な「人を動かす」  
- クリエイティブ → ENFP 的な「発想・熱量」  

**自分でタイプを暗記する必要はない**（教材）。コンセプトに合わせ AI が提案する流れを想定。

### 2. ストレングスファインダー（Top5）

「何が得意で、何に価値を置くか」を Top5 で固定し、**台本・商品の語り**に反映する。

**4ドメイン（教材の分類）**

- **実行力**: 達成欲、アレンジ、信念、公平性、慎重さ、規律性、目標志向、責任感、回復志向 等  
- **影響力**: 活発性、指令性、コミュニケーション、競争性、最上志向、自己確信、自我、社交性 等  
- **人間関係構築**: 適応性、運命思考、成長促進、共感性、調和性、包含、個別化、ポジティブ、親密性 等  
- **戦略的思考**: 分析思考、原点思考、未来志向、着想、収集心、内省、学習欲、戦略性 等  

**活用例（教材）**  
Top5：戦略性、学習欲、着想、個別化、分析思考  

- 台本：「データを見ると、多くの人が見落としているパターンが…」（分析＋着想）  
- 商品：「一般論ではなく、あなたの状況に合わせたやり方」（個別化）  

### 3. エニアグラム（9タイプ）— 動機と恐れ

| タイプ | 名称（一般的呼称） | 核心（教材） |
|--------|-------------------|--------------|
| 1 | 改革者 | 正確さ・正義 |
| 2 | 支援者 | 助け、感謝 |
| 3 | 達成者 | 成果・承認 |
| 4 | 個性派 | 独自性・理解 |
| 5 | 観察者 | 知識・理解 |
| 6 | 忠実者 | 安全・安定 |
| 7 | 楽天家 | 自由・楽しさ |
| 8 | 挑戦者 | 力・自己統制 |
| 9 | 調停者 | 平和・調和 |

**タイプ5 のコンテンツへの影響例（教材）**

- 動機：世界を理解したい  
- 恐怖：無能・無力に見られること  
- 発信：調べてから語る、曖昧が嫌い、「裏側・本質」に熱が入る → 「ちゃんと調べている」印象  

## あなたがやること：3つの質問

AI は [`04_コンセプト設計.md`](04_コンセプト設計.md) のコンセプト情報を前提にしやすい（初回セットアップ時）。

1. **どんなジャンルで発信しますか？**（Phase 1 で決めた内容の確認）  
2. **どの媒体ですか？**（YouTube / TikTok / Instagram 等）  
3. **こだわりはありますか？**（なければ「特になし」で可）  

### AI が出力しやすい成果物（教材）

- MBTI タイプと根拠  
- SF Top5 と根拠  
- エニアグラムタイプと根拠  
- **判断構造**（10項目以上：情報との向き合い方、問題時、アドバイス時、反対意見時、目標設定時 等）  
- ビジネス判断構造（収益モデル・リスク・協働の考え方 等）  
- 媒体別の表現の違い  
- **AI 実行用プロンプト**（台本生成に貼る用）  

**所要時間の目安**: 約10〜20分（対話含む）

## キャラクター設計書の構成（完成形イメージ）

| パート | 内容 |
|--------|------|
| 基本情報 | 名前、MBTI、Top5、エニアグラム、概要 |
| 判断構造 | 場面別の考え方・決め方（10項目以上） |
| ビジネス判断 | 収益・リスク・運営の哲学 |
| 媒体別表現 | ロング / Shorts・TikTok / Reels 等 |
| 口癖 | よく使う表現・**使わない**表現（各10個以上が理想） |
| AI 用プロンプト | 台本・執筆時に読み込ませる全文 |

### 完成例（教材・架空）

- **名前**: ウラアド（例）  
- **MBTI**: INTJ  
- **Top5**: 戦略性、学習欲、着想、個別化、分析思考  
- **エニアグラム**: タイプ5w6（教材表記）  
- **概要**: 裏側から仕組みを解析し、戦略で「稼ぐ仕組み」を設計する研究者型  

**判断構造（抜粋）**

1. 情報と向き合う → 事実優先  
2. 問題 → 構造分解して本質  
3. アドバイス → 個別最適  
4. 反対意見 → データで再検証  
5. 目標 → 長期戦略から逆算  

**口癖例**  
使う：「ここが"裏側"なんですが…」「データを見ると、実は…」  
使わない：「絶対に稼げます！」「とにかくやれば分かる」（景表・信頼リスク）  

## キャラクターが重要な理由（3つ）

### 1. Inauthentic 対策（教材の整理）

テンプレ羅列だけだと「誰が書いても同じ」になりやすい。  
キャラの**フィルター**で言い回し・価値判断を統一すると、**オリジナルの語り**として認識されやすい、という狙い。

### 2. 視聴者の固定化（ファン化のイメージ）

1本目 → 視点が新鮮  
数本 → 考え方が一貫している  
さらに → 切り口が好き → 登録  
LINE → もっと知りたい  
商品 → この人のなら、という信頼  

**一貫性が鍵**（教材）。

### 3. 全 Phase で「声」を統一

| Phase | 使う場所 |
|-------|----------|
| 2 | 商品テキストの語り口 |
| 3 | ステップ配信、LP のトーン |
| 4 | 台本、サムネの世界観 |

同じ声 → ブランド認識 → 信頼の蓄積（教材の因果イメージ）。

## 初心者が陥りやすい失敗と回避

### 1. キャラをスキップする

- 表現がテンプレ寄りになりやすい  
- チャンネル記憶に残りにくい  
- 台本・商品・配信でトーンがバラつきやすい  

**教材の主張**: 短時間の設計で**寿命・一貫性**に効く。スキップのリスクが大きい、という整理。

### 2. 自分と真逆を演じる

- ❌ 内向が無理に超明るいキャラ → **続かない**  
- ✅ 内向が「静かに深掘りする」キャラ → **自然で強みになりうる**  

**鉄則（教材）**: **自分の性格の延長線上**に置く。「性格を踏まえて設計して」と AI に伝えてよい。

### 3. 凝りすぎる

**上限目安：最大30分**（教材）

- 質問回答：約5分  
- 出力確認：約10分  
- 修正指示：約10分  
- 保存：約5分  

Phase 4 で動画を出しながら微調整。**最初から完璧不要**。

### 4. 設計書を使わない

- 台本生成 → **設計書 or AI用プロンプトを添付**  
- 商品・ステップ・LP → **同じトーン**で書く  

作って放置すると意味が薄い。**毎回数十秒の添付**で再現性が上がる（教材）。

## AI への指示方法

### 1. 初回セットアップ（推奨）

「**初回セットアップを開始して**」— リサーチ・コンセプトの後、キャラ設計フェーズへ。3問に答える流れ。

### 2. キャラだけ実行

コンセプトシート（Report 04）＋ジャンル＋媒体＋こだわりを渡し、MBTI・Top5・エニアグラム・判断構造・口癖・AI用プロンプトを出力させる（教材の長文プロンプト参照可）。

### 3. 既存キャラの微調整

設計書全文を貼り、「口癖を□□寄りに」「トーンを△△に」等、差分だけ指示して全文再出力。

## FAQ

| Q | A（要約） |
|---|-----------|
| 心理学の知識は？ | **不要**（教材）。3問＋確認が中心。 |
| 途中で変える？ | **口癖・トーンの微調整はいつでも可**。MBTI や根の性格の**大幅変更**は Phase 1 内がコスト低。Phase 2 以降は作り直し大。 |
| 自分の MBTI で作る？ | **自分に近い／理想に近い**が続けやすい。真逆は演じ疲れしやすい。分からなければ AI に任せる。 |
| 実在人物のモデル？ | ✅ **雰囲気・タイプ**の参考は可。❌ **口癖・キャッチの盗用**は NG（オリジナリティ・法的リスク）。 |
| コンセプトとの関係？ | コンセプト＝**何をするか（What 系）**、キャラ＝**誰が語るか（Who 系）**。揃うと差別化が伝わりやすい。 |

## まとめ

- キャラ＝**独自性・一貫した声**の源泉（フェイスレスでは特に）  
- **MBTI**＝考え方、**Top5**＝強み、**エニアグラム**＝動機と恐れ  
- **3問**で設計し、**設計書を AI に読ませ続ける**のが再現のコツ  
- 規約・判定は変わりうる。**誇大・コピー・虚偽**は避ける
