## N — Nameless Enemy（名もなき敵）

### Why（教材）

自己責めループに「**あなたは悪くない**」「**本当の障害はここ**」と**概念化**して見せる。**読者本人を敵にしない**。

### 心理効果の整理（教材・比喩）

- 理由づけによる安心、自己責めからの距離、**行動しやすさ**、発信者との**共闘**のニュアンス、エネルギーの**外向き**の転換、等

### 弱い N vs 強い N（教材）

| 弱い N | 強い N |
|--------|--------|
| 根性不足と言い切る | **古い常識・方法・圧力**を敵に命名し、読者と並んで叩く |

### 敵の3カテゴリ（教材）

1. **構造** — 仕組み・業界の前提  
2. **方法** — 初心者に合わないやり方の押し付け 等  
3. **社会・刷り込み** — 「正解は一つ」系の圧力 等  

### 配信例の骨子（教材）

**件名例**: 「あなたが動けない理由は、あなたのせいじゃない」

**展開**: 意志の弱さを否定 → **誰が言い始めたか分からない前提**が足を止めている → それを**名もなき敵**と呼ぶ → 自分は疑って実験した → あなたにもできる、という**共闘**。

### 鉄則（教材）

- 敵は**個人名にしない**（**概念・常識・構造**）  
- 読者を責めない・**過去の努力を全否定**しない  
- **恐怖だけで終わらない** → 続けて **I** で道を示す  

### つまずき（教材）

1. 敵が曖昧 → **「○○の罠」**のように命名  
2. 読者責めに見える  
3. **特定人物・サービス攻撃**  
4. N のあと **I がない**  

### AI プロンプト（N）

```
N（名もなき敵）を作成してください。

■ 前提
・ターゲット：{ }
・ジャンル：{ }
・悩み：{ }

■ 条件
1. 敵は個人ではなく古い常識・構造
2. 「あなたは悪くない」を含める
3. 敵に具体的なラベル（「○○の罠」等）
4. 共闘のニュアンス
5. 200〜300字程度

■ 出力
・見出し（敵の名前）
・本文
```

## I — Innovation（パラダイムシフト）

### Why（教材）

N のあと「**じゃあどうする？**」に答える。**A ではなく B だった**という**認知の更新**で熱量を上げる（教材）。

### 弱い I vs 強い I（教材）

| 弱い I | 強い I |
|--------|--------|
| 誰でも言う一般論 | **常識の逆**・**視点の移動**・**手順の圧縮**がはっきり |

### 3テンプレ（教材）

1. **逆転型** — 順番・前提の入れ替え  
2. **視点転換型** — みんなが見る所 vs 本当に効く所  
3. **シンプル化型** — **3ステップ以内** 等  

### 配信例の骨子（教材）

**件名例**: 「"信頼の形"を変えるだけで、勝ち方が変わる」

**展開**: 旧フロー（属人前提）→ 非属人では使えない → **新フロー**（導線・実験結果の具体性）→ 核心は**人への信頼 vs 情報への信頼**の違い、と1行で要約。

### 鉄則（教材）

- **3ステップ以内**に圧縮しやすい  
- **N で出した敵**を倒す話として**接続**  
- **自分にもできそう**に寄せる  
- 根拠は **E** と連携（空振り防止）  

### つまずき（教材）

1. 新しく聞こえない  
2. 手順が多すぎる  
3. 根拠ゼロの断定  
4. **N と話がつながらない**  

### AI プロンプト（I）

```
I（パラダイムシフト）を作成してください。

■ 入力
・N で設定した敵：{ }
・あなたの新しい勝ち方（要点）：{ }
・従来の常識：{ }

■ 条件
・「実はAではなくB」の型を1つ以上
・手順は3以内を目安
・N と接続文を明示
・誇大・根拠なし数値は禁止

■ 出力：件名案3つ、本文 400〜700字
```

## C — Case Study（体感事例）

### Why（教材）

理屈だけより**物語**で防衛を下げ、感情と**希望**に触れやすい（教材）。

### 弱い C vs 強い C（教材）

| 弱い C | 強い C |
|--------|--------|
| 実績の箇条書き自慢 | **Before / 転機 / After** と**感情・場面** |

### V字構造（教材）

**Before**（どん底を具体的に）→ **Turning Point** → **After**（変化と確信）→ **Message**（あなたも可能、**過度な約束は避ける**）

### リアリティ5要素（教材）

日時・場所・数値・会話・身体反応 — **すべて事実または創作の明示**（創作なら**フィクション表現**や**合成しない**運用方針を決める）。

### 配信例の骨子（教材）

**件名例**: 「概要欄を変えた翌日に起きたこと ── 実話です」

**展開**: Before（長年の稼ぎ方と内的な不安）→ 転機（概要欄に気づき・文言変更）→ 小さな数字から始まる驚き → After（件数・率・小さな販売・**得た確信**）。※数値は**サンプル**。

### 鉄則（教材）

- **絶望パート**を薄くしない  
- **完璧すぎ**ない（小さな失敗・違和感）  
- 読者と**接点のある土台**から始める  

### つまずき（教材）

1. 実績羅列  
2. Before が弱い  
3. 嘘っぽい完璧さ  
4. 読者と**あまりに遠い**特別すぎる設定だけ  

### AI プロンプト（C）

```
C（体感事例）を作成してください。

■ 入力
・主人公の初期状態：{ }
・転機：{ }
・結果（事実ベースで）：{ }
・読者との共通点：{ }

■ 条件
・V字構造
・五感・会話・数値のうち2つ以上
・誇大・未検証の数字禁止
・最後に希望（押し付けない）

■ 出力：件名案3つ、本文 600〜1000字
```

## K — Kickstart（最初の一歩）

### Why（教材）

感情が上がっても**次の一歩が重い**と止まる。**極小の行動**と**招待**で動かす。

### 弱い K vs 強い K（教材）

| 弱い K | 強い K |
|--------|--------|
| 毎日投稿から始めよう | **数秒で終わる**具体アクション＋**報酬の明示**（規約内） |

### CTA の型（教材）

- **命令型**は避け、**招待型・仲間型**  
- **自己決定**の余地を残す  

### マイクロステップのレベル（教材）

レベル1 クリックだけ → 2 登録・DL → 3 短い入力 → 4 読んで判断 → 5 申込等の決断。**序盤は 1〜2**、後半で上げる、という目安。

### 配信例の骨子（教材）

**件名例**: 「一緒に実験する場所を、作りました」

**展開**: ここまでの共感 → **募集と理由（構造上の人数制）** → 気になるだけなら無理に、と**逃げ道** → 進みたい人は**まずページを読む**、までを K に。

### 鉄則（教材）

- ハードルを**極小**に（「2文字返信」等は**運用・規約で可能か確認**）  
- 行動に対する**見返り**（特典等）を**正直に**  
- **命令でなく招待**  
- **最終判断は読者**  

### つまずき（教材）

1. 最初からハードル最大  
2. 報酬の説明がない・虚偽  
3. 命令口調一辺倒  
4. **プレッシャーのみ**  

### AI プロンプト（K）

```
K（最初の一歩）を作成してください。

■ 入力
・今通の目的（クリック／返信／閲覧 等）：{ }
・読者にしてほしい具体行動（最小）：{ }
・用意できる報酬（事実）：{ }

■ 条件
・招待・仲間口調
・レベル1〜2相当の易しさを序盤想定
・命令・虚偽の限定禁止
・「無理なら不要」と逃げ道

■ 出力：件名案3つ、本文 300〜500字
```

## 配信順の目安（教材）

**厳密な N→I→C→K 一列は不要**。例：

`1〜2通 D+B` → `3〜4通 N+I` → `5〜6通 E+C` → `7通目以降 K＋オファー`

各通 **2〜3要素**の混ぜ方が自然、という説明（`16` と整合）。
