## 視聴維持率とフェイスレス（教材）

YouTube では**視聴維持率**が配信に効きやすい、という整理。フェイスレスは画の変化が少なめになりがちなので、**台本（構成・感情の動き）**の比重が大きくなりやすい、という**経験則**（「9割」は**強い言い方の例**）。

## なぜストーリーか（教材）

- **情報の羅列**だけだと、有益でない限り**早い段階で離脱**されやすい。  
- **物語形式**は共感・記憶に残りやすい、という**説明の仕方**が多い（脳科学の**細部は専門文献で確認**）。

### 比較の例（教材・再現性は保証しない）

| 指標 | 例：ストーリー性のある台本（複数本の平均イメージ） | 例：情報羅列中心（少数本のイメージ） |
|------|--------------------------------------|--------------------------------|
| 再生 | 高めに出た事例 | 低めに出た事例 |
| 要因（例） | 転落→反省→成長 等の起伏 | フラットな説明 |

**18.3 倍**のような比較は**教材で使われた数値の例**。**自分のチャンネルでは検証**すること。

## 動画ブロックと D-BE_UNICK の**イメージ**（厳密な1対1ではない）

本稿では台本の**流れ**を説明するために、ステップ配信の **N・I・K**（[`18`](18_8要素の詳細解説_後編.md)）に**近い役割**で触れる。**公式の8要素設計図の置き換えではない**。

| 動画ブロック（目安の尺） | 役割 | 8要素との**イメージ** |
|--------------------------|------|------------------------|
| 冒頭フック（0〜30秒） | 指を止める | 注意の取り方（**D**の「違い」に繋げることもあるが**別物**） |
| 原因と共感（〜数分） | 苦しさの理由・「敵」の可視化 | **N（名もなき敵）**寄り |
| 新しい武器（中盤） | **What** の方向・構造化（How 全公開はしない） | **I（新しい勝ち方）**寄り |
| CTA（ラスト〜30秒） | **LINE** へ一点集中 | **K（最初の一歩）**寄り |

## ストーリー台本の「黄金の型」（教材）

| セクション | 時間の目安 | 目的 |
|------------|------------|------|
| 1. 冒頭フック | 0〜30秒 | 「最後まで見たい」と思わせる |
| 2. 原因と共感 | 30秒〜3分前後 | 悩みの構造・共感 |
| 3. 新しい武器 | 中盤 | 解決の**方向**（What）をストーリーで |
| 4. CTA | 最後30秒前後 | **続きは LINE／特典**（How の受け渡し） |

### 構成のウェイト例（1本〜10分前後のイメージ・調整可）

```text
冒頭フック     … 短く強く
原因と共感     … 中盤手前まで
新しい武器     … ボリューム最大
CTA            … シンプルに一点
```

※ **5% / 15% / 60% / 10%** の図は**あくまで目安**。尺・ジャンルで変える。

### 感情曲線（教材）

- **下降 → 気づき → わずかな希望** を含めやすい。  
- **完全ハッピーエンド**だけにすると「もう用はない」で離脱されうる → **余白**を残す設計も選択肢。  
- **自分ごと**の接点を**複数**置く。

## 10 種類のストーリー型（早見表）

**詳細・使い分けの正本**は `03フェイスレスストーリー台本システム/01_成功パターン構造ルール.md`。

| 型 | 向く動画 | キーワード |
|----|----------|------------|
| **A** 転落と気づき | 反面教師・教訓 | 一人の転落物語 |
| **B** ハイブリッド N 選 | NG・リスクの列挙＋ストーリー | リスト＋物語 |
| **C** 比較群像劇 | 正解が一つに決まらない話 | 複数人の岐路 |
| **D** 対比構造 | 成功 vs 失敗・習慣の差 | 二つの人生線 |
| **E** メンター | 習慣・マインドの変化 | 師匠／気づき |
| **F** スカッと逆転 | 搾取・権利・防衛 | 逆転劇 |
| **G** 再起・成長 | どん底からの復帰 | 再起 |
| **H** 選択の岐路 | 重大な決断 | 分岐点 |
| **I** 連鎖崩壊 | 放置のリスク | ドミノ型 |
| **J** 救いの手 | 孤立からの支援 | つながり |

**選び方の目安（教材）**: 「やめた方がいい」→ A/B。「こうすれば」→ E。「正解なし」→ C/D。**迷ったら D を試す**、は**経験則の一例**。AI に **auto** で型任せも可（`03` の手順に従う）。

### 型ごとのフック例（教材・事実性はジャンルで検証）

- **A**: 「リボ払い月〇万から始まった地獄。気づいたら借金が〇〇万…」（数値は**捏造禁止**・リサーチベース）  
- **B**: 「40代を過ぎてから絶対にやってはいけない生活習慣5選」  
- **C**: 「転職か残留か。同期〇人が選んだ道と、5年後の現実」  
- **D**: 「同期入社の2人。10年後、片方は〇〇、もう片方は…」  
- **E**: 「『食べないダイエット』をやめたら、なぜか〇ヶ月で〇kg…」（**誇大・景表に注意**）  
- **F**: 「上司に『お前の代わりはいくらでもいる』と言われたら？」  
- **G**: 「今、彼は毎朝〇km走っている。1年前はベッドから起き上がれなかった」  
- **H**: 「年収〇万の安定か、やりたいことで年収〇万か」  
- **I**: 「たかが肩こりと思っていた。半年後に…」（**医学表現は慎重に**）  
- **J**: 「70歳の彼女の電話帳には番号が3つしかなかった。でも今は…」

## 冒頭フック（0〜30秒）— 最重要（教材）

- **挨拶から入らない**「こんにちは、今日は〜」は**この型では避ける**（AI 出力なら**即削除**）。  
- **結末先見せ・問いかけ・逆説・格差・警告**などで**好奇心**を取りにいく。

| テクニック | 説明 | 例（型） |
|------------|------|----------|
| 結末先見せ | 悲惨 or 希望のピークを先に | 病院のベッド、通知ゼロ… |
| 直接問いかけ | 視聴者に問う | もし〇〇と言われたら？ |
| 逆説 | 直感とズレ | やめたら痩せた… |
| 格差 | 同スタートの分岐 | 10年後の対比 |
| 警告 | リスクの自覚 | カウントダウンが始まっているかも…（**煽りすぎ・虚偽は禁止**） |

## 本編：ナレーションと会話の比率（教材）

| プリセット | ナレーション:会話 | 向き |
|------------|-------------------|------|
| 会話重視 | 2:8 | ドラマ性（夫婦・職場等） |
| **ストーリー標準（デフォルト例）** | **3:7** | 汎用 |
| バランス | 5:5 | 制度・解説寄り |
| 語り手 | 7:3 | ドキュメンタリー風 |

**迷ったら 3:7**。**±5%** 程度のブレは許容し、読み上げ感を調整。

### 感情設計のルール（要約）

- 下降→気づき→わずかな希望。  
- 説教は**長引かせない** → 失敗と結果で**気づかせる**。  
- クライマックスは**身近な一言**など具体で。

## CTA（LINE 誘導）— メインのお願いを一つに（教材）

このボルトの導線では **LINE 登録（LP 経由）**を主ゴールにしやすい。**チャンネル登録・高評価・コメント・LINE**を**同列で長く並べる**と、**主要行動が分散**しやすい、という整理。

- **推奨**: 「概要欄の LINE から〇〇を受け取る」**一本**に寄せる。  
- **必須の開示・利用規約上の表現**は**公式に従い**、本稿の指針と**両立**させる。

## 良い台本 / 弱い台本（教材）

| 観点 | 弱い | 良い |
|------|------|------|
| 冒頭 | 説明・挨拶から | **感情・異常・問い**から |
| 構成 | 1・2・3の平坦羅列 | **転機・危機・起伏** |
| 教訓 | 説教 | **体験の因果**で伝える |
| CTA | お願いが多い | **LINE への一点**（＋規約上必要なもの） |

## つまずき（教材）

1. **ターゲット過絞り** → [`24`](24_ネタ出しの方法.md) の幅に戻す。  
2. **当たり前論** → 「え、そうなの？」を複数。  
3. **主人公が嫌い** → 序盤に弱さ・動機・反省の余地。  
4. **説教過多** → 説得パートの比率を下げ、**場面**で見せる。  
5. **非現実な数字** → リサーチ可能なレンジに（**景表・誇大禁止**）。

### 長尺 vs ショート（教材）

| 項目 | 長尺（例：20〜25分） | ショート（例：8〜12分） |
|------|----------------------|-------------------------|
| 向くテーマ | 転落・群像・深い共感 | 単純な対比・1テーマ |
| 制作コスト | **高め（一例：教材に〇〇円と出ることがある）** | **低め** |
| 使い方 | 本線向け | テスト向け、という割り切りも可 |

## 品質チェック（クイック・教材）

**Critical（1つでも NG なら見直し）**

- [ ] ターゲットが**極端に狭くない**（**70%**は [`24`](24_ネタ出しの方法.md) と同様**目安**）  
- [ ] 「え、そうなの？」が**複数**  
- [ ] **ストーリー**で包んでいる（羅列だけでない）  
- [ ] 序盤で主人公に**共感・応援**の余地  

**High**

- [ ] 説教が**長すぎない**  
- [ ] 数値・設定が**リアル**（根拠または一般的レンジ）  
- [ ] ナレ:会話が**設定に近い**  
- [ ] **キャラボイス**が一貫  

**Medium**

- [ ] 肩書き・実績の**誇大**がない  
- [ ] 誤字・不自然な固有名
