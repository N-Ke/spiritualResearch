## 特典とは（教材）

第1話で渡す **LINE 登録のお礼＋価値の最初の証明**。マーケ用語では **Lead Magnet（リードマグネット）**。

| 特典なし | 特典あり |
|----------|----------|
| 「登録してね」だけ | **具体的なベネフィット**が見える |

## Why（教材）

1. **YouTube 等は集客装置** — そこだけでは「本気モード」に乗りにくい。**LINE まで来た人**は行動コストを払っている。  
2. **リストは資産** — 連絡可能・再配信・他導線につなげやすい（**プラットフォーム規約遵守**）。  
3. **登録の心理的コスト**を、**特典の価値**で上回らせる。  

### 「無料だから適当」が危ない理由（教材）

特典は多くの場合、**あなたの価値の初接触**。ここで「しょぼい」と判断されると、**ステップが良くても回復が難しい**、という主張（教材）。

---

## 悪い特典 vs 良い特典（教材）

| 観点 | 悪い方向 | 良い方向 |
|------|----------|----------|
| 中身 | 検索コピペの寄せ集め | **体験・独自手順・一次情報** |
| 具体 | 精神論だけ | **手順・テンプレ・チェック** |
| 使えるか | 読んで終わり | **今日から1アクション** |
| 見た目 | 素のテキストのみ | 体裁・表紙・図（**誇大表現は避ける**） |
| 商品との関係 | 無関係 | **本商品の前編・プレビュー** |

## 4条件（教材）

1. **即効性** — すぐ試せる  
2. **具体性** — 手順・型がある  
3. **専門性** — あなただから出せる視点（**虚偽の権威は禁止**）  
4. **希少性** — 「ここで配っている」理由が自然（**嘘の限定は禁止**）  

「有料でもおかしくない密度を、方針として無料にする」は**戦略の一つ**（**コストと法的表現**は自己責任で調整）。

## 形式と効果（教材・目安）

| 形式 | 難易 | 登録・信頼への寄与（印象） | 向き |
|------|------|----------------------------|------|
| チェックリスト | 低 | 登録しやすい／実践しやすい | 手順系 |
| テンプレ | 中 | 高 | 作成系 |
| PDF レポート | 中 | 専門性の見せ方 | まとめ系 |
| 動画 | 高 | 人柄・実演 | スキル実演 |
| ツール・計算 | 高 | 差別化しうる | 数値設計系 |

**迷ったら** — チェックリスト or 短い PDF（教材）。

### ジャンル例（教材）

| ジャンル例 | 特典の方向性 |
|------------|----------------|
| 副業・動画 | テンプレ・企画シート |
| 健康・習慣 | チェックリスト（**医療効果の断定は避ける**） |
| 投資・お金 | レポート（**勧誘・断定的リターン表現に注意**） |
| 恋愛・対人 | チェックリスト・ワーク（**ターゲットに配慮**） |
| 学習・資格 | 計画テンプレ |

## ゼロから作らない（教材）

### 1. Phase 2 の**無料記事**を流用

`02オンラインテキスト商品化` で作った**無料記事**（教材では Step 0.5 相当）は、**教育・導線**目的で作られていることが多い → **タイトル・見出し・PDF 化・表紙**で「特典パッケージ」にする（所要は個人差、**30分は目安**）。

### 2. [`14_付属物制作ガイド.md`](14_付属物制作ガイド.md) の**付属物の一部**

完全版の**抜粋10項目**など、「**単体でも使えるが、続きが欲しくなる**」切り口（教材）。

### 3. ミニ新規

- **1悩みにフォーカル**  
- 長さより**実行しやすさ**  
- 詰め込みすぎない（読まれない）  

**ボリューム目安（教材）**

| 形式 | 目安 |
|------|------|
| チェックリスト | 15〜30 項目（過多にしない） |
| テンプレ | 3〜5 パターン |
| PDF | 5〜15 ページ程度のイメージ |
| 動画 | 10〜30 分程度のイメージ |

## 導線の全体像（教材）

`YouTube` 等 → **LP**（次工程）→ **LINE 登録** → **第1話（感謝＋特典＋使い方）** → ステップ → 商品。

---

## 見せ方・パッケージ（教材）

- **Canva** 等でアイキャッチ（正方形投稿サイズ等）  
- **具体ページ数・項目数**は中身と一致（**盛らない**）  
- **「円相当」**は**原価・市場価格の根拠**が取れるときのみ  
- **限定**は**本当に限定**のときのみ  
- CTA ボタン風デザインで**次の行動**を明示  

---

## つまずき（教材）

1. **情報過多** — 読まれない  
2. **コピペ** — 信頼と法リスク  
3. **本商品と無関係** — 購入までの一貫性が切れる  
4. **見た目のみ** — 中身が伴わないと長期で反噬

## AI プロンプト骨子（教材）

各プロンプトの出力は**必ず人間が事実確認・法務観点でチェック**（[`15_品質チェックと公開.md`](15_品質チェックと公開.md)）。

**チェックリスト**

```
[ジャンル]専門家として、LINE登録特典用チェックリストを。
ターゲット：{ }
タイトル案：「○○のための△△チェックリスト」
・項目20〜40、カテゴリ5〜7、「〜しているか？」形式
・初心者が今日1つできる具体項目
・最重要TOP10を明示
Markdown でコピペ可に
```

**PDF 用レポート（下書き）**

```
特典用レポート構成案を Markdown で。
テーマ：{ } / ターゲット：{ }
・3000〜5000字目安、はじめに→3〜5章→まとめ
・章ごとに具体例、用語は平易に
・最後は本商品への自然な導線（誇大・断定禁止）
```

**テンプレ集**

```
テンプレ5〜7パターン。使用場面・記入例・空欄型。
```

**タイトル案10**

```
内容：{ } / ターゲット：{ }
・20文字以内案を10個、具体数字・限定は事実のみ
```
