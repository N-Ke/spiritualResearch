## このレポートで分かること

AI に本文を書かせるときの **監督（ディレクター）** の役割、**1章ずつ**の PDCA、冷たい文章を和らげる **5 テクニック**、**修正地獄**の抜け方（教材）。

## はじめに

骨格・リサーチ・キャラが揃ったら **本文（教材では Step 3）**。パイプラインでは**下書きの多くを AI が担い**、あなたは**ライターではなく監督**— 指示・確認・修正指示・ OK 判断（教材）。

### 協業のイメージ（教材）

| | 従来に近いイメージ | AI 協業 |
|---|-------------------|---------|
| 時間 | 長時間になりやすい | **短くまとまりやすい**（個人差） |
| 文章力 | 本人に依存 | AI の基礎表現＋監督 |
| 疲労 | 執筆全体 | **チェック・指示**が中心 |

**丸投げは失敗しやすい**（教材）。AI は**指示が細かい新人ライター**のイメージ。

### 役割分担（教材）

| 監督（あなた） | AI（ライター） |
|----------------|----------------|
| どの章を書くか、トーン・文字数 | 執筆・リサーチ反映・キャラ反映 |
| 原稿チェック、**具体的**な修正指示 | 書き直し |
| OK で次章へ | — |

労力の比率（10% / 90%）は**あくまで教材の比喩**。実際は章数・難易で変動。

## 絶対ルール：一気に書かせない（教材）

長文を一度に書かせると、**後半の薄さ・ループ・キャラ崩れ・途切れ・トーン不一致**が起きやすい（コンテキスト限界の比喩）。

**成功パターン**: **1章ずつ** → チェック → 修正 → OK → 次章。

## 執筆の PDCA（毎章）

1. **Plan** — 章の目的・文字数・必須要素を指示  
2. **Do** — AI が生成  
3. **Check** — 下記4観点で読む  
4. **Act** — 修正指示 or 次章へ  

### Step 1：章ごとの指示例（教材・骨子）

**導入（はじめに）**  
共感・読了後の未来・引き／1,500〜2,000 字／キャラ厳守  

**ノウハウ章**  
明日からできる How と理由／2,500〜3,000 字目安／箇条書き3箇所以上・具体例2・問いかけ1 等  

**まとめ（おわりに）**  
要点・背中押し／1,000〜1,500 字／熱量  

（**正本**は `02` の Step 3 プロンプト。）

### Step 2：監督チェック（4観点）

1. **具体性（How）** — 「意識が大切」だけで終わっていないか。明日何をするか。  
2. **キャラ（Who）** — 感情・失敗談・口調が設計書と一致するか。  
3. **論理** — 飛躍、前章とのつながり。  
4. **可読性** — スマホ想定、箇条書き・太字、**一文の長さ**（60字以内を目安とする説もある、教材）。  

### Step 3：フィードバック

**曖昧な指示は避ける**（教材）。

| 悪い | 良い |
|------|------|
| もっと具体的に | 第3段落の○○を3つのアクションに分解 |
| もっと熱く | 最後の段落に背中を押す一文を追加 |
| なんか変 | 第2〜3段落の間に理由を1文 |

### Step 4：次章へ

前章と**矛盾しない**・前章末の予告から始める・トーン維持、と明示（教材）。

## AI の冷たさを壊す 5 テクニック（教材）

1. **対話（Q&A）** — 「そう思いますよね？」で間を置く  
2. **箇条書き・太字** — スキャンしやすく  
3. **読者への問いかけ** — 一人称化・行動の気づき  
4. **失敗談** — 成功談だけより共感（**事実に基づく**）  
5. **数字** — 「多く」より**検証済みの数値**（**出典のない割合は載せない**）  

※ 教材の「87%」「3.2倍」等は**例示**。掲載するなら**ソース必須**。

## キャラの声を維持する（教材）

**ブレの原因**: 長文でデフォルト口調に戻る／章でトーンが違う／設定が薄れる  

**対策**  
- **章の頭で設定再読**を指示  
- **口癖・「この人なら」** を例文で固定  
- **文体チェックリスト**（語尾・一人称・呼びかけ・感情・ロボット化していないか）  

## 何を AI に任せるか（教材）

**任せやすい**: 構成展開、例の生成、箇条書き整理、要約、言い換え（※著作権・事実確認は監督）  

**自分が担う**: 最終 OK/NG、**核となる価値観**、本音の一文、**どうしても伝えたい所**、直らない所の方針、**法的にグレーな表現**  

## 修正地獄から抜ける（教材）

1. **物理的な指示** — 「1500字に」「結論を先に」「語を△に置換」  
2. **80点で次へ** — 「次の章に進みたくなるか？」が Yes なら前向きに進む。完璧主義で止まらない。  
3. **一度に直すのは1点**  
4. **白紙リセット** — 何度もダメなら章ごと書き直し指示（問題点を明示）  

## 失敗パターン（教材）

1. 流れを把握せずいきなり第1章だけ書く → **全体目次と各章メモ**から  
2. リサーチを指示に含めない → **「この章ではリサーチの○セクションを使う」**と明示  
3. 文字数未指定 → **章ごとに目安**  
4. チェックせず次へ → **1章ごとに OK**  
5. 全部手直し → **基本は指示で AI に書き直し**  

## FAQ（要約）

| Q | A（教材） |
|---|-----------|
| 1章の時間？ | 指示〜OK で **10〜30 分**目安。10章なら数時間規模のイメージ。 |
| 直らない | 指示の曖昧さ・複数同時修正を疑う。3回で白紙リセットも。 |
| 同じ表現の繰り返し | **禁止フレーズ**と**代替パターン**を明示。 |
| 文字数バランス | 例：導入10%・中盤60%・まとめ15% 等（商品全体で調整）。 |
| 疲れた | 章区切りで休憩。1日2〜3章など分割。 |

## まとめ（教材の4行）

1. **監督**であり、全部自分で書かない前提でも**判断は自分**  
2. **1章ずつ**チェック  
3. 手で直すより**指示を変えて書き直し**  
4. **80点で進む**
