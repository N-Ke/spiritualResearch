## このレポートで分かること

付属物（ボーナス・特典）の**種類と選び方**、**AI 向けプロンプト例**、**見せ方**、失敗パターン（教材）。

## はじめに

本文だけで急いで公開すると、**行動までの負荷**が高く、満足度や実践率が落ちやすい（教材の主張）。付属物は **情報と行動のギャップ** を埋めるための補助線。

### よくある先送り（教材）

- 「内容が良ければ本文だけで十分」  
- 「付属物は後で」  
- 「チェックリストは面倒」  
- 「特典はダサい」  

→ 読者が**何をいつやるか**まで設計していないと、読了後に止まりやすい、という整理（教材）。

## なぜ付属物か（Why）

| 誤解 | 整理（教材） |
|------|----------------|
| 良い情報を出せば満足 | 読者が払っているのは **変化（結果）** に近い — **知識だけでは終わらない**設計が欲しい |

### 付属物の5つの価値（教材）

1. **認知負荷の軽減** — 「次に何をするか」が明確  
2. **行動の促進** — すぐ動ける  
3. **結果の創出** — 実践しやすく満足に寄与  
4. **信頼** — 「使える形で渡している」  
5. **リピート** — 次商品への土台（過度な約束はしない）

### 本文だけ vs チェックリスト付き（教材）

| 本文のみ | チェックリスト等 |
|----------|------------------|
| 長文 → 自分で整理 → 優先順位 → 行動（ステップが多い） | リストに沿って **1項目ずつ** 実行 |

### 「お得感」の演出（教材）

同じ中身でも、**使える形の追加**で「まとまって届く」印象になりやすい。**価格表示や比較表現は景表法・プラットフォーム規約に注意**（虚偽・誤認につながる表現は NG）。

### クレーム緩和のイメージ（教材）

本文への印象がマイナスでも、**実用的な付属物**で総合評価が相殺されうる、という心理モデル（**万能ではない**）。

## 付属物6種と効果マトリクス（教材）

| 種類 | 目的 | 効果（目安） | 難易度 | 向きやすい商品 |
|------|------|--------------|--------|----------------|
| ① チェックリスト | 行動順の明確化 | 行動 ★★★ | ★ | ハウツー・手順 |
| ② サマリー | 復習・全体把握 | 理解 ★★ | ★ | 長文・理論 |
| ③ チートシート | コピペ即利用 | 時短 ★★★ | ★★ | ツール・AI 活用 |
| ④ ワークシート | 自分事化 | 定着 ★★★ | ★★ | 自己分析・設計 |
| ⑤ 用語集 | 用語の安心 | 安心 ★★ | ★ | 専門・初心者向け |
| ⑥ スプレッドシート | 計算・管理 | 実務 ★★★ | ★★★ | 数値・計画 |

## 各付属物の AI プロンプト例（骨子・教材）

**共通**: プロンプトに **本文全文または章ごとの要約＋方針** を添付。著作権は**自分の本文**に限る。

### 1. チェックリスト

```
以下の本文から、読者が実行すべき「行動」だけを抽出し、チェックリスト形式で出力。
・□ 付き
・時系列で整理
・各項目は動詞で終わる形
・7〜15項目程度
```

### 2. サマリー

```
本文の要点のみ。2000字以内。
・章ごとに見出し
・各章 3〜5 箇条書き
・最後に「全体のまとめ」最重要3点
```

### 3. チートシート

```
プロンプト・テンプレ・定型文など、コピペで使える要素を抽出。
・番号と名前
・区切り
・各要素に「使い方」1〜2行
```

### 4. ワークシート

```
穴埋め形式。5〜10問、回答欄あり。易→難の順。
```

### 5. 用語集

```
専門語・略語・業界語。各 2〜3 行の平易な説明。10〜30語程度。
```

### 6. スプレッドシート設計

```
目的、列名と意味、必要なら式のセル指定。マークダウン表で。
```

## 迷ったときの選択（教材）

### 商品タイプ別

| タイプ | 必須にしやすい | 推奨の次点 |
|--------|----------------|------------|
| ハウツー・手順 | チェックリスト | チートシート |
| 理論・概念 | サマリー | 用語集 |
| 自己分析・設計 | ワークシート | テンプレ系 |
| AI・ツール | チートシート | チェックリスト |
| 数値・計画 | スプレッドシート | チェックリスト |

### ターゲット別（教材）

| レベル | 最優先 | 次点 |
|--------|--------|------|
| 超初心者 | 用語集 | チェックリスト |
| 初心者 | チェックリスト | サマリー |
| 中級 | ワークシート | チートシート |
| 上級 | スプレッドシート | チートシート |

### 時間がないときの最低ライン（教材）

1. **チェックリスト** — 行動抽出  
2. **サマリー** — 要点抽出  

「5分で完成」等は**個人差・本文量による**目安。

### 個数の目安（教材）

| 個数 | 印象 |
|------|------|
| 1 | 普通 |
| 2 | 好印象になりやすい |
| 3〜5 | 充実感（推奨ゾーン） |
| 10+ | 選べない・使われないリスク |

## パイプライン AI での一括指示例（教材）

**注意**: 「Step 4 を実行」と書く場合、**あなたの `02` の手順名**と一致するか確認。

**基本例**

```
Step 4（付属物・特典）相当の作業として、完成本文に基づき
【チェックリスト】と【プロンプトのチートシート】の2点を作成してください。
```

**詳細例**

```
同じく付属物として以下をマークダウンで：
① チェックリスト（行動10〜15、時系列）
② サマリー（2000字以内、章見出し＋箇条書き＋全体3点）
③ プロンプトチートシート（一覧＋使い方）
④ 用語集（10〜20語）
```

## 見せ方（デザイン）のコツ（教材）

| ツール | 向く付属物 | メモ |
|--------|------------|------|
| Canva | サマリー、チート | 表紙・体裁 |
| Notion | チェック、ワーク | デジタル感 |
| スプレッドシート | 数値・進捗 | 共有・式 |
| Google ドキュメント | サマリー、用語 | PDF 化しやすい |

**原則**: デザインに時間を取りすぎない。**中身と本文との整合**が先。

## 良い例・悪い例（教材）

**チェックリスト**

| 悪い | 良い |
|------|------|
| 「考える」「いろいろやる」 | 具体動詞＋1アクション1行（例：ジャンルを3つ書き出す） |

**サマリー**

| 悪い | 良い |
|------|------|
| 「第1章で説明した」だけ | 再現可能な要点の箇条書き |

## つまずき（教材）

1. **おまけ扱い** → 質が低いと商品評価を下げる  
2. **本文と無関係** → 水増し感  
3. **作りすぎ** → 迷う  
4. **デザイン過多** → 本編が遅れる  
5. **AI 丸投げ** → 誤り・用語ブレ。本文と**統一して校正**

## FAQ（要約）

| Q | A（教材） |
|---|-----------|
| 何個？ | 最低2、理想3〜5。無理ならチェックリスト＋サマリー。 |
| 時間？ | 目安1〜2時間（AI＋確認・体裁）。個人差大。 |
| 付属物だけ別売り？ | 基本おすすめしない（文脈が本文とセット）。 |
| 競合と同じ？ | 型として問題少ない。差は**具体度・あなたの文脈**。 |
| 後から追加？ | あり。購入者向けに「反映しました」と伝えると効果的になりうる。 |

## まとめ（教材）

1. 読者は **変化まで** を求めている前提で補助線を引く  
2. 迷ったら **チェックリスト＋サマリー** から  
3. **本文から抽出**し、ゼロから作りすぎない  
4. **3〜5個**に厳選し、目的を分ける
