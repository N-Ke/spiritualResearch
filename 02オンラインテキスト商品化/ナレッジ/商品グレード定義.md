# 商品グレード定義

商品の価格帯・ボリューム・付属物を3段階で定義する。
Step0でグレードを判定し、以降のステップ全体に影響する。

---

## グレード早見表

| 項目 | Lite | Standard | Premium |
|------|------|----------|---------|
| **価格帯** | 100-980円 | 980-5,000円 | 5,000-30,000円 |
| **対象** | 単一の悩み | 単一の悩みを深掘り | 悩みクラスター（3-5個）を包括 |
| **本編文字数** | 3,000-6,000字 | 8,000-15,000字 | 20,000-50,000字 |
| **章構成** | H2×5-7 | H2×7-10 | 5-10章（章ごとにH2/H3） |
| **図解** | 0-2枚 | 3-5枚 | 8-15枚 |
| **付属物** | なし | テンプレ1-2点 | ツール+テンプレ+ワークシート |
| **無料部分** | H2×2-3 | H2×3-4 | 導入章（丸ごと公開） |
| **制作回数目安** | Step0-5で6回 | 8-10回 | 15-25回 |
| **初心者の制作日数目安** | 1-2日 | 3-5日 | 7-14日 |

---

## Lite（100-980円）

### 商品像
1つの悩みを1本の記事で解決する。読者は「この悩みだけ解決したい」というピンポイントのニーズを持っている。

### 本編の構成
- 無料部分: 2-3個のH2（全体の40%程度）
- 有料部分: 3-4個のH2
- 合計: 3,000-6,000文字

### 図解
- 必須ではない
- フレームワーク図が1つあると説得力が増す
- なくても成立する内容にする

### 付属物
なし。本編のテキストだけで完結。

### 制作パイプライン
現行のStep0 → Step0.5 → Step1 → Step1.5 → Step2 → Step3 → Step4 → Step5

---

## Standard（980-5,000円）

### 商品像
1つの悩みを深く掘り下げ、読者が自力で実行できるレベルまで具体化する。テンプレートを付属させることで「読んだ後に使える」を実現。

### 本編の構成
- 無料部分: 3-4個のH2（全体の30%程度）
- 有料部分: 4-6個のH2
- 合計: 8,000-15,000文字
- 執筆は「無料部分 → 有料前半 → 有料後半」の3回に分割

### 図解
- 3-5枚
- 推奨配置:
  - フレームワーク図（独自概念の可視化）
  - フロー図（手順の全体像）
  - ビフォーアフター図（変化の可視化）
  - 比較表（選択肢の整理）

### 付属物
- テンプレート: 1-2点（本編で説明した内容をすぐ実行できる型）
- または ワークシート: 1点（自己診断や計画用）
- Step3.5のPhase A-Cで制作

### 制作パイプライン
Step0 → Step1 → Step1.5 → Step2 → Step3（3分割）→ Step3.5 → Step4 → Step5

---

## Premium（5,000-30,000円）

### 商品像
**複数の関連する悩み（クラスター）を包括的に解決する「変革パッケージ」。**
読者の現状を診断し、知識を提供し、ツールで実行を支援し、ワークシートで振り返りまで導く。
「読んで終わり」ではなく「相手の行動・生活に踏み込んで改善する」商品。

### 本編の構成
- 導入章: 悩みの全体像 + このガイドで何が変わるか（無料公開）
- 本編章: 各悩みの解決（3-5章）
- 統合章: 複数の悩みを横断する実践法
- ロードマップ章: 読者がこれから何をどの順でやるか
- 合計: 20,000-50,000文字
- 執筆は「導入章 + 各章ごと」に分割（5-10回）

### 図解
- 8-15枚
- 推奨配置:
  - 全体マップ（導入章: 悩みと解決策の全体像）
  - 各章のフレームワーク図
  - 各章の手順フロー図
  - ビフォーアフター図
  - ロードマップ図（最終章: 行動計画の可視化）

### 付属物

#### AIツール（1-3個）
読者がAIに貼って繰り返し使えるプロンプト。

| 種類 | 例 | 目的 |
|------|-----|------|
| 診断ツール | 「現状診断プロンプト」 | 読者の現在地を把握 |
| 生成ツール | 「◯◯自動生成プロンプト」 | 繰り返し使う作業を自動化 |
| 分析ツール | 「◯◯分析プロンプト」 | データや状況を構造化 |
| 改善ツール | 「◯◯添削プロンプト」 | 読者の成果物をAIが改善提案 |

#### テンプレート（3-5点）
空欄を埋めるだけで使える型。

| 種類 | 例 | 目的 |
|------|-----|------|
| 設計テンプレ | 「◯◯設計シート」 | 白紙から作らずに済む |
| 計画テンプレ | 「◯◯計画書」 | 行動計画を型に沿って作る |
| チェックリスト | 「◯◯確認リスト」 | 漏れなく実行する |
| 文言テンプレ | 「◯◯テンプレート」 | コピペで使える文章の型 |

#### ワークシート（2-3点）
問いに答えながら自分の状況を整理するシート。

| 種類 | 例 | 目的 |
|------|-----|------|
| 現状診断WS | 「◯◯セルフチェック」 | 今の状態を客観視 |
| 目標設定WS | 「◯◯ゴール設計」 | 目指す状態を具体化 |
| 振り返りWS | 「◯◯振り返りシート」 | 実行後の改善点を洗い出す |
| アクションWS | 「◯日アクションプラン」 | 具体的な行動計画を作る |

### 制作パイプライン
Step0（クラスター分析）→ Step1（章別テンプレ選択）→ Step1.5（クラスター横断リサーチ）→ Step2（章構成+付属物仕様+図解設計）→ Step3（章ごと分割執筆）→ Step3.5（付属物制作 3フェーズ）→ Step4 → Step5

---

## グレード判定フロー

Step0で悩みリストを作成した後、以下の順で判定する。

```
Q1: この悩みは単独で解決できるか？
  → YES → Q2へ
  → NO（複数の悩みが絡み合う）→ Premium候補

Q2: 解決に必要な手順は何ステップか？
  → 3-5ステップ → Lite
  → 6-10ステップ、または深掘りが必要 → Standard

Q3: 読者に「使える道具」が必要か？
  → テキストだけで十分 → Lite
  → テンプレ1-2個あると良い → Standard
  → ツール+テンプレ+WSが必要 → Premium

Q4: ターゲットの支払い意欲は？
  → 「とりあえず試したい」→ Lite
  → 「ちゃんと学びたい」→ Standard
  → 「本気で変わりたい」→ Premium
```

### 迷ったときの基準
- 悩みの数が3つ以上絡み合っている → Premium
- 「読んだ後に何かを作る/実行する」必要がある → Standard以上
- テキストを読むだけで解決する → Lite
- 繰り返し使う道具が必要 → Premium

## 価格設定の目安

### Lite
| 条件 | 価格帯 |
|------|--------|
| 3,000字以下、概要レベル | 100-300円 |
| 3,000-6,000字、具体的手順 | 300-500円 |
| 6,000字+テンプレ1点 | 500-980円 |

### Standard
| 条件 | 価格帯 |
|------|--------|
| 8,000-10,000字+テンプレ1点 | 980-1,500円 |
| 10,000-15,000字+テンプレ2点+図解 | 1,500-3,000円 |
| 15,000字+テンプレ+WS+図解多数 | 3,000-5,000円 |

### Premium
| 条件 | 価格帯 |
|------|--------|
| 5章+ツール1+テンプレ3+WS2 | 5,000-10,000円 |
| 7章+ツール2+テンプレ5+WS3+図解10枚 | 10,000-20,000円 |
| 10章+ツール3+テンプレ5+WS3+図解15枚 | 20,000-30,000円 |

初回は各グレードの下限付近で出し、レビュー・販売実績を作ってから段階的に上げる。
