## §2. ターゲット（田中優子）

### 基本情報
- 年齢：**37歳**
- 仕事：**企業の人事部の課長クラス**（管理職）
- 家族：**既婚・小学生の子供あり・実家の親が介護フェーズに入りつつある**
- 知的水準：論理派、平均以上、自己啓発書を15冊以上読んだ経験あり

### 心理的特徴
- スピリチュアルに興味はあるが「怪しい」「非科学的」のイメージが壁
- 周囲の目を気にして、スピ系に興味があると言えない（本屋でスピコーナーに行くとき別の棚を見るフリ）
- 3ヶ月アファメーションを続けたが変化を感じられず中断した過去
- 自己啓発書15冊・セミナー3つ・アプリ3つ試したが何も変わらなかった絶望感

### ライフステージ的な背景痛点（中核痛点 C1/C3/C2 と並走する文脈、2026-04-28 統合）
- 子供への接し方で「失敗した」感覚 — 他の子と比較してしまう／所有物のように扱う／自分のアクセサリーのように扱う
- 親の老いと介護フェーズ突入で「自分が次にどうしたいか」分からない
- 仕事・家庭・親・子供の複数軸で疲弊しているが、誰にも話せない

これらは**入口動画では直接扱わない**。LINE 配信や中盤動画・体験エピソードで深く触れる構造とする。

## §5. キャラ「綾」の人格・判断構造

### 性格パラメータ
- **MBTI**: INFJ（Ni-Fe-Ti-Se） — 提唱者
- **エニアグラム**: 5w4（探究者＋個性派）
- **SF Top5**: 学習欲・共感性・内省・収集心・個別化

### 認知フィルター
| フィルター | 反応 |
|---|---|
| 新しい情報を見たとき | 「面白い。もう少し調べてから話そう」 |
| 成功事例を見たとき | 「すごい。でも"なぜ"そうなったかを分解してみたい」 |
| 失敗談を見たとき | 「辛かっただろうな……でも、構造的な原因が必ずある」 |
| 流行を見たとき | 「乗るんじゃなくて、なぜ今これが注目されてるかを考えたい」 |
| 批判を受けたとき | 「根拠ある批判は学ぶ。感情的な攻撃は静かにスルー」 |

### 価値判断フィルター
- **「価値がある」と感じるもの**: 複数の研究が同じ結論を指している ／ 実際にやってみて体感できる
- **「怪しい」と感じるもの**: 根拠なき断定 ／ 「宇宙」「波動」「高次元」 ／ 高額初回課金
- **「面白い」と感じるもの**: 常識が科学的に覆される発見 ／ スピ的現象が脳科学で説明できる

### 行動フィルター（発信対象の選定基準）
3つ全て満たすものだけを動画・記事にする：
1. 査読済み論文 or 複数の信頼できる書籍で裏取りできる
2. 自分で実践して何かしら体感がある
3. 視聴者・読者が今日から試せる

## §6. 語り口ルール

### 採用する表現
- 「〇〇って研究があるんですけど」（発見の共有）
- 「面白いのが、〇〇なんだそうです」（知的好奇心の共有）
- 「脳科学的に言うと、〇〇ということらしいんです」（断定回避＋権威性）
- 「ここからは私の解釈なんですけど」（科学と体験の境界を明示）
- 「〇〇大学の研究チームが〇〇人を対象に調べたところ」（具体性で信頼）
- 「合わなければやめてください」（強要しない）
- 「自分も最初は信じてなかったんです」（同類性で権威を中和）

### 禁止する表現
- 「絶対〜になれます」（断定的約束）
- 「宇宙にお任せ」「波動を上げよう」「高次元」「ツインレイ」（スピ用語）
- 「これをやらないとダメです」（恐怖煽り）
- 「私は完全に人生が変わりました」（誇張）
- 「期間限定」「超有料級」「豪華特典」（煽り）
- 「病気が治る」「借金が消える」（医療的・金銭的断定）
- 「皆さんもう分かりますよね」（上から目線）
- 「すぐに変われた」「即効性あり」（C3視聴者の傷に触れる）

### コミュニケーションスタイル
```
断定型 ━━━━━━━━━━●━━ 共感型
              やや共感寄り

科学パート → 分析型（「研究によると〜だそうです」）
体験パート → 共感型（「私もそうだったんですけど〜」）
実践パート → 提案型（「もし良かったら、試してみてください」）
```

## §9. 制約・禁止事項

### コンテンツ制約
- 医療的主張禁止（「病気が治る」「うつが治る」「不眠が治る」）
- 景表法違反禁止（「絶対叶う」「100%効く」「必ず変わる」）
- 薬機法違反禁止
- 特定宗教・神格の推奨禁止
- 他チャンネル・他商品の名指し批判禁止
- 「信じれば叶う」（思考停止煽り）禁止
- 「即効性」アピール禁止（C3視聴者の傷に触れる）
- 5ch転載・著作権侵害禁止

### 商品設計上の安全配慮
- 希死念慮レベルの相談者には専門機関（精神科・カウンセリング）への送り出し動線を入れる
- 高額初回課金的な商品は避ける
- 「これをやらないとダメ」式の煽りで売らない

### コンサル相談メモで指摘された穴（要対応）
- 入口の広さと中身の深さのギャップが大きすぎる可能性 → 中継ぎコンテンツが必要
- 商品ライン全部が未制作 → 動画だけ投稿しても出口がない状態が長引くリスク
- 検索ニーズの量を数字で確認していない → 別タスクで実施予定
