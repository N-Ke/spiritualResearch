# Step5: 公開設定

---

## 【エージェント指示】

```
読み込むファイル:
  - 入力/ フォルダ内のキャラクター設計書
  - 実行結果/Step0_出力.md
  - 実行結果/Step2_出力.md（商品全体設計）
  - 実行結果/Step4_出力.md（品質チェック結果）
  - ナレッジ/プラットフォーム仕様.md
出力先: 実行結果/Step5_出力.md
次のステップ: なし（パイプライン完了）
```

**エージェントの実行手順:**
1. 上記ファイルを全て読み込む
2. 公開設定を生成 → `実行結果/Step5_出力.md` に保存
3. ユーザーに確認
4. OKなら完了メッセージを表示:
   「パイプライン完了です。実行結果/ フォルダに全ての成果物が揃っています。」

---

## このステップでやること

最終原稿をnoteまたはBrainに公開するための**設定値を決定**し、公開前の最終チェックを行います。設定内容は商品グレード（Lite/Standard/Premium）によって異なります。

## 2. 価格設定の根拠

- **文字数からの目安**: ◯◯文字 → ◯◯円が相場
- **テンプレ/ツール有無**: テンプレ/チェックリスト付きは＋◯円
- **提案価格**: ◯◯円

### Premium の場合の追加考慮
- **章数**: ◯章 × 基本単価
- **付属物の価値**: AIツール◯個 + テンプレ◯点 + ワークシート◯点
- **包括解決の付加価値**: 個別に買うより◯%お得
- **提案価格**: ◯円

- **価格設定のポイント**:
  - 初期は低め（実績作り）
  - 評価が集まったら段階的に上げる
  - セット販売（マガジン等）で単価を上げる

【プラットフォーム仕様メモ】

■ note
- 手数料: 決済手数料5% + プラットフォーム手数料10% = 実質15%（クリエイター85%取得）
- 最低価格: 100円
- 最高価格: 50,000円
- 有料エリア: エディタ内の「有料ライン」で設定
- 目次: H2/H3が自動で目次になる
- 返金: 24時間以内なら購入者が返金リクエスト可能

■ Premium商品の注意点
- note: 1記事の文字数上限なし。長文でもOK。マガジンとしてまとめる選択肢もあり。
- Brain: サムネイルが特に重要（5,000円以上は「何が含まれるか」を視覚的に訴求）
- 概要文で「◯章の本編 + ◯個のツール + ◯個のテンプレ + ◯個のワークシート」と明記して価値を可視化

■ Brain
- 手数料: 決済手数料3.6% + プラットフォーム手数料12% = 実質約15.6%
- 最低価格: 無料〜（有料は100円〜）
- 紹介料: 0-50%で設定可能
- サムネイル: 必須（1200×630px推奨）
- 返金: 出品者が返金ポリシーを設定
