## 修正リスト

### 修正1
- **チェック項目**: C15（1文の長さ）
- **該当箇所**: 導入章「このガイドで何が変わるか」セクション
  「読み終わったあなたの状態:」の直前の文
  「第1層は、今あるアドセンス。これは捨てない。第2層が、今から作るもの。リストマーケと低単価商品。第3層は、第2層が回り始めたら作るもの。」
- **問題**: 3文が連続していて一見問題ないが、各文の前後の改行が不足しており、スマホで見ると詰まって見える
- **修正案**: 各文の間に空行を入れる

```
第1層は、今あるアドセンス。これは捨てない。

第2層が、今から作るもの。リストマーケと低単価商品。

第3層は、第2層が回り始めたら作るもの。
```

## チェック詳細（主要項目の確認メモ）

- C01: 誇大表現なし。「確実に」「絶対に」は未使用。「月3万」と具体額を出しているが、個人の体験として語っており問題なし。
- C08: AI利用の表記 → 公開時にnoteの概要文で明記する。

### キャラ一貫性（C09-C14）
- C09: 一人称「僕」で統一。OK。
- C10: 「ちょっと実験してみたんだけど」「面白い発見があったんだけど」等、よく使う表現が自然に配置。OK。
- C11: 禁止表現（命令形、煽り、「簡単に稼げます」等）なし。OK。
