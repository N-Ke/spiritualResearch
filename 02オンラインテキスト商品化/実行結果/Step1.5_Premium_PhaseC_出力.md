# Step1.5 Phase C 出力（Premium版テスト）: オリジナル素材集

## オリジナル素材集: クラスターA「非属人YouTuber脱アドセンス完全ガイド」

---

### 素材1: 独自フレームワーク「収益柱の3層モデル」
- **使用する章: 第1章 + 統合章**
- **内容**: アドセンス依存を「1本柱の家」に例え、収益構造を3層で設計するフレームワーク
  - 第1層: 安定基盤（アドセンス — 今あるもの。捨てない）
  - 第2層: 成長エンジン（リストマーケ × 低単価商品 — 今から作るもの）
  - 第3層: 飛躍レバー（高単価パッケージ × 自動化 — 第2層が回ってから作るもの）
- **オリジナル性**: リサーチでは「アドセンスか自社商品か」の二項対立の議論が多かったが、ウラアドの視点は「捨てるんじゃなくて、上に積む」。12年やってきたアドセンスを否定せず、その上に新しい層を重ねる設計思想。
- **キャラの語り口**: 「アドセンスを捨てろっていう人もいるけど、僕はそうは思わない。12年の土台を壊すんじゃなくて、その上に新しい柱を立てる。それが一番安全で、一番面白い」

---

### 素材2: 逆張りポイント「信頼は"顔"じゃなく"体験密度"で作れる」
- **使用する章: 第3章**
- **内容**: リサーチでは「信頼構築=顔出し+実名」が大前提とされていた。しかし非属人チャンネルの事例分析で見えたのは、信頼の本質は「この人が言うなら」ではなく「このコンテンツに触れた体験の密度」。
  - 属人型の信頼構築: 人格 → 共感 → 信頼 → 購入
  - 非属人型の信頼構築: 価値提供 → 再体験 → 習慣化 → 信頼 → 購入
- **オリジナル性**: 一般的には「非属人だと信頼構築のハードルが高い」という文脈だが、ウラアドの実験から「ハードルが高いのではなく、ルートが違うだけ」と再定義。
- **キャラの語り口**: 「僕も最初は"顔を出さないと信頼なんて作れない"と思ってた。でも3ヶ月実験してみたら、信頼って"顔"じゃなかったんですよ。"この人のコンテンツに触れた回数と深さ"だった」

---

### 素材3: 実験データ「非属人チャンネルのリスト取得実験 3ヶ月」
- **使用する章: 第2章 + 第3章**
- **内容**: 仮想的な実験データとして:
  - 月1: 概要欄にフリーギフトリンクのみ → 登録率0.3%（100再生あたり0.3人）
  - 月2: 動画内で口頭誘導（合成音声）+ 概要欄 + 固定コメント → 登録率0.8%
  - 月3: 動画末尾に「特典プレビュー」セクション追加 + LINE連携 → 登録率1.5%
  - 3ヶ月でリスト347人。うちステップメール完走率62%、商品購入率8.2%
- **オリジナル性**: リサーチでは一般的な開封率・CVRデータは見つかるが、「非属人チャンネル限定のリスト取得率の推移」は情報がなかった。ウラアドの実験として語ることで唯一無二のデータに。
- **キャラの語り口**: 「ちょっと実験してみたんだけど、数字を全部出すね。最初の月は正直ボロボロだった。でも3ヶ月で見えてきたのは、"非属人でもリストは取れる。ただし導線の設計が全て"ということ」

---

### 素材4: ターゲットの声（ペルソナの内心モノローグ）
- **使用する章: 導入章 + 第1章**
- **内容**:
  - 「月50万あるのに、毎月YouTube Studioの数字見るたびに胃が痛い」
  - 「リストマーケやった方がいいのは分かってる。でも今のアドセンスの動画制作だけで手一杯」
  - 「顔出してないのに、商品なんて誰が買うんだよって思う」
  - 「Brain とか note とか、あの界隈のノリについていけない気がする」
  - 「でも、このまま10年後もアドセンスだけって考えると、ゾッとする」
- **オリジナル性**: リサーチでは「リストマーケ初心者」の一般的な悩みは出てきたが、「アドセンスで月10-80万稼いでいる非属人YouTuber特有の矛盾した感情」はなかった。稼いでいるからこそ動けないジレンマ。
- **キャラの語り口**: 「よく聞くのが"月50万あるのに怖い"っていう声。これ、外から見たら贅沢な悩みに聞こえるかもしれない。でも僕には痛いほど分かる。金額の問題じゃないんですよ。"選択肢がない恐怖"なんです」

---

### 素材5: 独自フレームワーク「商品化の4段ロケット」
- **使用する章: 第4章**
- **内容**: 初心者が「商品を作れない壁」を超えるための段階設計
  - 第1段: 経験の棚卸し（自分が人より詳しいこと3つを書き出す）
  - 第2段: 最小商品の制作（note で500円の記事を1本出す。完璧じゃなくていい）
  - 第3段: フィードバックループ（売れた/売れないに関わらず、反応を見て次を改善）
  - 第4段: 商品ラインの拡張（500円 → 1,500円 → 5,000円 → パッケージ化）
- **オリジナル性**: リサーチでは「商品の作り方」の情報は豊富だったが、「なぜ作れないか」の心理面に踏み込んだものは少なかった。ウラアドの視点は「作れないんじゃない、値段をつけるのが怖いだけ。だから最初は500円でいい」。
- **キャラの語り口**: 「"商品を作れない"の正体、僕が思うに、スキル不足じゃないんですよ。"自分の経験に値段をつける恐怖"。だから最初は500円でいい。缶コーヒー1本分。そこから始めよう」

---

### 素材6: 独自概念「非属人の"キャラ信用残高"」
- **使用する章: 第3章 + ロードマップ章**
- **内容**: 非属人が信頼を構築する過程を「信用残高」に例えたモデル
  - 入金方法: 無料で価値あるコンテンツを出す（+10）、約束を守る（+5）、失敗を正直に話す（+15）
  - 出金方法: 商品を売る（-20）、煽り文句を使う（-30）、品質の低い商品を出す（-50）
  - 信用残高が100を超えると「この発信者から買ってみようかな」が発生する
  - 属人型は「顔を見せる」だけで+30入る。非属人はこのボーナスがない分、コンテンツの質と頻度で補う
- **オリジナル性**: 「信用残高」の概念自体は一般的だが、非属人特有の入金ルート（体験密度）と属人型との差分を数値化した点がオリジナル。
- **キャラの語り口**: 「信頼って銀行口座みたいなものだと思っていて。顔出ししてる人は、顔を見せるだけで毎月自動入金がある。僕らにはそれがない。だから"コンテンツの質と頻度"で入金する。ルートが違うだけで、貯まる額は同じなんですよ」

---

### 素材7: 比較分析「属人 vs 非属人のファネル設計の違い」
- **使用する章: 第2章**
- **内容**:

| 要素 | 属人型 | 非属人型 |
|------|--------|---------|
| 認知 | 人→コンテンツ | コンテンツ→（後から）人 |
| 信頼構築 | 共感→憧れ | 価値提供→習慣化 |
| リスト取得 | 「この人から学びたい」 | 「このシリーズの続きが欲しい」 |
| 教育 | パーソナリティで繋ぎ止め | コンテンツの連続性で繋ぎ止め |
| 販売 | 「あなたから買いたい」 | 「この方法論に投資したい」 |

- **オリジナル性**: ファネル設計の情報は大量にあるが、全て属人型前提。非属人に「翻訳」した設計図は見つからなかった。
- **キャラの語り口**: 「リストマーケの教科書は全部"この人から学びたい"がスタートなんですよ。でも非属人は違う。"このコンテンツの続きが欲しい"がスタート。ここの翻訳をミスると、全部ズレる」
