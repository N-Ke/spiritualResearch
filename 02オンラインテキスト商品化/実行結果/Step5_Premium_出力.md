# Step5 出力（Premium版テスト）: 公開設定

## 1. プラットフォーム選定

### 推奨: note（メイン）+ Brain（サブ）

| 項目 | note | Brain |
|------|------|-------|
| 適性 | ◎（長文対応、SEO強い） | ○（紹介機能で拡散） |
| 文字数制限 | 上限なし | 上限なし |
| 価格設定 | 100-50,000円 | 100-設定上限 |
| 決め手 | 導入章の無料公開がしやすい | 紹介料50%で初期拡散を狙える |

**戦略**:
1. noteをメインに公開。導入章を無料部分として公開し、SEOで継続流入を狙う
2. Brainにも同内容を出す。紹介料50%を設定し、アフィリエイターによる拡散を狙う
3. 自身のXやYouTubeでは note のURLを導線メインにする（noteの方が心理的ハードルが低い）

## 2. 価格設定の根拠

### 提案価格: 12,800円

| 要素 | 内容 | 価値換算 |
|------|------|---------|
| 本編7章 | 約35,000文字 | 7,000円相当 |
| 図解10枚 | フレームワーク・フロー図等 | 1,500円相当 |
| AIツール2個 | 繰り返し使える診断+生成 | 2,000円相当 |
| テンプレ4点 | 設計シート・企画書・計画表・チェックリスト | 1,500円相当 |
| ワークシート3点 | 診断・棚卸し・振り返り | 1,000円相当 |
| **合計相当額** | | **13,000円** |

### Premium の追加考慮
- **章数**: 7章 × 約1,800円/章 = 12,600円
- **付属物の価値**: AIツール2個 + テンプレ4点 + WS3点 = 9個の付属物
- **包括解決の付加価値**: 個別に Lite/Standard を買うより約40%お得な設計
- **提案価格**: **12,800円**（初回限定で9,800円も検討可）

### 価格の段階戦略
1. **初回リリース**: 9,800円（「100部限定」等で緊急性を出す）
2. **通常価格**: 12,800円
3. **レビュー蓄積後**: 14,800円へ段階的に引き上げ

## 3. note 公開設定

### 基本設定
- **タイトル**: アドセンスの"その先"完全攻略 —非属人のまま商品を作って売るまでの全工程—
- **有料設定**: 有料（12,800円）
- **無料エリア**: 導入章全体（約3,000文字）
- **カテゴリ**: ビジネス
- **タグ**: #YouTube #アドセンス #リストマーケティング #非属人 #収益化 #副業 #オンラインビジネス

### サムネイル
- メインビジュアル: 3層ピラミッドのシルエット
- テキスト: 「アドセンスの"その先"完全攻略」
- サブテキスト: 「7章+9点の付属物」
- トーン: 落ち着いたネイビー背景 + 白文字

### 概要文（noteの冒頭に表示される部分）
```
YouTube歴12年・現役アドセンスコンサルが裏で実験した、
非属人チャンネルからリストマーケで第2の収益柱を作る全工程。

■ 含まれるもの
・本編7章（約35,000文字）
・図解10枚
・AIツール2個（診断+商品アイデア生成）
・テンプレート4点（設計シート・企画書・計画表・チェックリスト）
・ワークシート3点（依存度診断・強み棚卸し・振り返り）

■ こんな方へ
・アドセンスで月10-80万あるが、1本依存が怖い
・非属人のままリストマーケに挑戦したい
・自分の商品を作りたいが、何から始めればいいか分からない

■ 導入章は無料で全文読めます
まず読んでみて、「自分のことだ」と思ったら先に進んでください。
```

## 4. Brain 公開設定

### 基本設定
- **タイトル**: 【非属人×脱アドセンス】リストマーケで第2の柱を作る完全パッケージ（7章+9付属物）
- **価格**: 12,800円
- **紹介料**: 50%（6,400円/件）
- **カテゴリ**: YouTube / ビジネス

### サムネイル
- noteと同じビジュアルをベースに
- 追加テキスト: 「紹介料50%」のバッジ
- 「7章の本編 + AIツール2個 + テンプレ4点 + ワークシート3点」を視覚的に列挙

### 概要文
```
現役12年のアドセンスコンサルが裏で構築した、
非属人チャンネル×リストマーケティングの完全ガイド。

【パッケージ内容】

【90日で、ここまで到達】
・アドセンス以外の収益源ができている
・非属人のまま信頼構築の仕組みが回っている
```

## 6. 公開後のアクション

### 初日
- Xで告知（導入章の一部を引用 + リンク）
- 既存リストにメール配信（あれば）

### 初週
- 購入者にお礼メッセージ
- レビュー依頼（noteのスキ、Brainのレビュー）

### 初月
- 購入データの確認（何人が導入章で離脱したか等）
- フィードバックをもとに微修正

### Premium の次のステップ
Lite/Standard の記事を数本出して実績を作った後にPremiumを公開するのが安全。
すでにLite/Standardの記事がある場合は、それらの記事からPremiumへの導線を追加する。
