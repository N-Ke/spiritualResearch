# Step1.5 Phase A 出力（Premium版テスト）: リサーチプロンプト

## クラスターA横断リサーチプロンプト

### リサーチプロンプト1: 非属人チャンネルの収益構造転換事例

```
以下のテーマについて、2024-2026年の情報を中心に調査し、レポートを作成してください。

テーマ: 非属人YouTubeチャンネルが、アドセンス以外の収益源（自社商品・アフィリエイト・コンサル・オンライン講座等）を構築した事例

調査してほしいこと:
1. 非属人チャンネル（顔出しなし、Vtuber、ゆっくり解説、まとめ系等）が広告収入以外で稼いでいる具体的な方法と数字
2. 属人チャンネルと非属人チャンネルで、収益多角化のアプローチがどう異なるか
3. 非属人から自社商品販売に移行したチャンネルの成功パターンと失敗パターン
4. 非属人でメール/LINE登録者を獲得している手法（概要欄、固定コメント、エンドスクリーン等）

出力形式:
- 各調査項目について3-5つの事例・データを箇条書きで
- 情報源のURL（確認できる場合）
- 「確認できた事実」と「推測」を明確に区別すること
```

### リサーチプロンプト2: リストマーケティングの非属人適用

```
以下のテーマについて調査してください。

テーマ: メールマーケティング/LINEマーケティングを「顔出しなし・キャラクター運営」で行う場合の設計手法

調査してほしいこと:
1. 非属人（キャラクター/ブランド名義）でのメール配信の成功事例
2. ステップメールの設計フレームワーク（特にBtoC個人向けコンテンツ販売の場合）
3. 「信頼構築」を「人格」ではなく「コンテンツの価値」で行うアプローチ
4. 非属人でのフリーギフト（リードマグネット）の種類と効果
5. メールの開封率・クリック率のベンチマーク（日本国内、個人コンテンツクリエイター）

出力形式: 上記と同じ
```

### リサーチプロンプト3: 自社商品作り（初心者向け）

```
以下のテーマについて調査してください。

テーマ: YouTubeコンテンツクリエイターが初めてデジタル商品を作って販売するまでのプロセス

調査してほしいこと:
1. 初心者が最初に作るべきデジタル商品の種類（有料note/Brain/PDF/オンライン講座等）と、各プラットフォームの2025-2026年の動向
2. 「自分の経験に値段をつける」ハードルを超えた人の体験談やアドバイス
3. 価格設定の心理学（アンカリング、松竹梅等）をデジタル商品に応用した事例
4. 最初の1個を作ってから、商品ラインを拡張するまでのロードマップ事例
5. AIを使ったコンテンツ制作（構成設計、執筆、図解、テンプレ作成）の最新手法

出力形式: 上記と同じ
```

## Phase Bへの申し送り

ユーザーが上記3つのリサーチプロンプトを実行し、結果を取得したら、Phase Cに進む。

リサーチ結果を読む際のポイント:
- 「確認できた事実」と「推測」の区別を保持すること
- 非属人の事例が少ない場合は属人事例を「翻訳」する方針で使う
- 数字（開封率、CVR等）は幅を持たせて引用する
