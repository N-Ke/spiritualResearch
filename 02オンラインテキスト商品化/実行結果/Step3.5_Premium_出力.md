# Phase B: 付属物の設計（WS1: アドセンス依存度セルフチェック）

## 1. 付属物の概要

- **名前**: アドセンス依存度セルフチェック
- **種類**: ワークシート（現状診断WS）
- **読者の使用シナリオ**: 第1章を読んで「依存の恐怖の正体は選択肢がないこと」と理解した直後、15分かけて自分のアドセンス依存度を10段階で数値化する。書き終えると「自分が何にどれくらい依存しているか」が可視化される。
- **読者がこれを使った後の状態**: 漠然とした不安が「依存度◯/10」という数字になり、「じゃあ何を変えればいいか」の方向性が見える

## 2. 構成要素

### メインの構成
1. **導入（目的と所要時間の説明）** — 3行
2. **セクション1: 収入構造チェック（3問）** — アドセンス収入の割合、他の収入源の有無
3. **セクション2: リスク耐性チェック（3問）** — アドセンスが半減/停止した場合の影響
4. **セクション3: 精神面チェック（3問）** — 毎月の不安度、コントロール感
5. **セクション4: 行動余力チェック（1問）** — 新しいことに使える時間
6. **集計 & 診断結果** — 10点満点でスコア化、3段階の診断
7. **次にやることの提案** — スコア別のアクション

### 読者向け使い方ガイド
- このWSの目的（3行）
- 記入のコツ（完璧を目指さない、正直に書く）
- 所要時間（15分）
- 結果の見方（スコア別の読み方）

### 記入例
- 「月30万のアドセンスYouTuber」のサンプル回答

## 4. 制作の全体手順

### 手順1: 第1章の該当箇所を読み直す
- 目的: WSの問いかけが本文と整合するか確認
- 具体的な作業: 第1章の「恐怖を数値化する」セクションを再読
- 所要時間: 約5分

### 手順2: 10個の問いを確認する
- 目的: 読者が答えやすい形か確認
- 具体的な作業: 上の「完成イメージ」の問いを読んで、抽象的すぎないかチェック
- 所要時間: 約5分

### 手順3: Phase Cのプロンプトを実行
- 目的: AIに完成品を生成させる
- 具体的な作業: Phase Cの「C-3: ワークシート制作」プロンプトをコピペ実行
- 所要時間: 約10分

### 手順4: 確認・調整
- 出力を確認し、本文の内容と矛盾がないかチェック
- 読者目線で「これを見て書けるか」を確認
- 所要時間: 約10分

## 2. 使い方ガイド

### このワークシートでできること
あなたのアドセンスへの依存度を10問・28点満点で数値化し、「何がどれくらい危険か」を客観的に把握できます。

### 使い方
1. ワークシートをコピペするか、印刷する
2. 上から順番に10問に答える（約15分）
3. 集計して合計点を出す
4. 診断結果を確認し、次に読むべき章を把握する

### 書くときのコツ
- 完璧に正確な数字は不要。「だいたいこれくらい」でOK
- 1つの問いに3分以上悩んだら、直感で選んでOK
- メモ欄は自由に。後で見返したとき「あの時こう感じてたんだ」と分かるように
- 正直に書くほど、このWSの価値が高くなる

### 使うタイミング
- 第1章を読んだ直後（最初の診断）
- 90日後にもう一度（変化の確認）
