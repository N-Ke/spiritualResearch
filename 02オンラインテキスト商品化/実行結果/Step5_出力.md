# Step5 出力: 公開設定

## 1. 基本設定（note）

- **タイトル**: "信頼＝顔出し"を疑ってみた｜非属人リスト取得の記録
- **本文**: Step3の修正済み記事をnoteエディタに貼り付け
- **有料エリアの設定**: 「ここまで読んでくれてありがとう。〜読んでみてください。」の直後に有料ラインを設置
- **価格**: 500円
  - 理由: 全体約5,500文字 + 独自フレームワーク2つ + 再現手順 + テンプレ例あり。初回記事なので実績作りも兼ねて500円。レビューが集まったら980円に値上げ検討。
- **カテゴリ**: ビジネス / マーケティング
- **タグ**: `非属人YouTube` `リストマーケティング` `アドセンス` `YouTube収益化` `リスト取得`
- **マガジン追加**: 「非属人×リストマーケ実験ノート」マガジンを新規作成し、この記事を追加（今後の実験レポートをまとめる）
- **公開範囲**: 全体公開
- **AI利用の表記**: 記事末に記載済み（「本記事の制作にAIツールを活用しています」）

---

## 2. 価格設定の根拠

- **文字数からの目安**: 5,500文字 → 300-800円が相場
- **テンプレ/ツール有無**: 独自フレームワーク2つ + 再現手順5ステップ + 概要欄テンプレ3パターン → +200円
- **提案価格**: 500円
- **価格設定のポイント**:
  - 初回は500円で実績・レビューを集める
  - 5件以上のスキ/購入がついたら980円に値上げ
  - 今後の実験レポートシリーズと合わせてマガジン化（3本セット1,480円等）
