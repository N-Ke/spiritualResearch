# Step5 出力: 公開設定（YouTubeタイトル）

## 1. 基本設定

### noteの場合

- **タイトル**: YouTubeタイトルの考え方｜センスじゃなく構造で作る
- **本文**: Markdown → noteのエディタに貼り付け（Step3の出力を使用）
- **有料エリアの設定**: 「構造①」の解説直後（「ここまでの…」のセリフ後）に有料ラインを設置
- **価格**: **300円**
  - 理由: 約4,500文字の具体的手順あり記事。初回商品のため低めに設定し、実績作りを優先
- **カテゴリ**: ノウハウ
- **タグ**: `YouTube` `タイトル` `初心者` `動画制作` `再生数`
- **マガジン追加**: （新規作成推奨: 「ウラアドの実験ノート」等）
- **公開範囲**: 全体公開
- **AI利用の表記**: 記事末に追加 → 「本記事の制作にAIツールを活用しています」

### Brainの場合

- **タイトル**: YouTubeタイトルの考え方｜センスじゃなく構造で作る
- **サムネイル画像**: シンプルな暗めの背景に白文字で「タイトルは構造で作る」+ 小さく「3つの視点」
- **価格**: **300円**
- **紹介料設定**: 50%（初回は露出を最大化するため高めに設定）
- **カテゴリ**: YouTube / 動画制作
- **概要文**: 「タイトルが思いつかない」——その原因は、センスじゃなく"考え方の手順"を知らないことかもしれません。YouTube12年の経験から気づいた、再生されるタイトルの3つの構造をまとめました。この記事を読めば、次の動画からタイトル作りの迷いが減るはずです。
- **返金設定**: 24時間以内の返金OK（初回は安心感を重視）
- **AI利用の表記**: 概要文末に追加

---

## 2. 価格設定の根拠

- **文字数からの目安**: 約4,500文字 → Lite相場で300-500円
- **テンプレ/ツール有無**: なし（追加料金なし）
- **初回商品のため**: 低めの300円で実績作りを優先
- **提案価格**: **300円**

**今後の価格戦略**:
- 販売数10部＋レビュー2-3件がついたら → 500円に値上げ
- 他の記事とセット（マガジン）を作ったら → セット割引で単品は維持

## 3. 公開前チェックリスト

### 記事本体
- [ ] タイトルが正しく表示されるか
- [ ] 有料ラインの位置が正しいか（構造①の後）
- [ ] 見出し（H2/H3）が目次に正しく反映されるか
- [ ] 図解プレースホルダーの箇所に画像を挿入（Canva等で作成）
- [ ] 誤字脱字の最終チェック
- [ ] AI利用の表記が記事末にあるか

### 設定
- [ ] 価格が300円に設定されているか
- [ ] カテゴリ/タグが設定されているか

### プレビュー
- [ ] スマホでプレビュー表示を確認
- [ ] 無料部分が「良い記事」として見えるか
- [ ] 有料ラインの表示が自然か

## 4. 公開後の確認事項

- [ ] 記事URLが正しく開けるか
