## 結果サマリ — 2週間で47件のリスト登録

### ビフォー・アフターの数字

**変更前（最初の2週間）:**

- リスト登録数: 5件
- 概要欄の文言: 「無料プレゼントはこちら →」

**変更後（次の2週間）:**
- 概要欄のクリック率: 0.5%
- リスト登録数: 47件
- 概要欄の文言: 「僕が3ヶ月試した結果をまとめました →」

顔も名前も出してない。
チャンネルの登録者数も変わってない。
変えたのは、概要欄の文言だけ。

### 「面白い発見があったんだけど」

数字の変化も驚いたけど、もっと面白い発見があった。

登録してくれた人のメール開封率が、想像以上に高かったんです。

「無料プレゼント」で集めたリストの開封率は30%くらい。

つまり、「タダだからもらっとこう」で登録した人と、「この人の情報をもっと知りたい」で登録した人では、質がまったく違う。

リスト取得の本質は、テクニックじゃない。
「次も会いたい」と思ってもらうことだった。

## やったことの詳細 — 何を変えたか、なぜ変えたか

### 仮説「信頼の正体は顔じゃなくて体験密度」

実験を始める前に、1つの仮説を立てた。

12年コンサルをやってきて、面白い発見があったんだけど——

属人チャンネルでも信頼構築に苦戦する人は多い。
顔を出しているのに「なんか信用できない」と言われるケース。

逆に、非属人なのに「この情報は信頼できる」と言われるケース。

違いは何かというと、**情報の一貫性と具体性**だった。

僕はこれを整理して、「信頼の3層モデル」と呼んでいる。

**第1層: 能力の信頼**（この人は詳しい）
→ 情報の質と深さで構築

**第2層: 一貫性の信頼**（この人はブレない）
→ 発信テーマの統一で構築

**第3層: 体験の信頼**（この人の情報で変われた）
→ フリーギフトや無料コンテンツでの成功体験で構築

属人チャンネルは「顔」で第2・3層を一気にカバーする。
非属人は各層を個別に積み上げる。

時間はかかるけど、構築できないわけじゃない。
むしろ、顔がないからこそ「情報の質そのもの」が際立つ。

考え方を変えるだけで、景色がガラッと変わる。

### 概要欄で変えた3つのポイント

仮説をもとに、概要欄を見直した。
僕が使ったのは「概要欄の3要素チェック」というフレームワーク。

**要素1: 具体的な体験**
→ 「何を試した結果なのか」を書く

**要素2: 読者メリット**
→ 「これを読むとどうなるか」を書く

**要素3: ハードル下げ**
→ 「無料」「3分で読める」等の心理的ハードルを下げる

変更前の概要欄をチェックしたら、こうなっていた。

```
変更前: 「無料プレゼントはこちら →（リンク）」
→ 要素1: なし / 要素2: 弱い / 要素3: 「無料」だけ
```

要素1がまったくなかった。
「何がもらえるか」は書いてあるけど、「誰の何の体験に基づくものか」が伝わらない。

こう変えた。

```
変更後: 「僕が3ヶ月試した結果をまとめました｜非属人チャンネルのリスト取得実験ノート（無料）→（リンク）」
→ 要素1: 「3ヶ月試した結果」/ 要素2: 「リスト取得実験ノート」/ 要素3: 「無料」
```

3要素がそろった。
たったこれだけで、登録率が変わった。

## うまくいったポイント — 再現のコツ3つ

### コツ1: 情報の具体性で信頼を稼ぐ

非属人チャンネルの最大の武器って何か。

実はこういう方法もあるんですよ——
**「顔がないから、コンテンツの質がダイレクトに伝わる」** ということ。

属人チャンネルは「この人好き」で見てもらえる。
非属人チャンネルは「この情報が良い」で見てもらうしかない。

でもそれは弱点じゃなくて、強みでもある。

情報の質が高ければ、余計なノイズなしに価値が伝わる。
曖昧な表現を排除して、具体的な数字と手順を書く。
それだけで、非属人の「信頼レベル」はぐっと上がる。

### コツ2: 最小ルートで結果を出す

「まずブランディングを整えてから」と思う人が多い。
ロゴ、声、世界観、トーン……。

僕の考えは違う。
**ブランディングは後回しでいい。**

まず「1つの動画 → 概要欄 → 登録」の最短ルートだけ整える。
概要欄の3要素を満たすだけで、まず結果を出す。

面白そうと思ったらまず試す。
結果が出てから、ブランディングを整えればいい。

まず小さく試してみませんか？

### コツ3: "次も会いたい"を設計する

リスト取得の本質は何か。

テクニックじゃない。
「この人の次の情報も知りたい」という感情を作ること。

フリーギフトは「もらえるもの」じゃなくて「体験の延長」として設計する。

動画で「面白い」と感じる。
もっと知りたくなる。
概要欄をクリックする。
期待通りの情報がそこにある。

この流れが自然にできていれば、非属人でもリストは取れる。

「概要欄にLINE置いたけど誰も登録しない」という声をよく聞くけど、それは導線の問題じゃなくて、「次も会いたい」の設計が足りないだけかもしれない。

## うまくいかなかったこと — 正直に書く

成功した部分だけ書くのは不誠実なので、失敗も書く。

**失敗1: 「無料プレゼント」形式**

最初に試したのは、定番の「無料プレゼント」方式だった。
「◯◯の完全ガイドを無料でプレゼント！」的なやつ。

結果は散々。
登録はあったけど、メール開封率が低い。
「無料だからとりあえずもらっとこう」で登録した人ばかりで、登録後に反応してくれる人が少なかった。

この失敗から「タダで釣る」んじゃなくて「体験の密度で惹きつける」という仮説にたどり着いた。

**失敗2: CTA位置を動画の冒頭にした**

「最初に言っておけば印象に残るはず」と思って、動画の冒頭でリスト誘導したら、離脱率がガッと上がった。

冷静に考えたら、そうだよね。
視聴者は「情報」を見に来ているのに、いきなり「登録してね」は逆効果。

信頼を稼ぐ前にお願いしても、通らない。
CTAは動画の後半、「この人の情報は良い」と感じてもらった後に置くのが正解だった。

## 再現する場合の手順 — あなたのチャンネルで試すなら

ここまで読んでくれた方向けに、再現手順をまとめた。

**ステップ1: 概要欄の現状を診断する**
自分のチャンネルの概要欄を開いて、3要素チェックをする。
- 具体的な体験: ある？ない？
- 読者メリット: 明確？曖昧？
- ハードル下げ: できてる？

**ステップ2: 「具体的な体験」要素を追加する**
「無料プレゼント →」を、「僕が◯ヶ月試した◯◯の結果 →」に変える。
チャンネルのテーマに合わせてカスタマイズ。

例:
- 料理チャンネル → 「30日間で試した時短レシピ7選をまとめました」
- 投資チャンネル → 「6ヶ月検証した積立投資の実績レポート」
- ガジェット → 「10台比較して見えた、本当に使えるデバイスリスト」

**ステップ3: フリーギフトの内容を見直す**
「もらえるもの」ではなく「体験の延長」として設計されているか確認。
動画の内容と自然に繋がるギフトになっているか。

**ステップ4: 2週間計測する**
変更前と変更後の数字を比較。
- 概要欄のクリック率
- リスト登録数
- 登録後のメール開封率

**ステップ5: 結果をもとに調整する**
2週間の結果を見て、3要素のバランスを微調整。
一発で正解にたどり着く必要はない。小さく試して、小さく直す。

## まとめ + 次の実験予告

この実験で分かったことを3つにまとめる。

**1. 信頼の正体は「顔」じゃなくて「体験の密度」**
非属人でも、信頼の3層（能力・一貫性・体験）を積み上げれば、リストは取れる。

**2. 概要欄は「3要素」を満たすだけで変わる**
具体的な体験 + 読者メリット + ハードル下げ。特に「具体的な体験」が抜けがち。

**3. リスト取得の本質は"次も会いたい"**
テクニックより先に、「この人の情報をもっと知りたい」という感情を作ること。

次は、「リスト登録後のステップメールで、非属人のまま商品販売まで持っていけるか」を実験する予定。

結果が出たら、またここで共有する。

まず小さく、概要欄の文言を変えることから試してみませんか？
