# Step2: 構成設計

---

## 【エージェント指示】

```
読み込むファイル:
  - 入力/ フォルダ内のキャラクター設計書
  - 入力/ フォルダ内のAI実行用プロンプト
  - 実行結果/Step0_出力.md
  - 実行結果/Step1_出力.md （Premiumの場合: 全章分）
  - 実行結果/Step1.5_PhaseC_出力.md
  - ナレッジ/テンプレートパターン集.md
  - ナレッジ/商品グレード定義.md
出力先: 実行結果/Step2_出力.md
次のステップ: Step3_本文執筆.md
```

**エージェントの実行手順:**
1. 上記ファイルを全て読み込む
2. Step0の出力から商品グレードを確認する
3. グレードに合ったプロンプト（Lite/Standard共通 or Premium）を選択して実行
4. 出力を `実行結果/Step2_出力.md` に保存する
5. ユーザーに確認を取る
6. OKなら次のステップ（Step3）へ

---

## このステップでやること

Step1で決めたテンプレパターンと、Step1.5で作ったオリジナル素材をもとに、記事全体の構成を設計します。

商品グレードによって設計内容が変わります:
- **Lite**: H2/H3アウトライン + 無料/有料境界
- **Standard**: 上記 + 図解設計 + 付属物仕様
- **Premium**: 章単位の設計 + 図解設計 + 付属物仕様

```
### 構成:

```

3. 【第1章を読む】（約◯分）
