# オンラインテキスト商品化 - 使い方ガイド

## このシステムでできること

**キャラクター設計書**と**ターゲットの悩み**の2つを用意するだけで、AIエージェント（Cursor / Anti-Gravity）が note/Brain の有料テキスト商品を半自動で制作します。

3段階の商品グレードに対応:

| グレード | 価格帯 | 内容 |
|---------|--------|------|
| **Lite** | 100-980円 | 1つの悩みを解決する記事 |
| **Standard** | 980-5,000円 | 深掘り記事 + テンプレ + 図解 |
| **Premium** | 5,000-30,000円 | 複数の悩みを包括的に解決する変革パッケージ |

---

## 始め方（3ステップ）

### ステップ1: 入力ファイルを配置

`入力/` フォルダに以下の2ファイルを入れてください:

| ファイル | 内容 |
|---------|------|
| キャラクター完全設計書（.md） | キャラの人格・判断基準の定義 |
| AI実行用プロンプト（.md） | AIがキャラとして振る舞う指示 |

### ステップ2: AIエージェントを起動

Cursor または Anti-Gravity でこのフォルダを開き、AIに以下のように伝えてください:

> 「AGENTS.md を読んで、オンラインテキスト商品化パイプラインを開始してください」

### ステップ3: 確認しながら進める

AIが各ステップを順番に実行し、出力するたびに確認を求めます。

- **「OK」**と言えば → 次のステップへ
- **「ここを直して」**と言えば → 修正してから再確認

あなたの作業は**最初の入力**と**各ステップの確認**だけです。

AIは各ステップの完了時に **「◯/◯ステップ完了」** と進捗を表示するので、全体のどこにいるかが常に分かります。

## あなたがやること / AIがやること

| あなたがやること | AIがやること |
|---------------|------------|
| キャラ設計書を用意する | フォルダからプロンプトを自動読み込み |
| 各ステップの出力を確認する | 前ステップの出力を自動引き継ぎ |
| 修正指示を出す（必要な場合のみ） | 出力を `実行結果/` に自動保存 |
| Step0.7/Step1.5のリサーチを実行（省略可） | グレード判定・パターン選択を自動実行 |

## 商品グレードの選び方

### Lite を選ぶとき
- 1つの悩みを解決するだけで十分
- まず1本出して実績を作りたい
- 確認回数: 約9回 / 制作時間: 1-2時間

### Standard を選ぶとき
- 悩みを深掘りしてテンプレも付けたい
- 確認回数: 約13回 / 制作時間: 3-5時間

### Premium を選ぶとき
- 複数の悩みをまとめて解決するパッケージを作りたい
- 確認回数: 約21回 / 制作時間: 8-15時間

## 推奨する制作順序

1. まず **Lite で1-2本** 出して、パイプラインに慣れる
2. 次に **Standard で1本** 出して、付属物つきの制作に慣れる
3. Lite/Standard で実績ができたら **Premium に着手**

## よくある質問

**Q: 市場検証（Step0.7）は必須ですか？**
A: 省略可能です。「スキップ」と言えば、Step0の商品企画そのままで進みます。ただし、実施すると競合との差別化ポイントが明確になり、商品の売れやすさが大きく変わります。

**Q: リサーチ（Step0.7 / Step1.5）はどうやるのですか？**
A: AIが「リサーチ用プロンプト」を生成するので、それをChatGPTのDeep ResearchやPerplexity等にコピペして実行し、結果をこのチャットに貼り付けるだけです。

**Q: リサーチ（Step1.5）は必須ですか？**
A: 省略可能です。「スキップ」と言えば、AIの知識だけで進みます。ただし、リサーチした方がオリジナル性が高まります。

**Q: 途中で止めて後日再開できますか？**
A: はい。`実行結果/` フォルダに各ステップの出力が保存されているので、「Step◯から再開して」と伝えれば続きから進められます。

**Q: 出力内容が気に入らない場合は？**
A: 「ここをこう直して」と具体的に指示すれば、AIが修正します。何度でも修正可能です。

**Q: 同じキャラで複数の商品を作れますか？**
A: はい。Step0で商品ラインが複数設計されるので、「次は◯◯の記事を作って」と言えば、Step1から別の商品を制作できます。
