# D. 日常と所有物

綾の生活空間・小物・季節感を統一する。**Bロール（イメージカット）の素材辞書** として機能させる。

## 設計原則

1. **作り込みすぎない** — 雑誌的な完璧な部屋はNG。生活感が「少しだけ」見える状態が、視聴者の同期を取りやすい。
2. **トレンド感を入れない** — 韓国インテリア／北欧モダン／TikTok映え系は避ける。**「10年後に見ても古びない」素朴さ** を選ぶ。
3. **小物に物語を載せる** — 紅茶、窓辺、ポトスは引継ぎ書のエピソード素材と直結。台本で言及できる小物だけを置く。

## 居住設定

| 項目 | 仕様 |
|---|---|
| 場所 | 都内近郊の小さなマンション／アパート2階。駅から徒歩10〜15分の住宅街 |
| 間取り | 1LDK か 1DK |
| 同居 | 一人暮らし。ペットなし |
| 仕事 | 詳細は曖昧化。発信は数年前から。昼間の仕事については **直接言及しない**（視聴者の同期を取りやすくするため）|
| 家族関係 | 直接的言及は避ける（必要な動画でだけ「実家を出て5年」程度の薄い言及） |

## 季節バリエーション

週1投稿で季節更新するため、**4季のシーンセットを事前定義**する。

### 春（3〜5月）
- 服装: 白シャツ＋アイボリーカーディガン（基準画像）／薄いブルーグレーのニット
- 背景: 桜の散歩道／窓辺の柔らかい朝光／紅茶＋桜餅（時々）
- 紅茶: ダージリン（春摘み）→ 言及してOK
- 香り言及: ラベンダー、洗いたてのリネン

### 夏（6〜8月）
- 服装: 白の半袖リネンシャツ／ベージュのワンピース（袖丈は上腕まで）
- 背景: 朝の散歩道（早朝）／扇風機（古いタイプ）／麦茶ではなく **アイスティー**（紅茶世界観の維持）
- 紅茶: 水出しダージリン
- 香り言及: 雨の匂い、青い葉

### 秋（9〜11月）
- 服装: ベージュ／カーキのカーディガン＋白シャツ／薄手のニット
- 背景: 銀杏並木の散歩道／紅茶＋焼き菓子／日暮れの早い窓辺
- 紅茶: アッサム、ロイヤルミルクティー
- 香り言及: 焼き菓子、紅茶のスパイス

### 冬（12〜2月）
- 服装: 厚手の白／アイボリーのニット／ロングカーディガン／ストール（ベージュ）
- 背景: 朝の冷えた窓辺／湯気の濃い紅茶／ひざ掛け／間接照明強め
- 紅茶: チャイ、シナモン入り
- 香り言及: 蜜柑の皮、湯気

## 食・飲み物のルール

- 紅茶: 中心。各動画で1回は触れてよい。
- コーヒー: 言及しない（紅茶世界観の維持）
- 食事: 簡単な朝食（トースト＋ジャム＋紅茶）／手作り感のある夕食（白いお皿）。**外食シーンは出さない**（一人で過ごす世界観の維持）
- お菓子: 焼き菓子（クッキー・スコーン）。和菓子は季節時のみ（桜餅・栗）。コンビニスイーツは出さない
- アルコール: 出さない（過去の飲み会エピソードを除く）
