# B. 声の設計（Irodori-TTS VoiceDesign）

## 設計方針（一行）

**「夜21〜23時、イヤホンで聴いた人がホッとできる声」** を作る。

## 声のコンセプト

視聴者は疲弊した30〜50代女性。寝る前のながら視聴が中心。元気・明るさ・張りのある声は、視聴者の今の心情と乖離して逆効果になる。
綾の声は **同じ温度に降りてくる声** であるべき。情報を「伝える」のではなく、隣で「ぽつりぽつり話している」音像を目指す。

## パラメータ方向性

| 軸 | 方向 | 補足 |
|---|---|---|
| 性別・年齢感 | 女性／30代前半 | 設定年齢33歳と一致 |
| 声質（音域） | 中低音（alto 寄り） | 高すぎる声はNG。落ち着きと温度感を優先 |
| ピッチ | 標準より -2〜-5% | 低めだが暗くはない |
| テンポ | 標準より -10〜-15%（毎分280〜300字程度） | ゆっくりめ。ただし眠くならない速さ |
| 抑揚 | 控えめ（フラット寄り） | 朗読的な抑揚はNG。独白に近づける |
| 語尾 | 下げ基調 | 上げると煽り・売り言葉風になり禁止表現と衝突 |
| 息の混ざり | やや多め | 機械感を消す。ただし吐息で性的にしない |
| 間（句読点） | 標準より長め | 読点 +20%、句点 +30%、段落間 +50% |
| 感情の振れ幅 | 平静〜微笑（softness） | 喜怒哀楽の振れは小さく |
| スタイル | 独白寄り 7：対話 3 | アナウンス・朗読は0 |

## Irodori-TTS VoiceDesign 仮パラメータ

実機テストで微調整する前提の **初期値**。

```yaml
speaker:
  gender: female
  age_band: adult        # 30代前半相当
voice:
  pitch_shift: -3        # semitones（中央値からの差）
  speed: 0.88            # 標準=1.0
  breathiness: 0.40      # 0=なし / 1=多い
  intonation: 0.30       # 0=平坦 / 1=豊か
  energy: 0.55           # 0=弱 / 1=強。やや控えめ
prosody:
  pause_short_ms: 320    # 読点の間
  pause_medium_ms: 600   # 句点の間
  pause_long_ms: 1200    # 段落間
  end_tone: falling      # 語尾下げ
style:
  preset: monologue      # 独白寄り
  warmth: 0.70           # 温度感
  formality: 0.40        # ややくだけ気味（敬体は維持）
```

※ Irodori-TTS の正式パラメータ名と齟齬がある場合は、実装時に対応表を追記する。

## 参考音像（脳内モデル）

| 参考 | 採用したい要素 |
|---|---|
| 早見沙織のナレーション | 中低音の落ち着き、丁寧な間 |
| 新垣結衣の話し方（インタビュー時） | 押し付けない柔らかさ、自然な抑揚 |
| 滝沢カレンの読み上げ | ゆったりしたテンポ、独特の間（過剰にならない範囲） |

## NG音像

- 元気で明るい朝の挨拶系（テンション高い女性YouTuberの声）
- アナウンサー的な張りのある通る声（押し付けがましさが出る）
- アニメ調のかわいい・甘い声（ターゲット年齢層と乖離）
- ASMR的な囁きすぎる声（過度に親密で逆に警戒される）
- 説法・説教調（綾の語り口ルール「推量表現で押し付けない」と矛盾）
- 語尾上げの売り言葉風（「ね↑」「よ↑」は禁止）
- 棒読み・抑揚ゼロのロボット感

## シーン別の使い分け（仮）

| 動画パターン | 抑揚 | テンポ | 補足 |
|---|---|---|---|
| 体験談×原理解説型（自己開示パート） | やや上げる | さらに遅く | 思い出す感じ。間を多めに |
| 実践メソッド型（手順説明パート） | 標準 | 標準 | 聴き取りやすさ優先 |
| 反論処理型（疑問の先回り） | 抑揚を少し付ける | 標準 | 視聴者の心の声を代弁する語り口 |
| 締めのCTA | 控えめに | 遅く | 押し付けない最後の一押し |

## 検証手順（次にやること）

1. 上記パラメータで Irodori-TTS にサンプル30秒生成
2. 想定視聴環境（イヤホン・21時以降）で試聴
3. 以下の3点をセルフ評価
   - **温度感**: 機械的ではないか
   - **疲弊度との一致**: 疲れて聴いて違和感ないか
   - **眠気**: 心地よい眠気か／退屈な眠気か
4. パラメータ微調整 → サンプル v2 → 比較
5. v3 で確定 → 本フォルダの `voice_v_final.yaml`（仮）に保存

## 検証用サンプル台本（30秒・体験談調）

```
こんばんは、綾です。
今日はね、「頑張ることをやめた話」をしようと思います。
……正確に言うと、やめざるを得なかった、んですけど。
あの頃の私は、毎朝30分、感謝のワークをやっていて。
やっているのに、なぜかどんどん、苛立ちが増えていったんですね。
今日は、その続きの話をします。
```

句読点と「……」の位置を、上記 prosody 設定で再現できるか確認する。
