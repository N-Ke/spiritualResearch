## Step 2: 動画→商品の導線設計

ワークフロー06の「動画=What / LINE=Why / 商品=How」の原則に基づき、コンセプトv2の収益設計と統合。

```
YouTube動画（無料）= What
  │ 「潜在意識は脳の仕組み。科学的にこう説明できる」
  │ 対応: C1（科学的根拠）+ C3（3ヶ月の壁の正体）
  │ → 視聴者:「なるほど、怪しくないんだ」「仕組みは分かった」
  │ → CTA: 「具体的なワークを受け取りたい方はLINEへ」
  ▼
LINE特典（無料）= Why の深掘り＋最初のHow
  │ 「なぜ5分のワークが効くのか」＋ 実際のワークシートPDF
  │ 対応: C6（時間確保 → 5分でOK）+ C4（方法論 → テンプレ）
  │ → 登録者:「5分ならできる」「やってみよう」
  │ → ステップ配信で信頼構築 + C5・C11を無料でカバー
  │   （「雑念は正常」「段階的ステップ」等の初心者向け情報）
  │ → 商品案内
  ▼
Lite商品（有料・最初の1本）= How の詳細
  │ 対応: C3（3ヶ月の壁の正体と突破法）+ C7（失敗からの再スタート）
  │ → 購入者:「だから効かなかったのか」「今度こそ正しいやり方で」
  ▼
Standard商品（有料・2本目）= How の深化
  │ 対応: C2（バレない実践法）+ C5（自己疑念の克服）+ C9（両立）
  │ → 購入者:「環境が整った」「周囲を気にせず続けられる」
  ▼
バックエンド（有料・将来）= 伴走
  │ 対応: C10（コミュニティ）+ C11（実践スキルの深化）
  │ → 参加者: 長期継続＋仲間＋定期フィードバック
```

### 導線チェック（3問）

1. このテーマで動画を10本以上イメージできるか？ → **YES**（脳科学の研究は無限、体験談も蓄積される）
2. 動画後に「この商品が欲しい」と自然に思えるか？ → **YES**（動画=Whatで興味、商品=Howで具体手順）
3. 動画が What・商品が How に役割分担できているか？ → **YES**（動画は「なぜ効くか」、商品は「どうやるか」）

## テーゼ「頑張らない」と商品の整合性メモ

テーゼ「頑張らなくていい。仕組みを理解すれば、静かに変わり始める」と、30日間プログラム・3ヶ月講座は一見矛盾する。以下の定義で整合を取る：

- **「頑張らない」＝ 仕組みを理解せずに意志力だけで続けること**をやめる
- **「仕組みに任せる」＝ 脳科学で"なぜ効くか"を理解した上で、最小限の行動（5分/日）を脳に預ける**
- 30日プログラムは「毎日30分の修行」ではなく「毎日5分、脳のメンテナンスを仕組み化する30日」
- 3ヶ月講座は「3ヶ月頑張る」ではなく「3ヶ月かけて脳の配線が自然に変わるのを見守る＋仲間がいる環境」

→ 商品名・コピーでも「〇日間チャレンジ」「〇ヶ月マスター」等の努力を暗示する表現は避け、「脳のメンテナンス」「仕組み化」等のワードで統一する。
