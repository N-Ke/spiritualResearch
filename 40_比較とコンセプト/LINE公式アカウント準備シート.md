**綾の口調チェック**:
- [x] 断定しない（「きっと役に立つと思います」）

## 3. LINE登録特典（リードマグネット）の方向性

商品方向性シートで確定した2つの特典。Phase 1では**方向性のみ決定**、制作はPhase 2で行う。

### 特典1: 「脳科学式 5分リセットワーク」

| 項目 | 内容 |
|---|---|
| **形式** | PDF（1枚もの・A4表裏） |
| **対応クラスター** | C6（時間確保）＋ C4（方法論） |
| **内容の方向性** | 脳科学に基づいた5分間のワーク手順（ステップ形式）。「なぜこのワークが効くのか」の科学的根拠を1段落添える |
| **制作時間目安** | 1〜2時間 |
| **4条件チェック** | ①即効性: 5分で今日から試せる ②具体性: ステップ形式の手順 ③専門性: 脳科学の裏付け ④希少性: LINE限定配布 |
| **商品との連動** | ワークを試した人が「もっと深く知りたい」→ Lite商品（リスタートガイド）への橋渡し |

### 特典2: 「RASの仕組みが3分で分かる限定解説動画」

| 項目 | 内容 |
|---|---|
| **形式** | 限定動画（3〜5分） |
| **対応クラスター** | C1（科学的根拠への懐疑） |
| **内容の方向性** | RAS（網様体賦活系）の仕組みを図解で解説。「引き寄せの法則」が脳科学ではどう説明されるかを3分で伝える |
| **制作時間目安** | 3〜5時間（図解＋音声） |
| **4条件チェック** | ①即効性: 3分で「なるほど」 ②具体性: 図解で視覚的に理解 ③専門性: 脳科学の権威性 ④希少性: LINE登録者限定 |
| **商品との連動** | 「RASの仕組みが分かった」→ 「じゃあ具体的にどう活用するの？」→ ステップ配信 → Lite商品 |

### 特典の優先順位

**Phase 2で先に作るのは特典1（PDF）**。理由:
1. 制作時間が短い（1〜2時間 vs 3〜5時間）
2. PDF1枚なので修正が容易
3. 「5分ワーク」はコンセプトの核心であり、最初の価値証明として最適
4. 動画特典はPhase 4でYouTube動画の制作フローが確立してからの方が効率的

## 4. LINEステップ配信の方向性（Phase 3で詳細設計）

Phase 1では方向性のみ。ステップ配信の詳細はPhase 3（D-BE UNICKフレームワーク）で設計。

### ステップ配信でカバーするクラスター

商品方向性シート（修正版）より、C5・C11をLINEステップ配信（無料）でカバーする方針。

| 配信 | テーマ（仮） | 対応クラスター |
|---|---|---|
| 第1話 | 登録ありがとう＋特典配布＋自己紹介 | — |
| 第2〜3話 | 「瞑想で雑念が出るのは正常」＋「才能は関係ない」 | C5（自己疑念）＋ C11（実践の難しさ） |
| 第4〜5話 | 綾の体験談（EP3: 3ヶ月の壁 → EP4: 脳科学で腑に落ちた） | C3（信頼構築） |
| 第6話 | Lite商品の案内 | — |
