## コンセプトの核

### 誰に

**「潜在意識・引き寄せに興味はあるけど、理由がわからないと動けない人」**

- スピリチュアルに惹かれるが「怪しい」と感じている層
- 「なぜそうなるのか」を知りたい知的好奇心が強い層
- 他のスピ系チャンネルを見て「本当に？」と思った経験がある人
- 20〜40代、男女問わず（論理型アプローチなので男性比率がやや高まる可能性）
- 「やってみたけどうまくいかなかった」人（再チャレンジ層）

### 何を約束するか

**「潜在意識と引き寄せの"なぜ"を科学で解き明かし、"今日から何をするか"を毎回持ち帰れる」**

- 動画を見終わったとき、「なるほど、だからか」と腑に落ちている
- かつ「じゃあ今日はこれをやろう」という具体的なアクションが1つある
- 「永久保存版」として何度も見返す価値がある図解コンテンツ

### どう信じさせるか（権威・物語・体験のどれを主軸にするか）

**主軸: 科学的エビデンス（権威）＋ 視聴者の体験談（社会的証明）**

| 信頼の源泉 | 具体的な使い方 |
|---|---|
| 脳科学・心理学の研究 | RAS（網様体賦活系）、プラセボ効果、認知バイアス、神経可塑性、デフォルトモードネットワーク等の研究を引用 |
| 書籍・論文の引用 | FACTFULNESS、脳科学系書籍、心理学の実験結果 等を具体名で出す |
| 視聴者コメント・体験談 | 「実践してみたらこうなった」をコメント欄から拾い、動画内で紹介 |
| 図解の品質そのもの | 複雑な概念をわかりやすく可視化する能力自体が信頼になる |

### 何をしないか（境界・リスク回避）

| やらないこと | 理由 |
|---|---|
| 「必ず叶う」「確実に金持ちになる」等の断定 | 景表法リスク＋YouTube規約リスク＋科学的トーンと矛盾 |
| 特定の宗教・神格の推奨 | 炎上リスク＋ターゲット（論理型）と合わない |
| 霊視・チャネリング・オーラ等のオカルト系コンテンツ | ポジショニングが崩れる＋広告主リスク |
| 他チャンネルの批判・比較 | 炎上リスク＋ブランド毀損 |
| AI音声の使用 | 準属人の声が差別化の芯。2026年規約でもリスク要因 |
| 低品質な量産（週5本以上等） | 大量生産コンテンツ認定リスク＋品質維持不可 |
| 顔出し | 準属人のポジションを維持。将来の方針変更余地を残す |

## チャンネル設計

### 運用形態

**準属人型（KenjiSpirits参考）**

- **映像**: iPad/タブレット手描き図解 or ホワイトボード＋マーカー（KenjiSpiritsのペン＆ノートと視覚的に差別化）
- **音声**: 自分の声（落ち着いた語り口）
- **顔**: 出さない
- **キャラクター名**: チャンネル名と紐づけ（後述）

### メインジャンル

**G001: 潜在意識・マインド**

サブテーマとして金運・開運（G002）の要素も取り込み可能だが、軸はあくまで「潜在意識の仕組み」に置く。

### 差別化ポジションマップ

```
                  科学的・論理的
                      ▲
                      |
                      |    ★ あなたのチャンネル
                      |    （科学×実践ハイブリッド）
                      |
  非属人 ◄────────────┼────────────► 属人
                      |
           KenjiSpirits|     潜在意識ラボ/純
           （体験×実践）|     （概念解説×顔出し）
                      |
                      ▼
                  感覚的・スピリチュアル
```

### 動画フォーマット（テンプレート）

**メインシリーズ: 「Why → How → Do」型（10〜20分）**

| パート | 時間 | 内容 | 役割 |
|---|---|---|---|
| Hook（0:00-0:30） | 30秒 | 「〇〇を科学的に解明すると、△△だった」＋図解チラ見せ | 興味喚起・離脱防止 |
| Why（0:30-5:00） | 4.5分 | 脳科学・心理学の研究で「なぜそうなるのか」を図解解説 | 知的好奇心の満足・信頼構築 |
| How（5:00-10:00） | 5分 | その仕組みを使って「どうすれば変わるのか」を構造的に説明 | 理解の深化・「なるほど」体験 |
| Do（10:00-14:00） | 4分 | 「今日からできる実践ワーク」を1つ具体的に提示 | 行動化・保存価値 |
| Close（14:00-15:00） | 1分 | 寄り添いメッセージ ＋ LINE登録CTA ＋ チャンネル登録 | 信頼の仕上げ・リスト取得 |

**サブシリーズ候補:**

1. **「聴くだけ実践」シリーズ（30〜60分）**: アファメーション、瞑想ガイド、ワーク音声 → 再生時間でアドセンス稼ぎ
2. **「〇〇を30日やってみた」シリーズ**: 実践レポート → エンタメ性・社会的証明
3. **「1分で腑に落ちる」ショート**: メイン動画の核心部分を切り出し → チャンネル認知拡大

### 視聴維持設計（CH002から学んだ技法）

| 技法 | 適用方法 |
|---|---|
| 「飛ばし見=不足の証拠」型の意味付け | 「最後の実践ワークを飛ばすと、"理解だけで終わる人"のパターンに入ります」 |
| 冒頭ロードマップ | 「今日は3ステップで解説します: Why → How → Do」 |
| 名フレーズの挿入 | 「感謝の反対は当たり前」のような、1フレーズで腑に落ちる言い切りを毎回仕込む |
| 反復視聴CTA | 「このワークは1週間続けて、もう一度この動画を見ると理解が深まります」 |

## 収益設計

### 軸1: アドセンス

| 要素 | 戦略 |
|---|---|
| RPMの高さ | スピリチュアル系 RPM 500〜1,000円（一般の3-5倍） |
| 再生時間の稼ぎ方 | メイン15分＋サブ30-60分の2ライン |
| 反復視聴 | 「何度も見て腑に落とす」設計＋実践ワーク音声のリピート |
| 目標 | 6ヶ月: 月10〜20万円 / 12ヶ月: 月30〜50万円 |

### 軸2: リストマーケ（LINE誘導→バックエンド）

| フェーズ | 内容 | タイミング |
|---|---|---|
| Phase 0: 導線構築 | LINE登録→無料特典（ワークシートPDF＋限定解説動画） | チャンネル開設と同時 |
| Phase 1: リスト育成 | ステップ配信で信頼構築（7〜14日間） | 登録者1,000人〜 |
| Phase 2: 小商品テスト | コンテンツ販売（5,000〜30,000円）で反応を見る | 登録者3,000人〜 |
| Phase 3: バックエンド | オンライン講座 or 月額コミュニティ（市場反応で判断） | 登録者10,000人〜 |

**LINE特典の方向性（CH002参考）:**

- 「引き寄せが起きる脳の使い方」ガイドブック（PDF）
- 「潜在意識の書き換え実践ワークシート」
- 限定セミナー動画（メイン動画より踏み込んだ内容）
- アファメーションテンプレート集

→ **バックエンドの具体商品は未定のまま、Phase 0-1の導線だけは初期から構築する**

## リサーチからの根拠（市場調査レポートより）

| 根拠になったリサーチ | 一言 |
|---|---|
| 市場需要調査レポート（2026-04-25） | 日本スピ市場4.2兆円、RPM 500-1000円、非属人AI型で月83万の実績あり |

| CH003 金運上昇チャンネル分析 | 「属人ストーリー×多層マネタイズ」型 → 導線設計は参考、属人部分は不採用 |

| YouTube 2026年規約調査 | AI大量生産コンテンツのBAN強化 → 準属人×手描きは最も安全 |
| ラッコM&A売買事例 | リスト（メルマガ/LINE）付きチャンネルが高値で売買→リスト資産の構築は初期から |
