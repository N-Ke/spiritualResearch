## 10種類のストーリー型

### type_A：転落と気づき型
**参考:** 動画04
**向いているテーマ:** 反面教師として教訓を伝えたい
**推奨ジャンル:** 借金・多重債務、投資失敗、ギャンブル依存、浪費、退職金崩壊

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | 結末の先見せ（最悪の状態を冒頭で提示） |
| 過去回想 | 5% | 「N年前、彼は笑っていた」で時間を巻き戻す |
| 転落の始まり | 20% | 一見正しそうな選択が伏線になる |
| 段階的転落 | 40% | 3〜5段階で徐々に崩壊していく |
| 崩壊のクライマックス | 15% | 身近な人のセリフが刺さる一言 |
| 気づきと余韻 | 15% | 完全な救いではなく「わずかな希望」で終わる |

**感情曲線:** 衝撃→興味→危機感→呆れ→後悔→わずかな希望
**推奨ナレーション:会話比率:** 3:7（転落の過程は会話で臨場感を出す）

**ジャンル別の冒頭フック例:**
- 借金: 「督促状の山に埋もれたテーブル。そこに座る男の目は、もう何も映していなかった」
- 投資失敗: 「証券口座の残高はマイナス430万円。画面を見つめる手が震えていた」
- 浪費: 「クレジットカードが3枚とも止められた朝、彼女は初めて自分の通帳を見た」

---

### type_B：ハイブリッドN選型
**参考:** 動画05, 09
**向いているテーマ:** 具体的なNG行動やリスクを伝えたい
**推奨ジャンル:** 生活習慣の改善、やってはいけない行動系、注意喚起系

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | 視聴者への直接問いかけ or 警告 |
| 主人公紹介 | 10% | 共感できる設定（身近な悩みや不安） |
| N個の転落/教訓 | 60% | 各項目ごとに小さな物語。心理描写を必ず入れる |
| 崩壊 | 15% | すべての積み重ねが一気に爆発 |
| 結び | 10% | 教訓まとめ + 視聴者へのアクション提案 |

**感情曲線:** 好奇心→共感→段階的な危機感→崩壊の衝撃→気づき
**推奨ナレーション:会話比率:** 4:6（各N選の解説にナレーション比率をやや高めに）

**ジャンル別の冒頭フック例:**
- 健康: 「40代を過ぎてから絶対にやってはいけない生活習慣5選」
- 職場: 「職場で嫌われる人がやっている行動5選。あなたは大丈夫ですか？」
- 子育て: 「子供の自己肯定感を壊す親の言葉5選」

---

### type_C：比較群像劇型
**参考:** 動画11
**向いているテーマ:** 正解がない問題（選択肢で人生が分かれる）
**推奨ジャンル:** 人生の選択系、キャリア選択、子育て方針、治療法選択

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | 結末の断片 or 疑問提示 |
| キャラクター紹介 | 10% | 3〜4人の異なる背景・性格を提示 |
| 各キャラの選択 | 25% | 会話劇で自然に情報提供。各選択に明確な根拠 |
| 解説役登場 | 10% | 客観的視点を提供する人物 |
| 時間跳躍後の結果 | 35% | N年後、全員が何かしら後悔している |
| 結び | 15% | 「正解はない」＋「仲間の大切さ」で温かく着地 |

**感情曲線:** 興味→比較→学び→衝撃→驚き→温かさ
**推奨ナレーション:会話比率:** 2:8（会話劇で各キャラの考えを表現）

**ジャンル別の冒頭フック例:**
- キャリア: 「転職か残留か。同期4人が選んだ道と、5年後の現実」
- 子育て: 「塾か自由か。3組の家庭が選んだ子育て方針と、10年後の結果」
- 治療: 「手術か保存療法か。同じ病気でも、選んだ道で人生は変わった」

### type_D：対比構造型
**参考:** 動画07, 10
**向いているテーマ:** 生活習慣や考え方の違いを伝えたい
**推奨ジャンル:** お金の使い方、夫婦関係、健康管理、職場での振る舞い

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | N年後の格差を先に見せる |
| 2人の紹介 | 10% | 似た境遇だが性格が対照的な2人 |
| 分岐点×N回 | 55% | 各分岐点で「失敗の選択→成功の選択」を交互に描写 |
| 崩壊 | 15% | 失敗者に決定的な危機 |
| 結び | 15% | 失敗者視点の後悔 + 成功者の名言 + アクション提案 |

**感情曲線:** 切なさ→「あるある」共感→耳が痛い→絶望→学び・希望
**推奨ナレーション:会話比率:** 3:7（分岐点の描写は会話で見せる）

**ジャンル別の冒頭フック例:**
- 職場: 「同期入社の2人。10年後、片方は部長、もう片方は窓際族だった」
- 夫婦: 「同じ共働き世帯、年収も同じ。5年後、片方は離婚、片方はラブラブ旅行」
- 健康: 「同じ40代メタボ診断。3年後、片方はフルマラソン完走、もう片方は入院」

### type_E：メンター型
**参考:** 動画08
**向いているテーマ:** マインドセット変化や具体的な習慣改善
**推奨ジャンル:** 節約術、人間関係改善、自己啓発、ダイエット、子育て相談

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | 逆説的問いかけ（やめたら良くなった系） |
| 主人公の苦境 | 15% | 悪い習慣や考え方の具体的描写 |
| メンターとの出会い | 10% | 意外な人物が「答えを持つ人」として登場 |
| 教え×N個 | 45% | 難易度順に配置。各教えに具体的アクションを含む |
| 変化と成果 | 10% | 数字で変化を示す |
| 結び | 15% | 対比キャラの転落 + 主人公の成長 + 温かい結末 |

**感情曲線:** 共感→同情→驚き→学び→実践意欲→温かさ
**推奨ナレーション:会話比率:** 3:7（メンターの教えは会話で伝える）

**ジャンル別の冒頭フック例:**
- ダイエット: 「『食べないダイエット』をやめたら、なぜか3ヶ月で8キロ痩せた」
- 子育て: 「子供に『勉強しなさい』を言うのをやめたら、成績が上がり始めた」
- 節約: 「節約をやめたら、なぜか貯金が増え始めた。その理由は…」

### type_F：スカッと逆転型
**参考:** 動画06
**向いているテーマ:** 搾取からの防衛、権利保護
**推奨ジャンル:** パワハラ、モラハラ、詐欺被害、不当解雇、嫁姑問題

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | 問いかけ（あなたならどうしますか？） |
| 理不尽な状況 | 30% | 悪役の台詞を細かく描写し、視聴者の怒りを最大化 |
| 密かな準備 | 20% | 主人公が知識・証拠・法的根拠を集める |
| 逆転劇 | 25% | 象徴的な演出 + 法的根拠の提示 |
| 悪役の崩壊 | 10% | 因果応報を描写 |
| 結び | 10% | 主人公の新生活 + 今日からできるアクションリスト |

**感情曲線:** 怒り→憤り最大→期待→爽快感→因果応報→希望
**推奨ナレーション:会話比率:** 2:8（理不尽な状況と逆転は会話で見せる）

**ジャンル別の冒頭フック例:**
- パワハラ: 「もし上司に『お前の代わりはいくらでもいる』と言われたら、あなたならどうしますか？」
- 嫁姑: 「義母に毎日LINEで監視される生活。ある日、妻がとった行動とは」
- 詐欺: 「リフォーム詐欺で300万円を騙し取られた70歳女性。彼女は全額を取り返した」

### type_G：再起・成長型【新規】
**向いているテーマ:** どん底からの復活、再出発
**推奨ジャンル:** 病気からの回復、失業からの再起、依存症克服、離婚後の再出発

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | 復活後の姿を先に見せる（笑顔、活き活きした姿） |
| どん底の描写 | 20% | 主人公が最も辛い状態。周囲からの孤立も描く |
| きっかけ | 10% | 些細な出来事が再起のスイッチになる |
| 小さな一歩 | 25% | 失敗しながらも少しずつ前に進む過程を丁寧に描写 |
| 壁と挫折 | 15% | 一度はまた落ちかける。周囲の助けで踏みとどまる |
| 成長の実感 | 15% | 過去の自分と今の自分の対比。数字的な変化も示す |
| 結び | 10% | 「まだ途中」だが希望がある。視聴者への励まし |

**感情曲線:** 希望の予感→絶望→小さな光→応援→挫折の痛み→乗り越える喜び→温かい希望
**推奨ナレーション:会話比率:** 3:7（成長の過程は会話と内面描写で見せる）

**ジャンル別の冒頭フック例:**
- 病気回復: 「今、彼は毎朝5キロ走っている。1年前、ベッドから起き上がれなかった男が」
- 失業: 「月収50万円のサラリーマンが、ある日突然クビになった。1年後、彼は笑っていた」
- 依存症: 「アルコール依存症で家族を失った男。今、彼は断酒会のリーダーとして50人の仲間を支えている」

### type_H：選択の岐路型【新規】
**向いているテーマ:** 人生を変える重大な決断
**推奨ジャンル:** 転職・退職、離婚の決断、手術の決断、引っ越し・移住、介護の選択

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | 「あなたならどちらを選びますか？」と2択を提示 |
| 主人公の日常 | 10% | 選択を迫られる前の普通の日々 |
| 選択の発生 | 10% | 突然のイベントで決断を迫られる |
| 葛藤と情報収集 | 30% | 周囲の意見、データ、感情が入り混じる。夫婦の会話、友人の助言 |
| 決断の瞬間 | 15% | 最後に背中を押す「一言」がある |
| 選択後のリアル | 20% | 選んだ道の苦労と喜び。選ばなかった道への未練も正直に |
| 結び | 10% | 「正解は自分で作るもの」+今日からのアクション |

**感情曲線:** 日常→衝撃→迷い→揺れ→覚悟→不安→受容→希望
**推奨ナレーション:会話比率:** 3:7（葛藤は会話で描写する）

**ジャンル別の冒頭フック例:**
- 転職: 「年収800万円の安定か、やりたいことで年収400万円か。あなたならどちらを選びますか？」
- 離婚: 「子供のために我慢するか、自分の人生を生きるか。35歳の彼女が選んだ答え」
- 介護: 「親を施設に入れることは『見捨てる』ことなのか。息子が2年間悩んだ末に出した結論」

### type_I：連鎖崩壊型【新規】
**向いているテーマ:** 小さな問題を放置すると大惨事になる警告
**推奨ジャンル:** 借金の連鎖、健康の無視、人間関係の放置、家の修繕放置

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | 最終的な大崩壊のシーンを先見せ |
| 平穏な日常 | 10% | 「小さな問題」がすでに存在しているが気づかない |
| 最初の兆候 | 15% | 問題が顕在化するが「まだ大丈夫」と放置 |
| 連鎖の始まり | 25% | 1つの問題が2つ目を生み、2つ目が3つ目を生む |
| 加速する崩壊 | 20% | ドミノ倒しのように止められなくなる |
| 崩壊のピーク | 15% | すべてが同時に爆発。周囲の人も巻き込む |
| 結び | 10% | 「あの時、たった1つ対処していれば」+予防のアクションリスト |

**感情曲線:** 衝撃→安心→小さな不安→「え、まずい」→恐怖→絶望→学び
**推奨ナレーション:会話比率:** 3:7（連鎖の過程は会話と行動で見せる）

**ジャンル別の冒頭フック例:**
- 借金連鎖: 「リボ払い3万円から始まった地獄。気づいた時には借金総額が780万円になっていた」
- 健康無視: 「たかが肩こり、と思っていた。半年後に救急車で運ばれるとは、あの時は想像もしなかった」
- 人間関係: 「妻の『もういい』というLINEを既読スルーした。それが、10年間の結婚生活の最後のメッセージになった」

### type_J：救いの手型【新規】
**向いているテーマ:** 孤立した人が意外な助けで救われる
**推奨ジャンル:** 孤独、引きこもり、虐待、DV、ネグレクト、いじめ

| フェーズ | 配分 | テクニック |
|---------|------|-----------|
| 冒頭フック | 5% | 救われた後の穏やかなシーンを先見せ |
| 孤立の描写 | 20% | 誰にも相談できない主人公。助けを求めることすらできない |
| SOSのサイン | 10% | 言葉にできないSOSを発している主人公 |
| 気づく人の登場 | 15% | 意外な立場の人物がSOSに気づく（隣人、店員、教師、医師など） |
| 寄り添いと支援 | 25% | 急がない支援。信頼関係をじっくり築く過程 |
| 回復の兆し | 15% | 主人公が初めて自分の声で「助けて」と言える瞬間 |
| 結び | 10% | 完全な回復ではないが、もう一人じゃない。支援の窓口紹介 |

**感情曲線:** 安心の予感→胸が痛い→切なさ→温かさ→信頼→涙→希望
**推奨ナレーション:会話比率:** 3:7（寄り添いの過程は会話で見せる）

**ジャンル別の冒頭フック例:**
- 孤独: 「70歳の彼女の電話帳には、登録されている番号が3つしかなかった。でも今、彼女には毎朝『おはよう』と言ってくれる人がいる」
- DV: 「殴られる日に備えて、長袖のシャツを5枚買っていた。ある日、薬局の店員が声をかけてきた」
- いじめ: 「学校に行けなくなって1年。図書館の司書が、彼に毎週1冊の本を薦め始めた」

## 全型共通の鉄則ルール

### 冒頭フックのテクニック5種

| テクニック | 説明 | 例 |
|-----------|------|-----|
| 結末先見せ | 悲惨な結末 or 希望の結末を冒頭で提示 | 病院のベッドに男が一人。通知はゼロ |
| 直接問いかけ | 視聴者に「あなたなら？」と問う | もし上司に「お前の代わりはいくらでもいる」と言われたら？ |
| 逆説的フック | 直感に反する事実 | 「食べないダイエット」をやめたら、なぜか痩せ始めた |
| 格差提示 | 同じスタートから分岐した「今」を対比 | 同じ年収400万なのに、10年後には天国と地獄 |
| 警告 | 直接的な危機感 | あなたの健康破壊へのカウントダウンが始まっているかもしれません |

### 感情設計の絶対ルール

1. 必ず **「下降→気づき→わずかな希望」** の曲線を含む
2. **完全なハッピーエンドにしない**（余韻と考える余地を残す）
3. 視聴者が「自分ごと」として感じるポイントを **3箇所以上** 配置
4. クライマックスは **「身近な人の一言」** で設計する

### リアリティの基準

リアリティ基準は**ジャンルごとに設定**する。台本内で使用する数字・設定は、そのジャンルの視聴者が「ありえる」と感じる範囲にすること。

**設定方法:**
- 台本生成時に `{{REALITY_STANDARDS}}` 変数でジャンル固有の基準値を提供する
- 基準値はリサーチで収集した実際の数字を使用する

**ジャンル別リアリティ基準の例:**

| ジャンル | 基準項目 | 基準値 |
|---------|---------|--------|
| 借金 | リボ払い金利 | 15〜18% |
| 借金 | 多重債務者の平均債務額 | 300〜500万円 |
| 職場 | 平均年収（30代） | 400〜500万円 |
| 職場 | 転職後の年収変動 | -20%〜+30% |
| 夫婦 | 離婚率 | 約35%（3組に1組） |
| 夫婦 | 養育費の平均月額 | 4〜6万円 |
| 子育て | 大学4年間の費用 | 400〜800万円 |
| 子育て | 塾代（中学生） | 月2〜5万円 |
| 健康 | 糖尿病の医療費 | 月1〜3万円 |
| 健康 | 入院費（1週間） | 10〜30万円 |

### キャラクター設計

**主人公のルール:**
- 序盤から共感できる設定にする（弱さや不安を見せる）
- 「特殊スキル」を伏線として配置（元営業マン、元看護師など）
- 傲慢すぎる主人公は避ける
- ビジュアルは「普通の人」を意識
- **ジャンルのターゲット層に近い年齢・境遇にする**

**サブキャラの役割:**

| 役割 | 機能 |
|------|------|
| パートナー/家族 | 警告役 + 崩壊の演出 + クライマックスの名言 |
| メンター | 少ないリソースで幸せに暮らす「答えを持つ人物」 |

### ナレーション:会話比率の設計ルール

**デフォルト比率: 3:7（ナレーション30%：会話70%）**

| 部分 | ナレーションの役割 | 会話の役割 |
|------|-----------------|----------|
| 冒頭フック | 状況説明・先見せ | - |
| キャラ紹介 | 背景説明 | キャラの性格を表す一言 |
| 本編 | 時間経過・場面転換・心理描写 | ストーリーの推進力 |
| クライマックス | 最小限の状況説明 | 感情のピーク（名セリフ） |
| 結び | 教訓のまとめ | 最後の温かい会話 |

**比率の計算方法:**
- 台本全体の文字数のうち、ナレーション部分とセリフ部分をカウント
- ナレーション:セリフ の文字数比が設定比率（±5%）に収まること
- 会話にはセリフだけでなく、（演技指示）は含めない

### エンゲージメント施策

- 視聴者への問いかけ（コメント誘導）を **中盤に1回** 配置
- **「ここ大事です」「ゆっくりいきますね」** で緩急をつける
- **具体的な数字** を必ず入れる（リアリティと記憶定着）
- キーパーソンの名言セリフは **1本につき1〜2個** に絞る
- 最後に **「今日からできるアクション」** を1〜4個提示
