# 使い方ガイド（初心者向け）

> このシステムは、あなたが調べた「悩み」と「解決策」を渡すだけで、
> AIがプロ品質のストーリー型台本を自動生成してくれる仕組みです。
> どんなお悩み系ジャンルでも対応でき、フェイスレス動画自動生成システムとの連携でキャラクターも活用できます。

---

## 初期セットアップ（最初の1回だけ）

### 必要なソフトウェア（動画化する場合）

| ソフト | バージョン | 用途 | インストール |
|--------|-----------|------|-------------|
| Python | 3.10以上 | 画像・動画・音声生成 | https://python.org |
| Node.js | v18以上 | Remotion動画合成 | https://nodejs.org |

### APIキーの取得（動画化する場合）

| サービス | 用途 | 取得先 | 料金目安 |
|---------|------|--------|---------| 
| Google Gemini | 画像生成 | https://aistudio.google.com/apikey | 1枚約7円 |
| ElevenLabs | 音声生成 | https://elevenlabs.io | $5/月（Starter） |
| Kling AI | 動画変換 | https://klingai.com/global/dev | 1本約30円 |

※ 台本の生成だけなら、ソフトウェア・APIキーは不要です。AIチャットだけで台本が作れます。

## 制作フロー（全体像）

```
ステップ① キャラクター設計 → フェイスレスシステムでナレーターを作る（任意）
    ↓
ステップ② リサーチ → ターゲットの悩み・解決策を調べる
    ↓
ステップ③ 設定を選ぶ → ストーリー型 + ナレーション:会話比率を選択
    ↓
ステップ④ 台本生成 → AIに台本を作らせる
    ↓
ステップ⑤ 品質チェック → 失敗パターンをチェック
    ↓
ステップ⑥ 画像生成 → scripts/generate_images.py
    ↓
ステップ⑦ 動画変換 → scripts/generate_videos.py
    ↓
ステップ⑧ 音声生成 → scripts/generate_narration.py
    ↓
ステップ⑨ レイアウトJSON作成 → 配置指定
    ↓
ステップ⑩ 動画レンダリング → remotion/ で最終MP4を出力
```

1本あたりの制作コスト: **約1,700〜2,000円**

## 毎回やること

### ステップ① キャラクターを設計する（任意・初回のみ）

**フェイスレス動画自動生成システム** でナレーターキャラクターを作ります。

1. AIに「キャラクターを作って」と指示
2. 専門領域（例：「借金・債務問題」）を伝える
3. AIが自動でキャラクターを設計
4. 完成した設計書を保存しておく

**スキップしてもOK。** キャラクター設計なしでも、デフォルトのナレーター設定で台本を生成できます。

---

### ステップ② リサーチする

動画のテーマについて、2つの情報を調べます。

**A. ターゲットの悩み・欲求**
- YouTube・Yahoo知恵袋・X などで「世間がどんな不安を抱えているか」を収集
- 検索トレンドも参考に

**B. 解決策・情報**
- その悩みに対する具体的な制度や対処法
- リアルな数字（金額、期間、費用など）
- 専門家の見解

**調べ方の例（テーマ: リボ払いの恐怖）：**

| 項目 | 内容 |
|------|------|
| 悩み | リボ残高が減らない。毎月5000円返しているのに |
| 不安 | 利息がいくらか分からない。誰にも相談できない |
| 数字 | リボ金利15〜18%。100万円のリボで年間15〜18万の利息 |
| 制度 | 任意整理で利息カット。個人再生で元本5分の1。法テラスで無料相談 |
| 注意点 | 任意整理はブラックリスト5年。自己破産は7〜10年 |

**フェイスレスシステムの活用:**
フェイスレス動画自動生成システムの `SKILL_ターゲットリサーチ.md` を使えば、悩みを100〜150個自動で棚卸しできます。

---

### ステップ③ 設定を選ぶ

**A. ストーリー型を選ぶ**

迷ったら `auto` でAIに任せてOK。自分で選ぶなら：

| こんな動画を作りたい | 型 |
|--------------------|----|
| 「こうなったらヤバい」反面教師系 | **type_A** 転落と気づき |
| 「やってはいけない○選」リスト系 | **type_B** ハイブリッドN選 |
| 「どれが正解？」正解なし系 | **type_C** 比較群像劇 |
| 「天国と地獄」成功vs失敗の対比系 | **type_D** 対比構造 |
| 「こうすれば大丈夫」ポジティブ系 | **type_E** メンター型 |
| 「騙された→逆転」スカッと系 | **type_F** スカッと逆転 |
| 「どん底からの復活」感動系 | **type_G** 再起・成長型 |
| 「あなたならどうする？」選択系 | **type_H** 選択の岐路型 |
| 「放置したら大惨事」警告系 | **type_I** 連鎖崩壊型 |
| 「独りじゃない」救い系 | **type_J** 救いの手型 |

**B. ナレーション:会話比率を選ぶ**

| スタイル | 比率 | どんな動画向き |
|---------|------|-------------|
| 会話重視型 | 2:8 | ドラマチックな会話劇 |
| **ストーリー標準** | **3:7** | **迷ったらこれ【デフォルト】** |
| バランス型 | 5:5 | 解説多めのストーリー |
| 語り手型 | 7:3 | ナレーション主体の語り |
| カスタム | 自由 | 自分で数値を指定 |

---

### ステップ④ AIに台本を作らせる

1. `04_台本生成プロンプト.md` を開く
2. `{{変数}}` の部分を、ステップ②③の結果に書き換える
3. AIチャット（ChatGPT / Claude / Gemini など）に丸ごと貼り付ける
4. AIが台本を返してくる

**書き換える箇所：**

| 変数 | 何を書くか | 例 |
|------|-----------|-----|
| `{{GENRE}}` | お悩みジャンル | 借金・多重債務 |
| `{{TARGET_AUDIENCE}}` | ターゲット層 | 30〜50代男女 |
| `{{THEME}}` | テーマ | リボ払いの恐怖 |
| `{{TARGET_PAIN_POINTS}}` | 悩み | ・リボ残高が減らない ・利息が分からない |
| `{{SOLUTION_INFO}}` | 解決策 | ・任意整理で利息カット ・法テラスで無料相談 |
| `{{STORY_TYPE}}` | ストーリー型 | type_A（迷ったら `auto`） |
| `{{NARRATION_DIALOGUE_RATIO}}` | 比率 | 3:7（迷ったらデフォルト） |
| `{{TARGET_DURATION}}` | 尺（分） | 20 |
| `{{REALITY_STANDARDS}}` | リアリティ基準 | リボ金利15〜18%、債務平均300〜500万 |
| `{{CHARACTER_PROFILE}}` | キャラクター設計書 | （なければ空欄） |
| `{{ADDITIONAL_INSTRUCTIONS}}` | 追加指示 | 夫婦の会話を多めに |

---

### ステップ⑤ 品質チェック

AIが出力した台本を `02_失敗パターン回避チェックリスト.md` でチェック。

```
□ ターゲットが狭すぎない
□ 驚きポイントが3つ以上ある
□ ストーリーで包まれている
□ 説教が30%以下
□ 主人公に共感できる
□ ターゲット属性に配慮
□ 数値がリアル
□ ナレーション:会話比率が設定通り（±5%）
□ キャラクターボイスが一貫
```

NGがあればAIに「○○を修正してください」と指示して再生成。

## ステップ⑥〜⑩ 台本から動画を作る

台本が完成したら、以下の手順で動画を生成します。
詳細は `11_動画生成パイプライン.md` を参照してください。

### ⑥ 画像生成

```bash
cd scripts
python generate_images.py
```

### ⑦ 動画変換

```bash
python generate_videos.py
```

### ⑧ 音声生成

```bash
python generate_narration.py
```

### ⑨ レイアウトJSON作成

`remotion/src/data/` にJSONファイルを作成。

### ⑩ プレビュー＆レンダリング

```bash
cd remotion
npm run preview      # ブラウザでプレビュー確認
npm run render       # MP4動画を出力
```

## よくある質問

**Q. キャラクター設計書は必須ですか？**
→ いいえ。なくてもデフォルト設定で台本を生成できます。ただし、キャラクターを設計するとチャンネルの統一感が出て、視聴者のファン化につながります。

**Q. どのジャンルでも使えますか？**
→ はい。お悩み系であればどんなジャンルでも使えます。借金、健康、職場、夫婦、子育て、恋愛、介護など。

**Q. ナレーション:会話比率はどう選べばいいですか？**
→ 迷ったら **3:7**（デフォルト）がおすすめ。会話が多いほうがストーリーの臨場感が高く、離脱率が低い傾向があります。

**Q. AIが長い台本を途中で切った**
→ 「続きを出力してください」と送ればOK

**Q. AIの出力が気に入らない**
→ 具体的に「○○のシーンをもっと感情的にして」「ナレーション比率を下げてもっと会話を増やして」等で修正依頼

**Q. ストーリー型はどう選べばいいですか？**
→ `auto` でAIに任せるのが最も簡単。慣れたら自分で選んでください。迷ったらtype_D（対比構造型）が最も安定して伸びます。

**Q. 新しい型（type_G〜J）はどんな時に使う？**
→ type_G（再起・成長型）は病気回復や失業からの再起。type_H（選択の岐路型）は転職や離婚の決断。type_I（連鎖崩壊型）は「放置すると危険」系。type_J（救いの手型）は孤立した人の救済ストーリー。
