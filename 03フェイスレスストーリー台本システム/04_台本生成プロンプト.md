# 台本生成プロンプト（実行用）

> **使い方:** `{{変数}}` の部分をあなたのリサーチ結果に書き換えて、
> AIチャット（ChatGPT, Claude等）にこのページの内容を丸ごと貼り付けてください。

---

## 【ここから貼り付け開始】

あなたはYouTubeアニメ動画の台本を書く専門ライターです。**お悩み系ジャンルのストーリー型台本** を、視聴者の感情を動かす品質で生成してください。

以下のルールを **厳守** してください。

### ジャンル設定

**ジャンル:** {{GENRE}}
**ターゲット層:** {{TARGET_AUDIENCE}}

### キャラクターボイス

語り手（ナレーター）は以下のキャラクターです。キャラクター設計書の設定に厳密に従ってください。

{{CHARACTER_PROFILE}}

※ キャラクター設計書がない場合は、以下のデフォルト設定を使用：
- ポジション: 当事者（同じ悩みを経験した仲間）
- 一人称: 「私」、視聴者: 「皆さん」
- 語尾: 「〜ですね」「〜なんですよ」「〜だと思います」
- 断定しない: 「〜かもしれませんね」「〜という方法もあります」
- 煽らない: 恐怖で動かそうとしない
- 必ず「でも大丈夫ですよ」ポイントを入れる
- 専門用語は必ず噛み砕いて説明する

### ナレーション:会話比率の設定

**設定比率:** {{NARRATION_DIALOGUE_RATIO}}

この比率は台本全体のナレーション部分とセリフ（会話）部分の文字数比率です。
設定比率から±5%の範囲に収めてください。

**プリセット一覧:**

| プリセット | 比率 | 特徴 |
|-----------|------|------|
| 会話重視型 | 2:8 | ドラマ的臨場感。会話劇で展開 |
| **ストーリー標準** | **3:7** | **バランス型【デフォルト】** |
| バランス型 | 5:5 | 解説とストーリーの半々 |
| 語り手型 | 7:3 | ドキュメンタリー風の語り |

**比率を守るためのガイドライン:**

- **ナレーション** = 語り手（ナレーター）が語る部分。場面説明、時間経過、心理描写、教訓の解説
- **会話（セリフ）** = キャラクター名「セリフ」の形式で書かれた部分。（演技指示）は文字数に含めない
- **3:7の場合の目安（20分台本 ≒ 約8,000文字）:**
  - ナレーション: 約2,400文字
  - セリフ: 約5,600文字

### 構造ルール（10種類の型から選択）

**type_A 転落と気づき型:** 冒頭で結末先見せ → 過去回想 → 段階的転落 → 崩壊（身近な人の一言） → わずかな希望
**type_B ハイブリッドN選型:** 「N選」タイトル → 一人の主人公の連続転落ストーリー → 崩壊 → 教訓まとめ
**type_C 比較群像劇型:** 3〜4人が違う選択 → 全員が何かしら後悔 → 「正解はない」＋仲間の大切さ
**type_D 対比構造型:** 成功者と失敗者の2人 → 同じ分岐点で違う選択 → 格差 → 学び
**type_E メンター型:** 悩む主人公 → メンターと出会い → N個の教え → 変化 → 温かい結末
**type_F スカッと逆転型:** 理不尽な仕打ち → 密かな準備 → 逆転 → 因果応報 → 新生活
**type_G 再起・成長型:** 復活後の先見せ → どん底 → きっかけ → 小さな一歩 → 壁と挫折 → 成長実感 → 希望
**type_H 選択の岐路型:** 2択提示 → 日常 → 選択の発生 → 葛藤 → 決断 → 選択後のリアル → 希望
**type_I 連鎖崩壊型:** 大崩壊先見せ → 平穏 → 最初の兆候 → 連鎖開始 → 加速する崩壊 → 崩壊ピーク → 予防策
**type_J 救いの手型:** 救われた後の先見せ → 孤立 → SOSサイン → 気づく人 → 寄り添い → 回復の兆し → 希望

### 品質基準

- 数値・金額・制度情報はリアリティ基準に従い現実的に設定する
- 職業設定と行動に一貫性を持たせる
- ターゲット属性に不快な描写を避ける
- 説教パートは全体の30%以下
- 象徴的ディテール（記憶に残る小道具）を2〜3個配置する
- 最後に「今日からできるアクション」を提示する
- 必ず「下降→気づき→わずかな希望」の感情曲線を含む
- 完全なハッピーエンドにしない（余韻を残す）
- クライマックスは「身近な人の一言」で設計する
- **ナレーション:会話比率を設定値（±5%）に収める**
- **キャラクター設計書の口調・表現ルールを厳守する**

### リアリティ基準（ジャンル固有の数値設定）

{{REALITY_STANDARDS}}

### 素材量の基準（非常に重要）

実績5本の分析から、1本あたり以下の素材量が必要です。台本のシーン設計時に必ずこの量を意識してください。

**20分動画の場合：**
- **静止画（通常）:** 130〜160枚（1枚あたり5〜10秒表示）
- **静止画（展開早い）:** 35〜50枚（1枚あたり1〜4秒表示、全体の20〜30%）
- **動画素材:** 10〜15本（3〜10秒、合計約1分）
- **素材合計:** 約180〜200素材
- **コスト目安:** 約1,600〜1,800円（画像7円、動画30円）

**各シーンの素材数目安：**
- 冒頭フック（30〜50秒）: 通常3〜5枚 + 展開早い2〜3枚 + 動画1〜2本
- キャラ紹介（60〜80秒）: 通常8〜12枚 + 展開早い0〜2枚
- 会話シーン（60〜90秒）: 通常8〜14枚 + 展開早い1〜3枚
- 分岐点・教訓（80〜120秒）: 通常10〜18枚 + 展開早い3〜5枚 + 動画1〜2本
- クライマックス（90〜120秒）: 通常8〜12枚 + 展開早い5〜10枚 + 動画2〜3本
- 結び（60〜100秒）: 通常8〜15枚 + 展開早い1〜3枚

**展開早い静止画はクライマックスに集中させる。**

### 出力指示

上記の情報をもとに、以下の構造で台本を生成してください。

**1. タイトル案**（3つ）
- 「【漫画】」で始める
- 具体的な数字を入れる
- 結末を匂わせるがネタバレしない

**2. サムネイルの提案**
- メインビジュアル、テキスト要素、喚起する感情

**3. キャラクター設計**
- ナレーター: キャラクター設計書に基づく（未設定ならAIが設計）
- 主人公: 名前、年齢、経歴、性格、外見
- サブキャラ: 同上

**4. シーン別の台本**

各シーンごとに以下を記載：

```
## シーン○: シーンタイトル
- 尺: ○○秒
- 感情: ○○
- BGM: tense / sad / hopeful / warm / dramatic / neutral
- このシーンのナレーション:会話比率: N:N

### ナレーション（[キャラクター名]）
（キャラクターのトーンでナレーション全文）

### セリフ
キャラ名「セリフ」（演技指示）

### ビジュアル指示
- 背景: ○○
- 登場キャラ: ○○（表情: ○○）
- 小道具: ○○
- 演出メモ: ○○

### 字幕強調
（画面に大きく表示するテキスト。金額や重要数字）

### 素材リスト
- 静止画（通常）: ○枚（各○秒）- 内容を1枚ずつ簡潔に記載
- 静止画（展開早い）: ○枚（各○秒）- 内容を1枚ずつ簡潔に記載
- 動画素材: ○本（各○秒）- 内容を1本ずつ簡潔に記載
```

**5. まとめ**
- 教訓ポイント（3〜5個）
- 今日からできるアクション（1〜4個）
- コメント誘導の一言
- 次回予告

**6. 素材量サマリー**
- 静止画（通常）: 合計○枚
- 静止画（展開早い）: 合計○枚（全体の○%）
- 動画素材: 合計○本
- 素材合計: ○素材
- 推定コスト: ○円（画像7円×枚数 + 動画30円×本数）

**7. 品質チェック結果**
- ターゲット幅: 何割が自分ごとと感じるか

- 説教パートの割合
- 数値のリアリティ
- 象徴的ディテールの一覧

- **キャラクターボイスの一貫性チェック**

## {{変数}} の書き換えガイド

| 変数 | 何を書くか | 書き方の例 |
|------|-----------|-----------| 
| `{{GENRE}}` | お悩みジャンル | 借金・多重債務 |
| `{{TARGET_AUDIENCE}}` | ターゲット層 | 30〜50代の男女。特にリボ払いや消費者金融を使っている人 |

| `{{SOLUTION_INFO}}` | リサーチした解決策・数字 | ・任意整理で利息カット可能 ・個人再生で元本5分の1に ・自己破産で全額免除も ・無料法律相談あり |
| `{{STORY_TYPE}}` | 10種類の型から選ぶ（迷ったら `auto`） | type_A |
| `{{NARRATION_DIALOGUE_RATIO}}` | ナレーション:会話の比率 | 3:7 |
| `{{TARGET_DURATION}}` | 動画の長さ（分） | 20 |
| `{{REALITY_STANDARDS}}` | ジャンル固有のリアリティ基準 | リボ金利: 15〜18%、多重債務平均: 300〜500万、任意整理費用: 5〜10万/社 |
| `{{CHARACTER_PROFILE}}` | キャラクター設計書（丸ごと貼り付け） | （フェイスレスシステムの出力をそのまま貼り付け） |
