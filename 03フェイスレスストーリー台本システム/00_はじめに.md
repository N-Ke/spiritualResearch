# フェイスレスストーリー台本システム

## このシステムでできること

**どんなお悩み系ジャンルでも**、ストーリー型YouTubeアニメ動画の台本を、高い再現性で量産できるシステムです。

「老後年金_台本生成システム」で実証された成功法則（再生数13万〜33万回の実績）を、あらゆるジャンルに適用できるよう汎用化しました。

**10種類のストーリー型** と **13項目の品質チェック** を使って、誰が作っても一定以上の品質を担保します。

さらに、`フェイスレス動画自動生成システム` で設計したキャラクターをナレーターとして連動させることで、ジャンルに最適化された「声」で台本を生成できます。

## 対応ジャンル例

このシステムは以下のようなお悩み系ジャンルに対応しています（これらに限りません）：

| カテゴリ | ジャンル例 |
|---------|----------|
| お金の悩み | 借金・多重債務、老後資金、投資失敗、住宅ローン、生活費 |
| 仕事の悩み | 職場の人間関係、パワハラ、退職・転職、副業、独立開業 |
| 家庭の悩み | 夫婦問題・離婚危機、嫁姑問題、介護、相続 |
| 子育ての悩み | 不登校、受験、反抗期、教育費、ワンオペ育児 |
| 健康の悩み | 生活習慣病、メンタルヘルス、ダイエット、睡眠障害 |
| 恋愛の悩み | 婚活、恋愛依存、別れ、浮気・不倫 |
| 人間関係の悩み | 友人関係、ご近所トラブル、SNS疲れ、孤独 |

## 全体フロー

```
あなたがやること                            AIがやること
────────────────────────                ──────────────────
① フェイスレスシステムで              →   キャラクター設計書を生成
   キャラクターを設計
② テーマの悩みをリサーチ             →
③ 解決策・情報をリサーチ             →
④ 04_台本生成プロンプトに            →   ⑤ 台本を生成
   リサーチ結果を貼り付けて
   AIチャットに投げる
                                            ↓
                                       ⑥ 台本を受け取る

  ─── ここから動画化（scripts/ + remotion/ を使用）───

⑦ generate_images.py を実行           →   Gemini で画像生成
⑧ generate_videos.py を実行           →   Kling AI で動画変換
⑨ generate_narration.py を実行        →   ElevenLabs で音声生成
⑩ レイアウトJSONを作成                →
⑪ remotion/ で render                 →   最終MP4を出力
```

## 設計思想

```
成功台本の構造パターン    =  ストーリーの骨格（10型）
失敗回避チェックリスト    =  品質の守り（13項目）
キャラクター設計          =  当事者目線の語り口（フェイスレスシステム連携）
ナレーション:会話比率     =  臨場感のコントロール（デフォルト3:7）
```

この4つを掛け合わせることで、**ストーリーで感情を動かしながら、実用的な情報も伝える** 台本が、どんなジャンルでも生まれます。

## 連携システム

### フェイスレス動画自動生成システム

キャラクター設計・ターゲットリサーチ・コンセプト設計は、`フェイスレス動画自動生成システム` で行います。

| 機能 | 連携方法 |
|------|---------|
| キャラクター設計 | 設計書をそのまま台本プロンプトに組み込む |
| ターゲットリサーチ | 悩みデータベースを `{{TARGET_PAIN_POINTS}}` に使用 |
| コンセプト設計 | チャンネルコンセプトがナレーターの立ち位置を決定 |

## ナレーション:会話比率について

このシステムでは、台本のナレーション（語り手の解説）とセリフ（キャラクターの会話）の比率をコントロールできます。

| プリセット | 比率 | 特徴 | 向いているジャンル |
|-----------|------|------|-----------------|
| 会話重視型 | 2:8 | ドラマ的臨場感が高い。会話劇で展開 | 夫婦問題、職場の人間関係 |
| **ストーリー標準** | **3:7** | **バランス型【デフォルト】** | **全ジャンル** |
| バランス型 | 5:5 | 解説とストーリーの半々 | 制度解説系、投資失敗 |
| 語り手型 | 7:3 | ドキュメンタリー風の語り | 社会問題、歴史系 |
| カスタム | 自由 | ユーザーが自由に設定 | — |

デフォルトは **3:7**（ナレーション30%：会話70%）に設定されています。台本生成プロンプトの `{{NARRATION_DIALOGUE_RATIO}}` で変更可能です。
