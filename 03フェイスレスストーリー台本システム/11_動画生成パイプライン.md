# 動画生成パイプライン（汎用版）

> 台本から動画を生成するための技術仕様。
> 本ファイルは `老後年金_台本生成システム` のパイプラインを汎用化したものです。

---

## アーキテクチャ概要

```
台本.md
  ↓  (1) parse
script_data.json
  ↓  (2) generate_images
images/
  ↓  (3) generate_videos
videos/
  ↓  (4) generate_narration
narration/
  ↓  (5) layout.json 作成
remotion/src/data/layout.json
  ↓  (6) render
output/final.mp4
```

---

## ステップ1: 台本のパース

台本のマークダウンからJSONを生成する。

### 入力
```
## シーン○: シーンタイトル
- 尺: ○○秒
- 感情: ○○
- BGM: tense / sad / hopeful / warm / dramatic / neutral

### ナレーション
（ナレーション全文）

### セリフ
キャラ名「セリフ」（演技指示）

### 素材リスト
- 静止画（通常）: ○枚
- 静止画（展開早い）: ○枚
- 動画素材: ○本
```

### 出力（script_data.json）
```json
{
  "title": "動画タイトル",
  "genre": "ジャンル名",
  "narrator": "キャラクター名",
  "narration_dialogue_ratio": "3:7",
  "scenes": [
    {
      "id": 1,
      "title": "シーンタイトル",
      "duration_sec": 50,
      "emotion": "衝撃",
      "bgm": "tense",
      "narration": "ナレーション全文",
      "dialogues": [
        {
          "character": "キャラ名",
          "line": "セリフ",
          "direction": "演技指示"
        }
      ],
      "assets": {
        "images_normal": [
          {"id": 1, "duration_sec": 5, "description": "画像の説明"}
        ],
        "images_fast": [
          {"id": 1, "duration_sec": 2, "description": "画像の説明"}
        ],
        "videos": [
          {"id": 1, "duration_sec": 5, "description": "動画の説明"}
        ]
      }
    }
  ]
}
```

## ステップ2: 画像生成（generate_images.py）

### 仕様
| 項目 | 値 |
|------|-----|
| モデル | Gemini（Imagen 3） |
| 解像度 | 1280×720（HD） |
| プロンプト形式 | 英語（自動翻訳） |
| 出力先 | `images/scene_{N}/img_{M}.png` |
| コスト | 約7円/枚 |

### ジャンル別の画像プロンプトガイドライン

| ジャンル | 色調 | 雰囲気 | 避けるべき表現 |
|---------|------|--------|-------------|
| 借金・多重債務 | 暗め→徐々に明るく | リアリスティック。暗い部屋、書類の山 | 実在の企業ロゴ、カード番号 |
| 職場人間関係 | ニュートラル→対比で明暗 | オフィス、会議室、居酒屋 | 特定企業名、差別的表現 |
| 夫婦問題 | 暗い→逆転後に明るい | 家庭内。食卓、リビング | 暴力的すぎる描写 |
| 子育て | 温かい色調全体 | 家庭、学校、子供食堂 | 子供の過度に悲しい表情 |
| 健康不安 | 冷たい（病院）→温かい（回復） | 病院、自宅、公園 | グロテスクな医療描写 |

### 実行
```bash
cd scripts
python generate_images.py --config ../script_data.json --output ../images/
```

## ステップ3: 動画変換（generate_videos.py）

### 仕様
| 項目 | 値 |
|------|-----|
| サービス | Kling AI |
| 入力 | 静止画1枚 + モーションプロンプト |
| 出力 | 3〜10秒の動画（MP4） |
| コスト | 約30円/本 |
| 出力先 | `videos/scene_{N}/vid_{M}.mp4` |

### 実行
```bash
python generate_videos.py --config ../script_data.json --images ../images/ --output ../videos/
```

---

## ステップ4: 音声生成（generate_narration.py）

### 仕様
| 項目 | 値 |
|------|-----|
| サービス | ElevenLabs |
| 入力 | ナレーション + セリフテキスト |
| 出力 | WAVファイル（シーンごと） |
| コスト | 約$1〜2/本 |
| 出力先 | `narration/scene_{N}.wav` |

### キャラクターごとの声の対応
- ナレーター → ElevenLabsの指定ボイスID
- キャラクターごとに異なるボイスIを設定可能
- `voice_config.json` で管理

```json
{
  "narrator": "voice_id_1",
  "characters": {
    "キャラ名1": "voice_id_2",
    "キャラ名2": "voice_id_3"
  }
}
```

### 実行
```bash
python generate_narration.py --config ../script_data.json --voices voice_config.json --output ../narration/
```

## ステップ5: レイアウトJSON作成

`remotion/src/data/layout.json` に、各シーンの素材配置を定義する。

```json
{
  "fps": 30,
  "width": 1920,
  "height": 1080,
  "scenes": [
    {
      "id": 1,
      "startFrame": 0,
      "endFrame": 1500,
      "bgm": "tense",
      "layers": [
        {
          "type": "image",
          "src": "images/scene_1/img_1.png",
          "startFrame": 0,
          "endFrame": 150,
          "position": "center",
          "animation": "fade_in"
        },
        {
          "type": "video",
          "src": "videos/scene_1/vid_1.mp4",
          "startFrame": 150,
          "endFrame": 300
        },
        {
          "type": "narration",
          "src": "narration/scene_1.wav",
          "startFrame": 0,
          "endFrame": 1500
        },
        {
          "type": "subtitle",
          "text": "テロップテキスト",
          "startFrame": 100,
          "endFrame": 250,
          "style": "emphasis"
        }
      ]
    }
  ]
}
```

## ステップ6: プレビュー＆レンダリング

### プレビュー
```bash
cd remotion
npm run preview
```

### レンダリング
```bash
npm run render
# 出力: output/final.mp4
```

### 出力仕様
| 項目 | 値 |
|------|-----|
| 解像度 | 1920×1080 |
| FPS | 30 |
| コーデック | H.264 |
| 音声 | AAC 128kbps |
| 推奨尺 | 20〜25分 |
