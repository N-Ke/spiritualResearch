## 4. 動かしてはいけない不変ルール

> 「繰り返し参照される最重要前提」のみここに **見出しと 1 行ポインタで** 置く。
> 詳細は各正本ファイルが SoT（Source of Truth）。**本ファイル §4 は索引で、内容を持たない**。

### 制作テンプレ（媒体別）

- **YouTube ロング動画**：Why（科学）→ How（体験）→ Do（実践）の15分構成
- **note 有料記事（Lite）**：3,000-6,000字、H2×5-7、付属物なし
- **note 有料記事（Standard）**：8,000-15,000字、図解3-5枚、テンプレ1-2点付属
- **note Premium**：20,000-50,000字、5-10章、図解8-15枚、ツール+テンプレ+ワークシート付属
- **テンプレ本体**：[`../02オンラインテキスト商品化/`](../02オンラインテキスト商品化/) 配下の Step0〜Step5 と ナレッジ/

### 商品名・命名ルール（不変）

- **「脳科学」を商品名・コピーには入れない**（説明手段として中で使う、変化＝ベネフィットで売る）
- **NG語彙**：「ありのまま」「輝く」「素敵」「最高の人生」「絶対」「必ず」「100%」「劇的」「苦しみから抜け出す」「意識が低い」
- **シグネチャー語**：「やすみグセ」「羽をやすめる」（ユーザー合意・2026-04-29）
- **AI 特有の謎表現を排除**：「降ろす」「○○時間」「○○・しおり」「夜、」（文脈なし差し込み）等

### 視聴者・ターゲット（不変）

- **ターゲット**：田中優子（37歳・人事部課長・既婚・小学生子持ち・親介護フェーズ・スピ懐疑層）
- **中核痛点**：A 全部試した疲労型（C3+C7+C12）+ B ポジティブ強要疲労型（新規 C13）
- **訴求軸**：（1）「3ヶ月で何も変わらなかった」呪いから解放 +（2）「ワクワク・感謝の強要」から自由

### 安全配慮（不変）

- 医療的主張禁止（病気・鬱・不眠が治る等）
- 景表法違反禁止（絶対叶う・100%効く等）
- 薬機法違反禁止
- 希死念慮レベルの相談者には専門機関（精神科・カウンセリング）への送り出し動線必須

### 既存リサーチ運用との関係（不変・衝突解決）

- **既存の `_registry.md` / `_genre_registry.md` / `_genre_sheet.md` は維持**（チャンネル調査・ジャンル分析の SoT）
- **ミナト方式の Block Ledger は新規制作物のみに適用**：自分の動画（OWN-VID-XX）、自分の note記事（OWN-NOTE-XX）
- **既存 ID 規約（CH###/G###/VID###）は維持**、新規制作物 ID（OWN-XX）は別系統で衝突しない

### レビュー運用（不変）

- 新作 v1 完成後は3体並列レビュー推奨（セールスライティング観点／ターゲット心理観点／綾哲学整合観点）
- 複数成果物が揃ったら横断レビュー推奨

### 進捗管理ルール（2026-04-29 追加）

- **Block Ledger 方式**を採用（[`block_ledger_template.md`](block_ledger_template.md)）
- **新規 Block は本テンプレからコピー**して `blocks/works/{ID}.md` または `blocks/assets/{name}.md` に作成
- **更新コマンド**：`/handoff-update {block}`
- **空欄禁止**：`<TBD: 理由>` または `N/A: 理由` で埋める
