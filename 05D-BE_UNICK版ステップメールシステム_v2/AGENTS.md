# D-BE UNICK ステップメールシステム v2 - エージェント設定

## エージェント概要

本エージェントは、D-BE UNICK フレームワーク（8要素）に基づき、
高品質なステップメールシリーズを戦略設計から品質チェックまで一気通貫で生成します。

## スキル参照

- メインスキル: `SKILL.md`
- ナレッジ: `ナレッジ/` フォルダ内の全ファイル

## 実行フロー

```
入力/ → Phase1(戦略設計) → Phase2(配列設計) → Phase3(本文生成) → Phase4(品質チェック) → 実行結果/
```

## 初回実行時の手順

1. ユーザーに商品/サービス情報を `入力/` フォルダに配置するよう案内
2. `SKILL.md` に従い、Phase 1 から順に実行
3. 各 Phase の出力を `実行結果/` に保存
4. Phase 4 で品質チェックを実施し、必要に応じて Phase 3 を再実行

## 倫理ガードレール

以下に該当する内容は生成しないこと：
- 読者を責める・恥ずかしめる表現
- 嘘の限定性や根拠のない煽り
- 侮辱的・攻撃的な表現
- 「買わない人はダメ」的なプレッシャー
- 読者を「客」として扱う表現（「仲間」として扱う）
