# D-BE UNICK ステップメールシステム v2

## 概要

D-BE UNICK フレームワーク（8要素）に基づき、高品質なステップメールを戦略設計から本文生成・品質チェックまで一気通貫で実行するAIシステム。

### v2の進化ポイント

| 項目 | v1 | v2 |
|------|----|-----|
| ナレッジファイル数 | 3 | **12**（8要素個別 + 4システム） |
| ナレッジ総容量 | ~8.8KB | **~120KB以上**（約14倍） |
| 8要素の説明 | 各1-2行 | **各200-350行超**の完全ガイド |
| 統合ソース | なし | **新版6教育+心理操作テクニック集**の全知見 |
| コピーライティング支援 | なし | **11セクション**のツール集 |
| 件名テンプレート | 「3つ以上提示」のみ | **8カテゴリー×各4-6例**のテンプレート |
| マーケティング戦略 | なし | **キャラクター5大要素+ビジネスカルテ** |
| 設計図の詳細度 | 37行の表 | **各通の構成・ポイント・文字数目安付き** |
| 品質チェック | 簡易 | **4カテゴリー×20項目以上** |
| テクニック集 | なし | **サンドイッチ構造・リアルタイム感・コントラスト表現** |

## 使い方

### Step 1：入力情報の準備
`入力/` フォルダに、以下の情報を含むファイルを配置する：
- 商品/サービスの概要
- ターゲット（ペルソナ）
- USP（独自の売り）
- 価格帯
- 既存の実績・事例
- 目指すゴール

### Step 2：SKILL.mdに従って実行
AIに `SKILL.md` を参照させ、4フェーズを順に実行する：

1. **Phase 1：戦略設計** → 戦略設計シートの作成
2. **Phase 2：配列設計** → 各通の設計表の作成
3. **Phase 3：本文生成** → メール本文の生成（8要素個別ファイル参照）
4. **Phase 4：品質チェック** → 品質検証と改善

### Step 3：結果の確認
`実行結果/` フォルダに出力された成果物を確認する。

## D-BE UNICK 8要素（概要）

| # | 要素 | 日本語 | 一言説明 | 個別ファイル |
|---|------|-------|---------|----|
| D | Declaration of Distinction | 違いの宣言 | 「この人は他と違う」と感じさせる | `D_違いの宣言.md` |
| B | Benefit | 恩恵 | 読者が得られる具体的な変化を描く | `B_恩恵.md` |
| E | Evidence | 裏づけ | 主張を支える証拠・実績・数値 | `E_裏づけ.md` |
| U | Urgency / Uniqueness | 必然性 | 「今、行動する理由」を自然に創出する | `U_必然性.md` |
| N | Named Enemy | 名もなき敵 | 真の敵を名指し、共に倒す対象にする | `N_名もなき敵.md` |
| I | Innovation | 新しい勝ち方 | 敵を倒す新しいアプローチを示す | `I_新しい勝ち方.md` |
| C | Case Study | 体感事例 | 追体験できるレベルの詳細な事例 | `C_体感事例.md` |
| K | Kickstart | 最初の一歩 | 今すぐ踏み出せる具体的な行動 | `K_最初の一歩.md` |

### 核心の思想
- 従来のPASONA型：「問題煽り → 不安 → 売る」
- D-BE UNICK：**「違いの宣言 → 共鳴 → 仲間」**

### 倫理原則
- 読者を責めない・恐れだけで動かさない
- 嘘や過度な煽りを排除する
- 「仲間を生む」ことをゴールとする

## ナレッジファイルの詳細

### 8要素個別ファイル（コアナレッジ）

各ファイルは以下の構成で統一：
- 概要と役割
- 心理メカニズムの解説
- テンプレート・パターン集
- フレーズ集（20-60パターン）
- ステップメール内での配置戦略
- 実践的チェックリスト
- AIによる生成時の鉄則

**統合ソース**: 新版の「目的の教育」「問題点の教育」「手段の教育」「信用の教育」「投資の教育」「行動の教育」「読者心理操作テクニック集」の全知見（計10,000行超）を、D-BE UNICK倫理フレームで再構築して統合。

### D-BE_UNICK_コピーライティング強化ツール.md
11セクション構成のツール集：
1. 共鳴ワード集（D・B要素用）
2. 変革ワード集（I・N要素用）
3. 緊急性ワード集（U要素用）
4. 信頼構築ワード集（E・C要素用）
5. 行動促進ワード集（K要素用）
6. 感覚言語パレット（五感別）
7. コントラスト表現テンプレート
8. 件名テンプレート集（8カテゴリ）
9. サンドイッチ構造テンプレート
10. メールフォーマット・読みやすさの鉄則
11. ライティングチェックリスト

### D-BE_UNICK_ステップメール設計図.md
2つのシーケンスタイプの完全設計：
- **タイプA**：分岐後セールス型（10日間・10-13通）
- **タイプB**：高額商品販売型（10日間・23通）
各通の構成・件名例・使用要素・文字数目安・ポイントを網羅。
先行予約・リアルタイム感などのテクニック集も収録。

### D-BE_UNICK_戦略設計シートテンプレート.md
5セクション構成の戦略設計テンプレート：
1. 基本情報
2. マーケティング戦略設計（キャラクター5要素・ビジネスカルテ・統合戦略）
3. D-BE UNICK 8要素カードマッピング
4. シーケンス設計
5. トーン&マナー設定
