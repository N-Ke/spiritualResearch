## セクション5：トーン&マナー設定

```
【基本文体】
□ です・ます調（基本）
□ だ・である調
□ 混合（本文はです・ます、強調部分はだ・である）

【キャラクターの口調】
→ 例：「穏やかだが芯がある」「情熱的で引っ張る」「知的で冷静」

【使用する一人称】
→ 例：私 / 僕 / 俺 / 自分

【読者への呼称】
→ 例：あなた / 〇〇様 / みなさん → D-BE UNICKでは「あなた」推奨

【絶対に使わない表現（NG集）】
→ 読者の努力不足を責める表現
→ 「ゴキブリ」「バカ」等の侮辱的表現
→ 根拠のない煽り（「急がないと一生後悔」等）
→ 嘘の限定性（実際には限定でないのに「限定」）
→ 「買わない人はダメ」的なプレッシャー
```
