## 2. 許可付与の心理機制（**Permission-Granting Mechanism**）

このチャンネルの本質的な **報酬設計** は、視聴者が日常で背負わされている義務を **「やらなくていい」と公認する** こと。

| video_id | 解除される義務 | 「許可」の言い方 |
|---|---|---|
| VID001 | 努力すること | 「がんばるのやめた」 |
| VID003 | 感謝すること | 「感謝もしなくていい」 |
| VID004 | 落ちている自分を直すこと | 「炙り出されてランドリーされてる」 |
| VID005 | 寝るべき時に寝る／ワークをすること | 「**できない事はやらない**」 |
| VID002 | 努力で願望実現すること | 「気分にコミット」（努力からの離脱） |

**機能:**
- 視聴後 **即時の「ホッとする」** 体感を作る → **報酬予測誤差**が正方向。
- VID003 本人の言葉:「**やめるとスッキリします。**」「**ホッとするっていうのはこれ心地よいことなんですよ。つまりやめることによってエネルギー上がっているんですよ**」 — 視聴行為そのものの即時報酬を **明示的に内製化** している。
- 結果、**コメントで「楽になった」「肩の力が抜けた」型の感謝**が多い（VID001 コメント上位の常套）。

**自チャンネルでの示唆:**

## 7. 語彙のロックイン（**Lexical Lock-in**）

5 本通じて使われ続けるコア語彙が、**チャンネル固有のジャーゴン**として固定されている。

**コア語彙（出現頻度・体感）:**
- **心地よさ**（最頻 — チャンネル名と同等の概念）
- **気分**
- **エネルギー / 元気 / 元の気**
- **波動**
- **感じていること / 感じる**
- **しなければならない**（シリーズ語）
- **ゼロポイント / プラスポイント / マイナスポイント**
- **ブロック**
- **しないといけない**
- **コミット / 許可 / OK**
- **二極化 / 同期**

**機能:**
- 視聴者の **語彙置換**が起きる（コメントが同じ語で書かれる → コメント分析で確認）。
- **他チャンネルへの移行コスト**が上がる（他では「波動 = エネルギー = 元気」の三位一体は使わない）。
- 書籍『**心地よさセンサー**を磨けば人生は必ずうまくいく』など書籍タイトルもこの語彙系で統一されているため、**メディアまたぎでも語彙が崩れない**。

## 10. ファネル進化（**Funnel Evolution**）

**2020 年版 (VID004 / VID005):**
```
YouTube → メルマガ（kenji15）／stand.fm（ラジオ）
      → cloverpub 講座（願望実現基礎・31 日・¥）
      → tokihanate タロット講座（ベーシック／プロフェッショナル）
      → 別 ch（Kenji Tarot）
```
- 直接の有料商品（タロット講座・願望実現基礎）が **概要欄から 1 クリックで購入導線**。
- メルマガが **二次リスト**。

**2025-26 年版 (VID001 / VID002 / VID003):**
```
YouTube → 公式 LINE（first-class15.com）
      → 10 大特典で価値を蓄積
        ・電子書籍（9080 字）
        ・ロードマップスライド（112 枚）
        ・シークレット音声 2 本（計 63 分）
        ・YouTube 神回 5 本書き起こし PDF（16,447 字）
        ・Kenji 塾質問回答ラジオ（26 分）
        ・非公開動画 3 本
        ・「現実創造アルティメット企画」参加権
      → 「アルティメット企画」での有料化（推定）
      → 書籍 3 冊（KADOKAWA 2 冊・大和出版 1 冊・Amazon 直リンク）
      → Voicy・X・別 ch・お仕事問合せ（tokihanate.com）
```
- **LINE がほぼ唯一の入口**。**直接の販売 URL は概要欄上位から消えている**。
- かわりに **特典の物量で価値知覚を持ち上げ**、LINE 内で送客する **二段ロケット**。
- 書籍は「Amazon 直リンク」で **権威としての出版実績**＋**少額アクション可能なエントリー**を兼ねる。

**進化の意味:**
- **リスト主義への明確な移行**。LINE の上で何度も接触できることで、**1 回の視聴で売り切らない設計**。
- 「期間限定」が多用されるが、**LINE 登録は時限性が低い**ことが多く（実態は新規にも特典提供）、**緊急性は心理装置**として機能している側面が大きい。

## 11. 三項リストの定型（**Rule of Three**）

5 本に「3」の構造が織り込まれる。

| video_id | 三項 |
|---|---|
| VID001 | やめる 3 つ（できないことをやろうとする／ムカつくことを考える／現実を変えようとする） |
| VID002 | 心地よさ 3 つ（コーヒー・本・ドラゴンボール） |
| VID004 | 変化の兆し 3 パターン（妙な安心感・落ちる・興味が変わった） |
| VID005 | 自己の苦手 3 つ（人と合わせる・笑顔・しっかりした仕事） |
| 概要欄 | 書籍 3 冊・SNS 3 系列・特典 10 個（10 = 3+3+3+1 で読みやすく改行） |

**機能:**
- **記憶の上限**として教科書的な 3。**白板に書ける物理量**でもある。
- 「3 つ」と先に宣言するだけで、視聴者は **「終わりが見える」** ため離脱しにくい。

## 12. 過激コピー × 後出し緩和（**Bold Hook, Soft Caveat**）

タイトル／サムネは強気だが、本編は **必ず緩和節を入れる**。

| 強気側 | 本編の緩和 |
|---|---|
| 「努力をやめた途端」 | 「努力が悪いんじゃない／努力ルートも選択」 |
| 「コレをやめるだけで」（感謝） | 「感謝が悪いんじゃない／感謝こじらせが対象」 |
| 「これがきてたら大変化」 | 「すぐに現実は動いていない／内側の変化」 |
| 「できない事はやらない」 | 「オフィスで寝るのは別／責任ある場面は別」 |
| 「人生根こそぎ変える」 | 「コミットできなければ無理にやらない／勉強してから」 |

**機能:**
- **CTR は強気で取り、本編で炎上ラインを下げる**。
- 視聴者の中で「タイトルは強いがちゃんとした人」のキャラクター像が育つ → **信頼の二段構え**。
- 切り抜き拡散時にも、引用句単独で過激には見えても、文脈は緩和されているので長期的なブランド毀損を抑える。

## 13. プロダクション的なミニマリズム

- **ホワイトボード + 一人語り + 顔出し + 編集薄め** がすべての動画で固定。
- VID001 の sub.ja.srt（自動字幕）が補助で残るが、**字幕焼き付け・派手な SE/BGM・アニメーション**は使わない。
- これは **属人ブランドのコスト最小化**：本人の存在＝コンテンツであり、**演出が増えるほどホスト本人が薄まる**ジレンマを回避する選択。
- 反面、**スケール（チーム展開・コンテンツ複製）が困難**なので、書籍・LINE・Voicy などの **メディア横展開**で軸足を増やしている。

## 15. 倫理・コンプラ観測の境界線

**比較的安全に運用できているもの:**
- 個別ワークの提示は **「やってもよい／やらなくてもよい」**の許可型 → 義務化を避けている。
